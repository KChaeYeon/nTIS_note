## 연구 질문
ECAP(evoked compound action potential) 피드백을 이용한 폐쇄형 경골신경 자극 시스템이 OAB 방광 조절을 정밀하게 수행할 수 있는가?

## 방법
- **설계:** 폐쇄형(closed-loop) 신경 자극 시스템 개발 및 동물/생체 검증
- **핵심 기술:** ECAP 실시간 측정 → 신경 활성화 정도 정량화 → 자극 파라미터 자동 조절
- **이식형 장치:** 완전 이식형 경골신경 자극 + ECAP 기록 디바이스
- **결과 변수:** 방광 조절 효과, 신경 활성화 안정성, 파라미터 자동 조절 정확도

## 주요 결과
- 말초신경(경골신경)에서 ECAP 일관되게 측정 가능 확인
- ECAP 피드백 기반 자동 파라미터 조절 → 자극 일관성 유지
- 방광 기능 조절에서 기존 open-loop 대비 정밀도 향상
- 경골신경 TIS OAB 치료의 **개인화(personalized)** 경로 제시
- 말초신경 폐쇄형 자극 시스템의 기술적 실현 가능성 입증

## 다관점 요약
- **신경공학 관점:** ECAP 기반 폐쇄형 말초신경 자극이 TIS에도 적용 가능. TIS는 비침습적으로 ECAP 측정이 어렵지만, 피부 표면 EMG나 방광 압력 센서를 폐쇄형 피드백으로 활용하는 future direction 논의 가능.
- **임상 관점:** 이식형 + 폐쇄형이 OAB 치료의 궁극적 방향. 비침습 TIS는 그 전단계(덜 침습적) 포지션.
- **방법론 관점:** ECAP가 말초신경 자극의 표준 생체 지표로 부상. TIS 연구에서 근전도(EMG) 또는 신경 복합 활동 전위가 유사 역할 가능.

## 관련 갭
- **MG-1:** 생체신호 실시간 피드백 기반 폐쇄형 TIS 시스템 전무 → 본 논문이 선행 기술 기반 제공
- **G-TN1:** 경골신경 TIS 자체의 방광 효과 검증이 선행되어야 폐쇄형 TIS 개발 의미 있음
