## 연구 질문
경골신경 자극(TNS)의 최근 발전 현황과 남은 과제는 무엇인가? PTNS·TTNS·iTNS 세 방식의 효능·안전성·내구성 비교는?

## 방법
- **설계:** Narrative / Comprehensive review
- **범위:** TNS 세 가지 방식(PTNS, TTNS, iTNS) 비교; 기전, 임상 효능, 안전성, 실용성, 미래 방향 포괄

## 주요 결과

### 종합 효능 요약
- 최신 RCT·코호트·메타분석 결합 → **반응률 60–80%** (urgency, 빈뇨, nocturia, UUI, QoL 개선)
- TTNS: PTNS와 단기 효능 유사 + 비침습·자가 시행 가능 실용적 장점
- iTNS(이식형): 장기 내구성 우수, 삭수신경조절(SNM)과 비열등(non-inferior) 효능

### 방식별 비교
| 방식 | 침습도 | 효능 | 내구성 | 편의성 |
|------|--------|------|--------|--------|
| PTNS | 반침습(바늘) | 60–71% | 유지 치료 필요 | 클리닉 방문 |
| TTNS | 비침습(패치) | PTNS 동등 | 자가 치료 가능 | 가정 사용 |
| iTNS | 침습(이식) | SNM 비열등 | 장기 내구 | 충전/관리 필요 |

### 미해결 과제
- 자극 파라미터(PW, 주파수, 세기, 치료 기간) 이질성 → 표준화 필요
- Non-responder 예측 바이오마커 전무
- 비침습 방식의 장기 안전성 데이터 부족

### 비교 데이터
- SNM vs. iTNS: 유사 효능, iTNS가 재시술률 낮은 경향 (추가 추적 필요)
- 보툴리눔독소(BoNT-A): 효능 가장 강하나 부작용(요폐) 위험 → TNS 안전 프로파일 우수

## 한계점
- 방식별 RCT 직접 비교 제한적 (head-to-head 데이터 부족)
- iTNS 장기 데이터 아직 부족 (2–5년 추적 필요)

## 다관점 요약
- **신경공학 관점:** 비침습 TIS가 TTNS의 단점(낮은 선택성, 표피 자극)을 극복하는 차세대 기술로 자연스럽게 포지셔닝 가능.
- **임상 관점:** AUA 가이드라인에서 SNM·PTNS·BoNT-A 동등 3차 치료 권고 — TNS의 임상적 지위 확립.
- **방법론 관점:** 파라미터 이질성 문제가 TIS 연구에서도 동일하게 적용 — 표준화된 파라미터 보고 필요.

## 관련 갭
- **G-TN1, G-TN2:** 리뷰에서 비침습 TNS의 한계로 '선택성 부족' 지적 → TIS 방식이 이를 해결하는 차세대 접근임을 지지
- **EG-5:** 비침습 말초신경 TIS 기능적 증거 전무 gap 확인

## nTIS 연구와의 연관성
- Introduction 마지막 단락에서 "기존 TNS의 한계 → TIS 방식의 필요성"을 논증하는 데 핵심 인용
- 연구 결과가 나온 후 Discussion에서 TIS와 기존 TNS 방식 비교 논의 기준점 제공
