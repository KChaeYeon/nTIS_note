# Botzanowski et al. 2022 — Noninvasive Peripheral Nerve TIS (TINS)

**Citation:** Botzanowski B, Donahue MJ, Bhaskaran A, et al. Noninvasive stimulation of peripheral nerves using temporally-interfering electrical fields. *Advanced Healthcare Materials* 2022;11(17):e2200075. DOI: [10.1002/adhm.202200075](https://doi.org/10.1002/adhm.202200075)

---

## 핵심 요약

**TINS(Temporal Interference Nerve Stimulation) 완전 비침습 적용 최초 실증.** 피부 부착형 다전극 배열을 이용해 생쥐 좌골신경(sciatic nerve)을 경피적으로 선택 자극. 기존 경피 전기자극(TES) 대비 낮은 전류로 심부 표적 신경 활성화.

---

## 연구 설계

| 항목 | 세부 사항 |
|------|----------|
| 동물 모델 | 쥐 (murine sciatic nerve model) |
| 전극 | 피부 부착형 유연 다전극 배열 (flexible multielectrode array) |
| 캐리어 주파수 | 3 kHz (f₁), 3 kHz + Δf (f₂), Δf = 0.5–4 Hz |
| 타겟 신경 | 좌골신경 (sciatic nerve) → trifurcation: tibial/sural/peroneal |

---

## 주요 결과

- Δf 주파수(0.5–4 Hz)에서 **하지 근육 수축 유발** → TINS 기능 검증
- TES 대비 **유의미하게 낮은 전류 진폭**으로 동일 반응 달성
- 유연 전극 배열로 신경 위치 정렬 → 타겟 선택성 향상
- RC 회로 모델: 고주파 캐리어 → 세포막 통과 불가, 포락선(Δf)만 통과

---

## 방법론적 의의

```
피부 표면 전극 A (3000 Hz) ─────┐
                                 ├→ 좌골신경 위치: Δf=1Hz 포락선 → 신경 활성화
피부 표면 전극 B (3001 Hz) ─────┘
```

- **완전 비침습** 경피 TIS의 말초신경 적용 가능성 증명
- 가요성(flexible) 전극의 피부 밀착 → 전극-신경 정렬이 효과에 결정적

## nTIS 연구와의 연관성

| 연관 항목 | 내용 |
|---------|------|
| 직접 선행 연구 | 경골신경 TINS의 직접적 방법론적 근거 |
| 전극 설계 | 유연 다전극 배열 → 발목 경골신경 커프 전극 설계 참고 |
| 캐리어 주파수 | 3 kHz carrier → Kim 2023 FEM 모델 동일 설정 |
| 한계 | 쥐(murine) 모델 → 인간 스케일링 검증 필요 |
