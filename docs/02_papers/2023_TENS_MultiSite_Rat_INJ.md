## 연구 질문
경피 전기 자극 부위(복부, 귀, 경골신경)에 따라 rat detrusor overactivity 모델에서 방광 수축 억제 효과가 다른가? 경골신경의 킬로헤르츠 범위 자극이 효과적인가?

## 방법
- **설계:** 동물 실험 pilot study, 마취(anesthetized)
- **표본:** 암컷 Sprague-Dawley rat 3마리 (소규모 pilot)
- **OAB 모델:** Detrusor overactivity (방법 확인 필요)
- **자극 부위 비교:**
  1. 복부벽 (T6-L1 수준) — 20 Hz
  2. 귀 (미주신경 분지) — 20 Hz
  3. 발목 (경골신경) — **킬로헤르츠 범위** (kHz) 자극
- **결과 변수:** 방광 수축 횟수, 수축 압력, 수축 지속시간

## 주요 결과
- **복부 및 귀 자극 (20 Hz):** 수축 횟수 감소 — 효과적인 방광 억제
- **경골신경 kHz 자극:** 방광 수축 억제 효과 확인, 20 Hz 자극과 다른 방식으로 작용
- 세 부위 모두 방광 수축 횟수 감소 효과 (정도의 차이)
- **kHz 범위 경골신경 자극이 OAB 치료 가능성 제시** — TIS 캐리어 주파수(kHz)와 연관성

## 한계점
- 소규모 (n=3 rat) — 통계적 검정력 매우 낮음 (pilot만)
- 마취 rat → 각성 상태와 다른 신경 반응
- 캐리어 kHz 자극의 정확한 주파수·파라미터 미상세 보고
- Sham 대조군 없음

## 다관점 요약
- **신경공학 관점:** kHz 경골신경 자극이 OAB에 효과적이라는 점은 TIS(carrier 1–4 kHz)의 경골신경 적용 직접 근거. 단, 여기서 kHz는 단순 고주파 — TIS의 간섭(interference) 원리와 다름.
- **임상 관점:** 복부·귀 자극도 OAB에 효과 → 다중 자극 부위 연구. 단, 임상 번역에는 표준 PTNS 위치(경골신경)가 더 직접적.
- **방법론 관점:** 소규모 pilot이지만 kHz 경골신경 자극의 선행 동물 데이터로 가치 있음. TIS의 kHz 캐리어 + kHz 범위 효과 비교 연구로 발전 가능.

## 관련 갭
- **G-TN1:** kHz 범위 경골신경 자극 동물 연구 존재하지만, **TIS(두 주파수 간섭)** in vivo는 전무
- **KG-4:** kHz 단독 vs. kHz 간섭(TIS) 효과 차이 미규명

## nTIS 연구와의 연관성
- Introduction에서 "고주파 경골신경 자극 → OAB 억제" 선행 증거 인용 시 활용
- TIS 캐리어 주파수(kHz) 선택의 동물 실험 근거 제공
- 단, TIS의 고유성(envelope 형성 vs. 단순 kHz 자극)을 명확히 구분해야 함
