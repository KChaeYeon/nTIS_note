## 연구 질문
Rat detrusor overactivity 모델에서 경골신경 자극(TNS)이 방광 구심성 신경(afferent) 활동에 미치는 영향은 무엇인가?

## 방법
- **설계:** 동물 실험 (in vivo, urethane 마취)
- **표본:** 수컷 Wistar rat, OAB 모델 (0.5% acetic acid 방광 주입으로 detrusor overactivity 유도)
- **자극:** 경골신경 30분, 5 Hz, PW 200 µs, 강도 ~3× toe-movement threshold
- **결과 변수:**
  - Bladder afferent nerve activity (pelvic nerve branch electrophysiology)
  - Cystometry (방광 내압, 용적, 수축 간격)

## 주요 결과
- 0.5% acetic acid 주입 → 방광 용적 **유의미하게 감소** + 구심성 신경 활동 **급격 증가**
- TNS 30분 → 방광 용적 회복 + 구심성 신경 활동 **유의미하게 억제**
- TNS가 OAB 상태에서 과활동 방광 구심성 신호를 정상화 수준으로 복원
- 효과는 자극 중 및 자극 직후 유지

## 한계점
- Urethane 마취 → 중추 신경계 억제 조건 (각성 rat 대비 기전 해석 제한)
- 단일 주파수(5 Hz)만 검토 → 최적 주파수 확인 안 됨
- Acetic acid 농도: 0.5% (일부 연구는 0.25%–1.0% 범위 사용, 비교 제한)
- 수컷 rat만 → 성별 차이 미검토

## 다관점 요약
- **신경공학 관점:** 경골신경 자극의 방광 억제 메커니즘을 **구심성 신경 활동 직접 기록**으로 최초 규명. TIS 연구에서도 동일 측정 지표 사용 가능.
- **임상 관점:** Acetic acid OAB rat 모델 → 사람 OAB와 신경생리적 유사성 높음. PTNS 임상 효과의 동물 기전 근거.
- **방법론 관점:** Pelvic nerve electrophysiology + cystometry 동시 기록이 방법론적 강점. TIS 실험 설계 시 동일 방법 채택 권고.

## 관련 갭
- **G-TN1:** 경골신경 **TIS** in vivo 전무 — 본 논문은 침습 전기 자극(PTNS 유사)
- **G-TN2:** TIS vs. PTNS 직접 비교 없음

## nTIS 연구와의 연관성
- **실험 프로토콜 템플릿:** Acetic acid OAB 모델 + cystometry + 구심성 신경 기록이 본 연구의 방법론 기반
- **비교 대조군 설계:** TIS 그룹 vs. PTNS 그룹(이 논문의 방법) 직접 비교 실험 설계에 참고
- Methods 섹션 OAB 모델 설명 및 측정 지표 기술에 핵심 인용
