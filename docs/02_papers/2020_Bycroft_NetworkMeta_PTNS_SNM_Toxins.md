## 연구 질문
OAB 3차 치료(보툴리눔독소 A, 천수신경조절, 경골신경 자극)의 효능과 안전성을 네트워크 메타분석으로 비교하면 어떤 결과가 나오는가?

## 방법
- **설계:** 체계적 문헌고찰 + 네트워크 메타분석 (NMA)
- **포함 기준:** OAB 3차 치료 RCT
- **비교 대상:**
  1. OnabotulinumtoxinA (BoNT-A, 보툴리눔독소)
  2. Sacral neuromodulation (SNM, 천수신경조절)
  3. Peripheral tibial nerve stimulation (PTNS)
  4. Sham/placebo
- **결과 변수:** 배뇨 빈도, 절박성 요실금, QoL, 부작용

## 주요 결과

### 효능 순위 (네트워크 메타분석)
| 치료 | 절박성 배뇨 빈도 | 절박성 요실금 | QoL |
|------|----------------|--------------|-----|
| BoNT-A | 최우수 | 최우수 | 우수 |
| SNM | 우수 | 우수 | 우수 |
| PTNS | 중간 | 중간 | 중간 |
| Sham | 최하 | 최하 | 최하 |

- **BoNT-A가 전체적으로 효능 최강** — 단, 요폐(urinary retention) 부작용 위험
- **SNM > PTNS** (효능), 하지만 PTNS는 안전 프로파일 우수
- **PTNS:** BoNT-A 대비 낮은 효능, 하지만 부작용 거의 없음

### 안전성 비교
- BoNT-A: 요폐 위험 높음 (~20–30%)
- SNM: 이식 관련 부작용, 재시술 필요
- PTNS: 부작용 최소 — **안전성 최우수**

## 한계점
- 연구 간 이질성 높음 (결과 지표, 측정 도구, 치료 기간 다양)
- PTNS 장기 유지 치료 데이터 부족 → 장기 효능 비교 제한
- 네트워크 메타분석의 간접 비교 한계

## 다관점 요약
- **신경공학 관점:** PTNS 효능이 BoNT-A·SNM보다 낮지만, 안전하고 비침습적. TIS가 PTNS의 선택성과 효능을 동시에 향상시킬 수 있다면 1·2·3위 방법의 장점을 통합 가능.
- **임상 관점:** AUA 가이드라인에서 세 방법 모두 3차 치료 동등 권고 — 환자별 맞춤 선택. TIS는 PTNS의 비침습 장점 + 개선된 신경 선택성으로 이 구도를 바꿀 수 있음.
- **방법론 관점:** 향후 TIS RCT 계획 시 네트워크 메타분석에 편입될 수 있도록 표준화된 OAB 결과 지표(OABSS, I-QOL) 사용 필수.

## 관련 갭
- **G-TN2:** PTNS vs. TIS 직접 비교 전무 — 본 네트워크 분석은 PTNS의 임상적 위치를 정의하는 배경
- **EG-5:** TIS가 이 네트워크에 편입되려면 기초 in vivo 증거 먼저 필요
