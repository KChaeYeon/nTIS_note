# Sunshine et al. 2021 — Phrenic Nerve TIS for Respiratory Restoration

**Citation:** Sunshine MD, Nair J, Dobson CM, Fuller DD. Restoration of breathing after opioid overdose and spinal cord injury using temporal interference stimulation. *Communications Biology* 2021;4:107. DOI: [10.1038/s42003-020-01604-x](https://doi.org/10.1038/s42003-020-01604-x)

---

## 핵심 요약

**최초의 말초신경 TIS 복수 생체 적용 연구.** 두 고주파 캐리어(5000 Hz / 5001 Hz)로 횡격막(phrenic nerve)를 비침습적으로 선택 자극하여 오피오이드 과다복용 및 경추 척수손상 후 호흡을 회복시켰다.

---

## 연구 설계

| 항목 | 세부 사항 |
|------|----------|
| 동물 모델 | Sprague-Dawley 쥐 |
| 자극 부위 | 목 근육 내 삽입 와이어 (오피오이드 모델), 경추 경막외 전극 (SCI 모델) |
| 캐리어 주파수 | f₁ = 5000 Hz, f₂ = 5001 Hz → Δf = 1 Hz |
| 타겟 | 횡격막 신경(C3-C5 경로), 횡격막 |

---

## 주요 결과

- 오피오이드 과다복용(펜타닐) → TIS 즉시 적용 → 호흡 **즉시 회복**
- C2 척수 반절단(SCI) → 단측/양측 횡격막 활성화 가능
- 기존 경피 전기자극(TES) 대비 **낮은 전류 진폭**으로 효과 달성
- 횡격막 이외 근육(대흉근 등) **불활성화 확인** → 타겟 선택성

---

## 방법론적 의의

```
f₁ (5000 Hz)  ─────────────────────────────►
f₂ (5001 Hz)  ─────────────────────────────►
                        ↓ 간섭
         횡격막 위치에서만 Δf = 1 Hz 포락선 생성
```

- TIS로 **심부 표적 신경(phrenic nerve)**을 비침습 자극 가능함을 최초 실증
- 경골신경 TIS 적용의 개념적 선행 연구

## nTIS 연구와의 연관성

| 연관 항목 | 내용 |
|---------|------|
| 방법론적 선례 | 말초신경(횡격막신경) TIS의 최초 in vivo 실증 |
| 전극 구성 | 2채널 경피 전극 배치 원리 동일 |
| 한계 | 침습적 전극(neck wire) 사용 — 완전 비침습 PTNS 달성은 미완 |
