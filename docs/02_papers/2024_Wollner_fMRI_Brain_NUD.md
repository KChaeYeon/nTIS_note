## 연구 질문
경피 경골신경 자극(TTNS) 급성 적용 시 건강인의 뇌 기능적 연결성(functional connectivity)은 어떻게 변화하는가?

## 방법
- **설계:** Feasibility proof-of-concept, 3T fMRI
- **표본:** 건강인 자원자
- **자극:** 급성(acute) TTNS, 교대 자극/휴식 블록 설계
- **fMRI 분석:** 자극 전후 resting-state fMRI + 자극 중 fMRI
  - 전뇌 seed-based 기능적 연결성 상관 분석 (TTNS vs. 휴식 비교)
- **결과 변수:** 뇌 기능 네트워크 연결성 변화

## 주요 결과
- TTNS 자극 기간 동안 특정 뇌 네트워크 연결성 변화 확인
- 감각-운동 피질, 대상회(cingulate cortex), 도엽(insula) 등 방광 제어 관련 영역 활성화
- 자극 전 resting-state vs. 자극 중 connectivity 비교로 급성 TNS의 중추 효과 정량화 가능성 제시
- fMRI가 TTNS의 중추 작용 기전 연구에 실용적임을 확인

## 한계점
- 건강인 대상 → OAB 환자에서의 뇌 활동 변화와 직접 비교 불가
- 급성 단일 세션 → 만성 치료 후 뇌 가소성 변화 미평가
- fMRI 환경에서 자극 장비 호환성 제한 (MRI 호환 자극기 필요)
- 소규모 feasibility 연구 → 통계적 검정력 제한

## 다관점 요약
- **신경공학 관점:** TTNS가 방광 제어 관련 중추 신경망(insula-cingulate)에 직접 영향을 미침 확인. TIS 버전의 경골신경 자극이 더 선택적 중추 반응을 유도할 수 있는지 비교 연구 필요.
- **임상 관점:** OAB 환자에서 뇌 연결성 이상이 원인인 경우 TNS의 중추 재조정 효과가 치료 기전일 수 있음.
- **방법론 관점:** fNIRS(2023)보다 공간 해상도 우수하나 MRI 환경 제약. 향후 TIS fMRI 연구의 방법론 참고.

## 관련 갭
- **G-TN1:** TIS 방식의 경골신경 자극 중추 반응 연구 전무
- **MG-1:** 실시간 뇌 피드백 기반 폐쇄형 TIS 시스템 설계 필요성

## nTIS 연구와의 연관성
- Discussion에서 TIS의 중추 조절 메커니즘 논의 시 fMRI 근거로 인용
- 향후 연구 방향: Rat OAB 모델 검증 후 인체 TIS fMRI 연구로 확장 가능성 제시
