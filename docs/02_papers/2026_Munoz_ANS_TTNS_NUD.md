## 연구 질문
건강인에서 TTNS 급성 시행이 자율신경계(ANS) 활동에 어떤 영향을 미치며, HRV를 통해 신경자율적 경로를 모델링할 수 있는가?

## 방법
- **설계:** Open-label, 전향적, 탐색적(exploratory), 단일군 연구
- **표본:** 건강인 20명 (여성 11명, 중앙값 나이 31세)
- **프로토콜:** 3단계
  1. 기저 안정 (10분)
  2. 지속적 sub-motor TTNS (왼쪽 경골신경 표면 전극, 10분)
  3. 자극 후 회복 (10분)
- **결과 변수:** HRV 지표 (RMSSD, LF/HF ratio, pNN50, SDNN)

## 주요 결과
- TTNS 자극 중 HRV 파라미터 변화 관찰 → 자율신경 반응 확인
- 부교감 지표(RMSSD, pNN50) 증가 경향 → 부교감 신경 활성화 시사
- 자극 후 회복기에 ANS 반응 원상 복귀
- 건강인에서도 TTNS의 자율신경 조절 효과 확인 → OAB 환자의 자율신경 불균형 교정 가능성

## 한계점
- 건강인 대상 → OAB 환자에서의 자율신경 반응 직접 적용 불가
- 소규모 (n=20), 탐색적 연구 수준
- Sub-motor 강도 → 감각역치 이상의 임상 강도와 차이 가능
- 단일 세션, 단기 자율신경 측정

## 다관점 요약
- **신경공학 관점:** 경골신경 TTNS가 단순 방광 반사 억제를 넘어 자율신경계 전반 조절 가능. TIS로 확장 시 ECG/HRV를 실험 outcome으로 추가 측정 가능.
- **임상 관점:** OAB는 자율신경 불균형과 연관될 수 있음 → TNS/TIS가 방광 + 자율신경 동시 조절 치료 가능성.
- **방법론 관점:** 연구자가 보유한 ECG/HRV 장비를 TIS 실험 부가 측정 지표로 활용 직접 가능. 저비용 추가 outcome.

## 관련 갭
- **G-A1, G-A5:** 경골신경 TIS의 자율신경 효과 연구 전무
- **AG-6:** 생체신호 동기화 폐쇄형 TIS (ECG/RSP 기반) 연구 가능성 지지

## nTIS 연구와의 연관성
- **즉시 활용 가능:** ECG/HRV 장비 보유 → Rat OAB TIS 실험에 ECG 동시 기록 추가 출력 지표로 측정 가능
- 단일 실험에서 방광 + 자율신경 동시 효과 검증 → 논문 contribution 확장
- Methods에서 "부가 측정 지표: HRV" 추가 정당화 근거
