## 도출된 연구 방향

### TIS 단독 사용 시 주요 방향 5가지

**방향 1. 수면 신경생리 + TIS**
- MCI/치매 환자 다중 세션 수면 방추파 TIS
- 개인 방추파 피크 주파수 기반 Δf 개인화
- 선행: SWA TIS 2025 → 임상군 확장

**방향 2. 폐쇄형 TIS (아무도 안 함)**
- EEG 기반 수면 서파 위상 잠금 TIS
- IED 예측 → 자동 트리거 TIS
- 선행: ECAP CL-SCS (SCS 분야 선례)

**방향 3. 신경정신 적응증**
- 알츠하이머 대규모 후속 + 신경 biomarker
- DOC 시상 TIS 계통적 연구
- 우울증/PTSD TIS (임상시험 결과 待)

**방향 4. 말초·자율신경 TIS**
- 경부 TIS vs. taVNS 자율신경 직접 비교 (ECG/HRV 기반)
- 경골신경 TIS for 과민성 방광
- 설하신경 TIS 장기 OSA 프로토콜

**방향 5. 파라미터 체계화 + 반응 예측**
- 캐리어 주파수 체계 비교
- 베이스라인 EEG + ML → TIS 반응 예측 모델

### nTIS 고유 연구 방향 5가지

**방향 A. 다중 표적 동시 자극**
- PD: STN + GPi 동시 nTIS
- 뇌전증: 다발성 발작 초점 동시 억제

**방향 B. 수면 SO-Spindle-Ripple 동시 조율**
- 피질(Δf=1Hz) + 시상(Δf=12Hz) + 해마(Δf=5Hz) 동시
- 세 표적 동시 자극은 현재 어떤 방법으로도 불가능

**방향 C. 개인화 최적화 알고리즘**
- MRI/DTI → FEM → 위상-강도 동시 RL 최적화
- TI-Toolbox nTIS 확장

**방향 D. 척수 nTIS 재활**
- 배측/복측 + 좌/우 동시 선택적 자극

**방향 E. n-phase 기전 시뮬레이션**
- Hodgkin-Huxley + n-phase 위상 자유도
