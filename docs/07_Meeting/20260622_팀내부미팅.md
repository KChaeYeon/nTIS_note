## 논의 내용

### 1. TIS 시스템 채널 캘리브레이션 결과
- 채널별 동일 scale factor 인가 시 출력 일치 여부 테스트 완료
- **이슈**: 0번 포트에서 동일 scale factor 대비 출력 편차 발생 → 파형 왜곡 캡처 확인 필요
- 교수님 직접 테스트한 시스템: 채널 간 편차 1% 미만 (3.1~3.4 mA 수준에서 안정적)
- High frequency noise (사인파에 띠 형태의 HF 성분 혼입) → AM modulation 관점에서는 무시 가능 수준

### 2. Tibial Nerve TIS 자극 테스트 데이터 (교수님 직접 수행)
- 복숭아뼈 기준 여러 electrode 배치 방식 테스트:
  - 복숭아뼈 양쪽 끼우는 방식 (최적)
  - 발목 위아래 longitudinal 배치
  - 발목 위 + 발 아래 cross 배치
- **결과**: 복숭아뼈 양쪽 배치에서 자극 확률 가장 높음
- **발가락 움직임 (motor)**: 15 mA 수준에서 확인
- **감각 (sensory)**: Motor threshold 아래에서 발바닥 엄지 쪽 간지러움 sensation 확인

### 3. 교수님 핵심 연구 아이디어 (이번 미팅 핵심)

#### 3-A. Carrier Frequency 최적화
- 기존 접근: 1~5 kHz carrier
- **교수님 제안**: 100~150 kHz carrier가 peripheral nerve에서 20~30% 더 효율적
  - 근거: peripheral nerve fiber 특성이 중추신경과 다름
  - 100 kHz 기준으로 10 Hz beat frequency → 10컬로치(10 kHz?) 관련 내용 재확인 필요
  - **연결 갭**: G-T2 (최적 carrier frequency 미결정)

#### 3-B. Theta Burst 패턴 자극
- **아이디어**: Continuous AM modulation 대신, 20 Hz on/off burst 형태로 자극
  - Carrier 100~150 kHz를 20 Hz로 펄스화
  - Theta burst stimulation (TBS)와 유사한 패턴
  - Peripheral nerve에서 더 효과적일 수 있음 (뇌와 다른 메커니즘)
- **주의사항**: Rectangular burst는 피부 불편감 가능성 → waveform smoothing 필요

#### 3-C. AM Modulation Zone vs Inhibition Zone 구분 (가장 중요한 신규 아이디어)
- **문제 제기**: 기존 시뮬레이션/팬텀 논문은 AM modulation만 표시 → inhibition zone 미표시
- **교수님 직관**:
  - Electrode에 너무 가까운 영역 → intensity가 항상 threshold 이상 → **항상 자극됨(inhibition zone)**
  - Electrode에서 적당히 먼 영역 → AM modulation으로 켜졌다 꺼졌다 → **치료 타겟 영역**
  - 너무 가까우면 AM modulation 자체가 만들어지지 않음 → **buffer distance 존재**
- **제안**: 새로운 시각화 지표 필요
  - 기존 AM_max map + inhibition zone(intensity > threshold) overlay
  - → "dual overlay map"
- **민형에게 숙제**: 민형 논문 작성 시 이 두 영역을 구분해서 재정의할 것

#### 3-D. LED Phantom 정량화 실험 (민형 논문 연계)
- 교수님 요청:
  1. Intensity를 점진적으로 올리며 비디오 촬영
  2. 각 LED 위치의 voltage 실측 → AM modulation 시작 voltage 기록
  3. LED 빛 강도(비디오)와 voltage 변화 상관관계 매핑
  4. "항상 켜진" vs "깜빡이는" LED 경계 정량화

#### 3-E. 3-Phase TIS Controllability 검증 (앞발/뒷발 구분)
- **관찰**: Rat에서 앞발/뒷발이 완전히 같은 게 아닌 것 같다 → 구분 자극 가능할 수도 있음
- **제안**: 3-phase electrode 배치로 자극 위치 앞/뒤로 이동 → 앞발/뒷발 반응 구분 확인
- "데모를 보여줄 때 controllability를 얘기해줄 수 있으면 훨씬 설득력이 있다"

#### 3-F. Rheobase / Chronaxie 분석
- 교수님이 직접 rat phantom에서 측정한 데이터 공유:
  - Rheobase: ~25 V
  - Chronaxie: ~30 μs (50 V 인가 시 minimum duration)
- Nerve마다 다름 → Tibial nerve 특성에 맞는 pulse duration 최적화 근거로 사용 가능

### 4. 연세대 동물실험 연계
- 목표: 다다음 주 내 연세대 방문 → Rat motor cortex TIS 테스트
- 준비 사항:
  - 민형: 실험 데이터 정리 후 전달
  - Rat 뇌 atlas (Paxinos) 기반 타겟 위치 (앞발/뒷발 motor cortex) 확인
  - 시스템 가져가서 팬텀 3D controllability 테스트 병행

### 5. 페이퍼 방향 논의 (민형 논문)
- 기존: AM modulation map만으로 논문 구성
- 교수님 제안: Inhibition zone 개념 추가 → "AM modulation이 항상 켜진 파트"를 재정의
- "AM modulation이 안에서 생길 수 있는지, 포커스되는 거를 시뮬레이션으로 보여줄 수 있으면 좋겠다"
