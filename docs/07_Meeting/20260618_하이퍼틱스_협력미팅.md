## 핵심 논의 내용

### 1. 하이퍼틱스 TIS 기술 현황
- **n-phase TIS (튜노 테크놀로지)**: 기존 2-phase TIS의 MIT 특허를 회피하기 위해 3-phase (0°/120°/240°) 방식 자체 개발
  - 전극 위치 고정 상태에서 intensity 조절만으로 전후좌우 steering 가능
  - 2-phase는 좌우만 가능 → 3-phase는 전후좌우 모두 가능
- **원숭이 실험**: 국가 영장류센터에서 TIS + fMRI — hippocampus activation 확인 (2마리, 논문 submission 중)
- **화학 sensing 동시 측정**: 도파민 레벨 실시간 측정 기술 (FSCV) 보유 → TIS 자극 전후 striatum dopamine 변화 확인
- **자극 시각화 도구**: LED array로 AM modulation 영역 실시간 확인 — 업계 최초

### 2. OAB 시장 및 임상 방향
- 현재 치료 실패 환자 약 30% → 보툴리눔 독소 외 선택지 없음
- 기존 implantable 치료기기 (Axonics SNS, iCoin 티벨신경 자극기) 임상 효과 우수하나:
  - 한국 시장 활성화 실패 (환자 거부감, MRI 비호환 등)
  - PTNS는 한의사 규제 문제로 국내 미도입
- **비침습 wearable 접근**: 양말 형태 전극 → 가정에서 착용 후 병원 리포트 모델
- **개발 일정**: 9월 시스템 v2 출시 → 안전성 평가 → 탐색 임상시험 신청

### 3. Tibial Nerve 전극 배치 논의
- 복숭아뼈(medial malleolus) 기준 배치 테스트 완료:
  - 발목 앞뒤 수직 배치 vs 복숭아뼈 양쪽 평행 배치 여러 variant 시도
  - 15 mA 수준에서 toe 움직임(motor) 확인
  - Sensory threshold (발바닥 간지러움) → motor threshold (toe 움직임) 구간이 치료 타겟
- **교수님 판단**: Tibial nerve는 너무 작아서 내부 steering보다 피부 바깥에서 nerve 전체를 포괄하는 방식이 현실적

### 4. 향후 협력 방향
- 하이퍼틱스 시스템으로 동물실험 연계 가능성 논의
- 이기성 교수: "임상은 빨리 해야 한다. 정년 2년 남았다"
- 탐색 임상 신청 목표: 2026년 말~2027년 초
