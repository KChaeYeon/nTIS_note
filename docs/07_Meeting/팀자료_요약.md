## 송솔웅 — 학위논문 요약

**제목:** Development of a non-invasive deep tissue electrical stimulation: n-phase temporal interference stimulation

**핵심 기여:**

### 1. nTIS 이론 체계 (Chapter 2)
- 기존 2-pair TIS → n-phase 일반화 프레임워크 수립
- nTIS 전기장: **타원형 회전 궤적 (Elliptically Rotating Electric Fields)**
- 수학적 구조:
  $$\mathbf{E}_{total}(\mathbf{r}, t) = \sum_{k=1}^{n} \mathbf{E}_k(\mathbf{r}) \cos(2\pi f_k t + \phi_k)$$
- n-phase 전류 아키텍처 + Balanced Current Injection 조건 확립
- Modulation envelope 분석 (3가지 Case: Coincidence / Upper-bound / Intersection)

### 2. FEM 수치 검증 (Chapter 3)
- 3D 균질/이질 모델 (Rat brain, 3D cubic)
- 전기장 방향: Horizontal / Vertical / Diagonal / Rotational
- Grossman 2017 논문 재현 검증 완료
- nTIS Steering 및 Controllability 특성 규명

### 3. LED 팬텀 실험 (Chapter 4)
- Phase-synchronized 다채널 신호 생성 시스템 구축
- LED 팬텀: 실제 전기장 분포 가시화
- 수치 예측 vs. 실험 결과 비교 검증

**핵심 결론:**
- nTIS는 기존 TIS 대비 공간 제어 자유도 증가
- 3-phase TIS → 회전 전기장으로 특정 방향성 조절 가능
- LED 팬텀으로 이론 검증 완료

**관련 파일:**
- `Manuscript/Thesis_SolwoongSong_Uploaded.pdf` (최종 제출본)
- `Figures/` — Horizontal, Vertical, Diagonal, Rotational, 3D rat 전기장 결과
- `Presentation/` — 주간회의 발표 자료 시리즈
- `디펜스1/`, `디펜스2/` — 학위 심사 자료

- 표적: Motor cortex M1 (left)
- 캐리어: 2000 Hz + 2005 Hz (Δf = 5 Hz)
- 총 전류: 5 mA (전극 당 2.5 mA)
