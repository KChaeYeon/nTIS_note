### STEP 1 ✅ OAB 병태생리학

**핵심 내용:**
- OAB 정의 (ICS 2002): urgency 필수 증상 / OAB ≠ DO 구분
- 역학: 전세계 ~11%, 한국 ~12.2% (40세 이상)
- 방광 해부학: Urothelium(능동 센서) → Lamina propria → Detrusor muscle → IUS → EUS
- 정상 배뇨 기전: 저장기(교감 ON, 부교감 OFF) ↔ 배뇨기(부교감 ON)
- 배뇨 스위치: 척수가 아닌 뇌간 **PMC (Pontine Micturition Center)**
- 신경 3경로: 교감(T10-L2, 저장) / 부교감(S2-S4, 배뇨) / 체성(S2-S4, 괄약근)
- OAB 발생 기전 3가지: Myogenic / Neurogenic / Urothelial (ATP → P2X3)
- 치료 흐름: 약물 → PTNS/SNM → 비침습(TIS)
- 경골신경 표적 근거: L4-S3 → S2-S4와 동일 레벨 → 방광 부교감 반사 억제

### STEP 2 ✅ 경골신경 해부학 & 자극 기전

**핵심 내용:**
- 경골신경 위치: 발목 안쪽 내과 5 cm 위 / 뿌리 L4-S3
- 내부 구조: Epineurium → Fascicle → Axon → Myelin
- 신경섬유: Aα > Aβ > Aδ > C (굵을수록 자극 역치 낮음)
- 탈분극 기전: 전류 → Na+ 채널 강제 개방 → -70mV → -55mV(역치) → 탈분극
- 활동전위 전파: 도미노 원리 (탈분극된 부위가 옆 채널 개방)
- Saltatory conduction: Myelin → Ranvier node에서만 탈분극 → 점프 전도 → 빠름
- PTNS 파라미터: 20 Hz, 200 μs, 0.5~9 mA, 30분/회
- 방광 억제 경로: Aδ/Aβ → S2-S4 → GABAergic interneuron → 부교감 차단
- 장기 효과: 반복 자극 → 신경 가소성 → 억제 회로 자체 강화
- PTNS vs TTNS vs TIS 침습도·깊이 비교

### STEP 3 ✅ SNM / PTNS / TTNS 기존 기술 분석

**핵심 내용 (SNM → PTNS → TTNS 순서):**

**SNM:**
- 수술 이식형 (InterStim, FDA 1997) / S3 신경근 직접 지속 자극
- Bycroft 2020 NMA: BoNT-A > SNM > PTNS (효능) / PTNS 안전성 1위
- 한계: 수술·비용(~3만달러)·배터리 교체·장기 실패율 20~40%

**PTNS:**
- FDA 승인 2000년 (Urgent PC) / 바늘 삽입 / 주 1회 12주
- 3단계 기전 (Al-Danakh 2022): Aβ → S2-S4 → GABA → PMC
- SUmiT Trial (Peters 2010): 54.5% vs 20.9% (p<0.001)
- OrBIT Trial: PTNS = Tolterodine (동등 효능)
- STEP Study: 3년 유지율 77%
- 한계: 침습·병원 의존·파라미터 비표준화

**TTNS:**
- 피부 전극 / 비침습 / 자가 사용 가능 / FDA 미승인
- 2024 RCT (n=44): PTNS = TTNS (동등 효능) → 비침습화 근거
- 2024 TTNS+Mirabegron (n=40): 병용 > 단독 → TIS+약물 병용 설계 근거
- Muñoz 2026 (n=20): TTNS → 부교감 활성(HRV) → ECG 측정 가능
- 최적 파라미터: 10 Hz > 20 Hz (13 RCT, n=972)
- 한계: 자극 깊이·선택성 부족

**통합 논문 테이블:**
| 논문 | 핵심 결과 | 우리 연구 연관성 |
|------|---------|----------------|
| Bycroft 2020 | BoNT-A>SNM>PTNS / PTNS 안전 1위 | TIS 임상 위치 |
| Peters 2010 (SUmiT) | PTNS 54.5% vs Sham 20.9% | PTNS 대조군 근거 |
| Al-Danakh 2022 | Aβ→S2-S4→GABA 3단계 | TIS 동일 기전 |
| 2024 PTNS vs TTNS | PTNS = TTNS | 비침습화 근거 |
| 2024 TTNS+Mirabegron | 병용 > 단독 | TIS+약물 병용 |
| Muñoz 2026 | TTNS → 부교감↑ (HRV) | ECG/HRV 측정 |
