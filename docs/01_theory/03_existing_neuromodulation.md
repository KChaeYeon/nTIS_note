# SNM / PTNS / TTNS 기존 기술 분석

## 학습 목표

OAB 신경조절 치료 3가지의 기전·임상 근거·한계를 선행논문 기반으로 분석하고,  
TIS 연구의 필요성을 도출한다.

---

## 전체 그림

```
OAB 3차 치료 신경조절 3형제

SNM  ── 허리 수술 → S3 신경근 직접 자극           (이식형, 침습 최강)
PTNS ── 발목 바늘 → 경골신경 → S2-S4 억제        (반침습, FDA 승인)
TTNS ── 발목 전극 → 경골신경 → S2-S4 억제        (비침습, 자가 사용)

                ↓ 공통 한계 ↓
    침습 or 자극 깊이 부족 or 수술 비용·합병증

                ↓ 해결책 ↓
    TIS: 비침습 + 깊은 표적화 + 선택성
```

---

## 1. SNM (Sacral Neuromodulation) — 천수 신경 조절

### 어떻게 하나?

```
신체 삽입 위치 모식도:

        [뇌]
          │
        [척수]
          │
     허리 S3 레벨 ← 전극 삽입 위치
     ┌────┴─────┐
     │  전극    │──── 리드선
     └──────────┘         │
                    ┌─────┴──────┐
                    │  배터리    │ ← 엉덩이 위쪽 피하 이식
                    │ (IPG)      │   (약 성인 손바닥 크기)
                    └────────────┘
                          │
                    환자 리모컨으로 ON/OFF 조절

수술로 척수 S3 신경근 옆에 전극 이식

     허리 (S3 레벨)
          │
    ┌─────┴──────┐
    │   전극 이식 │ ← 척추 옆 천공(Foramen)으로 삽입
    └─────┬──────┘
          │
    피하에 배터리 삽입 (엉덩이 위쪽)
    (InterStim, Medtronic — FDA 승인 1997)
          │
    지속적 전기 자극
    → S3 신경 → 방광·요도·골반저 조절

수술 2단계:
  Step 1: 시험 자극 (1~2주) → 효과 50% 이상 개선 확인
  Step 2: 영구 이식 (전신마취)
```

### PTNS/TTNS와 무엇이 다른가?

```
PTNS/TTNS: 경골신경 → 척수 S2-S4 간접 억제 (구심성 경로 이용)
SNM:       S3 신경근 직접 지속 자극 (원심성 + 구심성 동시)

→ SNM이 신경에 더 직접적 → 효과 더 강함
→ 대신 수술 필요
```

### 임상 근거

**네트워크 메타분석** *(Bycroft et al. 2020, Toxins)*

3가지 OAB 3차 치료 RCT 종합 비교:

| 치료 | 효능 순위 | 부작용 |
|------|---------|--------|
| **BoNT-A (보툭스)** | 1위 (최강) | 요폐 20~30% |
| **SNM** | 2위 | 수술 합병증 |
| **PTNS** | 3위 | **최소 (안전성 1위)** |
| Sham | 최하 | - |

```
핵심 insight (Bycroft 2020):
  AUA 가이드라인 → 3치료 모두 3차 치료 동등 권고
  효능 최강 = BoNT-A (요폐 부작용)
  안전성 최강 = PTNS
  → TIS가 PTNS 안전성 + 더 나은 효능 → 판도 바꿀 가능성
```

**SNM 장기 결과:**
- 5년 추적 (Siegel 2000): 요실금 완전 소실 47%, 50% 이상 개선 72%
- 적응증 확대: OAB → 비폐색성 요저류 → 변실금 (FDA 확대)

### SNM 한계

```
한계 1: 수술 필요
  전신마취, 입원, 감염·이식 실패 위험

한계 2: 비용
  장비 비용 ~3만 달러 (미국 기준)
  한국 보험 적용 제한적

한계 3: 배터리 교체
  5~7년마다 재수술

한계 4: MRI 제한
  구형 기기 MRI 촬영 불가
  (최신 MRI-conditional 기기 일부 가능)

한계 5: 장기 실패율
  ~20~40% 장기 효과 소실
```

## 2. PTNS (Percutaneous Tibial Nerve Stimulation) — 경피 경골신경 자극

### 어떻게 하나?

```
장비: Urgent PC (Cogentix Medical) — FDA 승인 2000년

     발목 안쪽
        │
   ┌────┴────┐
   │  바늘   │ ← 34 gauge 침 (얇은 바늘)
   │  삽입   │   내과(내측 복숭아뼈)에서 5 cm 위
   └────┬────┘
        │
   경골신경 근처 (~3~5 mm)
        │
   전기 자극 전달
   (20 Hz, 200 μs, 0.5~9 mA)
        │
   30분 치료

치료 일정: 주 1회 × 12주 → 유지 치료 (월 1회)
```

### 작용 기전 — 3단계 모델

*(Al-Danakh et al. 2022, Parkinson's Disease)*

```
1단계 (말초 → 척수):
  경골신경 Aβ 구심성 섬유 자극
       ↓
  척수 S2-S4 후각 도달
       ↓
  방광 구심성 신호 억제 (Gate control 유사)

2단계 (척수):
  GABAergic interneuron 활성화
  (GABA = 억제성 신경전달물질)
       ↓
  배뇨 척수 반사 회로 억제
       ↓
  Detrusor Overactivity 감소

3단계 (중추):
  억제 신호 상행 → PMC → 대뇌 피질
       ↓
  방광 과민성 하향 조절

신경섬유 선택성:
  10~20 Hz → Aβ 섬유 우선 활성화 → 무통 억제
  >50 Hz   → C섬유 활성화 위험 → 역효과 가능
```

### PTNS 핵심 임상 근거 (3대 RCT)

**① SUmiT Trial** *(Peters et al. 2010, J Urol)* — FDA 승인 근거

```
설계: PTNS vs Sham (n=220, 12주)

결과:
  치료 반응률: PTNS 54.5% vs Sham 20.9% (p<0.001)
  빈뇨 감소: -2.4회/day
  절박뇨 개선: 유의미

→ PTNS 효과가 위약 대비 명확히 입증된 핵심 RCT
→ FDA 3차 치료 승인 근거
```

**② OrBIT Trial** *(Finazzi-Agrò et al. 2010)*

```
설계: PTNS vs 항무스카린제(Tolterodine) 비교

결과:
  12주 후 반응률: 두 군 간 차이 없음 (동등)
  부작용: PTNS가 훨씬 적음

→ 신경조절이 약물의 유효한 대안
→ 부작용 면에서 PTNS 우위
```

**③ STEP Study** *(MacDiarmid et al. 2010)*

```
설계: 12주 치료 완료 후 3년 장기 추적

결과:
  월 1회 유지 치료 시 효과 유지율: 77%
  유지 치료 중단 시 효과 소실

→ 장기 유지 치료의 필요성 + 가능성 동시 확인
```

**임상 반응률 범위:** 60~71% (복수 연구 종합, Al-Danakh 2022)

### PTNS 한계

```
한계 1: 침습성
  바늘 삽입 → 감염·출혈·통증
  항응고제 복용 환자 사용 불가

한계 2: 병원 의존성
  주 1회 × 12주 = 12번 병원 방문
  유지 치료: 평생 월 1회 내원

한계 3: 반응률 한계
  ~45%는 효과 없음

한계 4: 파라미터 비표준화
  최적 주파수·전류 연구마다 달라 불명확
  (Al-Danakh 2022 한계점으로 명시)
```

## 3. TTNS (Transcutaneous Tibial Nerve Stimulation) — 경피부 경골신경 자극

### PTNS vs TTNS vs TIS 단면도 비교

```
── PTNS (경피 바늘) ────────────────────────────
  발목 안쪽 단면:

  [지면 전극]
       │
  ─────┼──────────────────  ← 피부 표면
       │
  ─────┼──────────────────  ← 피하 지방
       │
     [바늘 끝] ←──────────  ← 경골신경 바로 옆 (~3~5 mm)
       │                       직접 전류 전달
  ──────────────────────────

  장점: 신경 바로 옆 → 정확한 자극
  단점: 바늘 삽입 → 침습, 병원 방문 필요

── TTNS (경피부 표면 전극) ──────────────────────
  발목 안쪽 단면:

  [전극A]        [전극B]
     │                │
  ───┼────────────────┼───  ← 피부 표면 (전류 일부 흡수)
     │ ~~~~~~~~~~~~~~│
  ───┼────────────────┼───  ← 피하 지방 (전류 손실)
     │ ~~~~~~~~~~~~~~│
     └───────┬────────┘
             ▼
      경골신경 (전류 일부 도달)

  장점: 비침습, 자가 사용 가능
  단점: 전류 손실 → 자극 깊이·재현성 낮음

── TIS (간섭 집속, 우리 연구) ───────────────────
  발목 주변 전극 4개:

  [전극 A1] ~~ f1=2000 Hz ~~┐
                              ├─→ 경골신경 위치에서
  [전극 A2] ~~ f1=2000 Hz ~~┘   f1-f2 = 20 Hz envelope 생성!
                              
  [전극 B1] ~~ f2=2020 Hz ~~┐
                              ├─→ (두 주파수 간섭)
  [전극 B2] ~~ f2=2020 Hz ~~┘

  단면도:
  [A1]                [A2]
    │   고주파 전류    │
  ──┼──────────────────┼──  ← 피부 (고주파 → 따끔 없음)
    │  ↘            ↙ │
  ──┼────────✕─────────┼──  ← 간섭 발생 지점 = 경골신경!
    │  ↗            ↖ │        (저주파 envelope만 생성)
  [B1]                [B2]

  장점: 비침습 + 피부 안전 + 경골신경 깊이 선택적 자극
```

### PTNS와의 차이 요약

```
PTNS: 바늘 → 피부 뚫고 신경 바로 옆 자극
TTNS: 피부 표면 전극 → 피부/지방 통과 → 신경 간접 자극
TIS:  전극 4개 간섭 → 심부에서만 저주파 생성 → 표면 안전

장점: 바늘 없음 → 자가 사용 가능, 가정 치료 가능 (TTNS/TIS 공통)
TIS 추가 장점: 선택성 + 깊이 동시 달성
```

### TTNS 핵심 임상 근거

**① PTNS vs TTNS 직접 비교 RCT** *(2024, Basic and Clinical Neuroscience)*

```
설계: OAB 환자 44명, 주 3회 × 4주
  PTNS군 (n=22) vs TTNS군 (n=22)

결과:
  두 군 모두 OABSS, I-QOL 유의미 개선 (p<0.05)
  군 간 차이: p>0.05 → 동등

핵심: PTNS = TTNS 효과 동등
→ 침습적 바늘 없이 동일 효과 달성 가능
→ 비침습화 추세의 직접 근거
```

**② TTNS + Mirabegron 병용** *(2024, Scientific Reports, n=40)*

```
설계:
  그룹 A: TTNS + Mirabegron 50 mg/일 (병용)
  그룹 B: Mirabegron 50 mg/일 (단독)
  기간: 주 2회 × 3개월

결과:
  병용군 > 단독군 (OAB-q bother, QoL 모두 유의미 우수, p<0.05)
  추가 부작용 없음

이유: 기전이 다른 두 치료의 상가(additive) 효과
  TTNS → 중추 신경 회로 억제 (S2-S4 GABA)
  Mirabegron → 방광 근육 β3 수용체 이완
  → 서로 다른 경로를 동시 타겟

→ TIS + 약물 병용 연구 설계의 근거 논문
```

**③ TTNS 자율신경계 효과** *(Muñoz et al. 2026, Neurourology and Urodynamics)*

```
설계: 건강인 20명, TTNS 10분, HRV 측정
  (기저 → 자극 → 회복 3단계)

결과:
  자극 중 부교감 지표 증가:
    RMSSD ↑ (부교감 활성)
    pNN50 ↑ (부교감 활성)
  자극 후 회복기: 원상 복귀

의미:
  TTNS는 방광 반사 억제만이 아니라
  자율신경계 전반을 조절
  → OAB의 자율신경 불균형 교정 가능성

우리 연구 적용:
  연구팀 보유 ECG/HRV 장비 →
  Rat TIS 실험에 ECG 동시 기록 추가 가능
  → 방광 효과 + 자율신경 효과 동시 측정
  → 논문 contribution 확장
```

**④ 최적 파라미터 메타분석** *(13개 RCT, n=972)*

```
결론:
  주파수: 10 Hz > 20 Hz (더 효과적)
  펄스폭: 200 μs (표준)
  → Al-Danakh 2022의 Aβ 섬유 선택 기전과 일치
     (10~20 Hz → Aβ 우선 활성화)
```

### TTNS 한계

```
한계 1: 자극 깊이 한계
  피부/지방층 통과 → 전류 손실
  신경 도달 전류량 불확실 → 재현성 낮음

한계 2: 선택성 부족
  표면 전극 → 주변 근육·피부 동시 자극
  → 불쾌감, 근육 수축

한계 3: FDA 미승인
  PTNS 2000년 승인, TTNS는 미승인
  (대규모 표준화 RCT 부족)

한계 4: 파라미터 비표준화
  연구마다 주파수·전류·전극 위치 달라
  직접 비교·메타분석 어려움
```

## 4. 세 기술 종합 비교

| | SNM | PTNS | TTNS |
|--|-----|------|------|
| 침습도 | 높음 (수술) | 중간 (바늘) | 없음 |
| 자극 위치 | S3 신경근 (직접) | 경골신경 (간접) | 경골신경 (간접) |
| 효능 (NMA) | 높음 (2위) | 중간 (3위) | 중간~ |
| 지속성 | 지속 자극 | 월 1회 유지 필요 | 월 1회 유지 필요 |
| 자가 사용 | 가능 (리모컨) | 불가 | 가능 |
| 비용 | 매우 높음 | 중간 | 낮음 |
| 부작용 | 높음 (수술) | 낮음 | 최소 |
| FDA 승인 | ✅ 1997 | ✅ 2000 | ❌ |
| 자율신경 효과 | 있음 | 간접적 | ✅ 확인 (2026) |

## 6. 공통 한계 → TIS의 필요성

```
SNM  한계: 수술 + 비용 + 합병증
PTNS 한계: 침습 (바늘) + 병원 의존
TTNS 한계: 자극 깊이 부족 + 선택성 낮음 + FDA 미승인

↓ 공통 미충족 수요 (Unmet Need) ↓

  ✅ 비침습 (바늘·수술 없이)
  ✅ 깊은 경골신경 표적화
  ✅ 선택적 자극 (주변 조직 최소화)
  ✅ 자가 사용 가능
  ✅ PTNS/TTNS 이상 효능

↓ TIS의 이론적 해결책 ↓

  피부 전극 4개
       ↓
  고주파 2쌍 (2 kHz + 2.02 kHz) 동시 인가
       ↓
  피부: 고주파 → 불쾌감 없음 (안전)
       ↓
  경골신경 위치: 두 주파수 간섭 → 20 Hz envelope 생성
       ↓
  비침습 + 깊이 + 선택성 동시 달성
  → PTNS 동등 이상 효과 이론적 가능
```

## 개념 요약 & 지식 맵

```
OAB 신경조절 기술 발전 흐름

SNM (1997) ──── 수술·이식 → 효과 강 / 침습·비용 문제
    │
PTNS (2000) ─── 바늘·외래 → 안전 / 병원 의존 문제
    │
TTNS (진행 중) ─ 전극·가정 → 비침습 / 깊이·선택성 문제
    │
TIS (우리 연구) ─ 간섭 집속 → 비침습 + 깊이 + 선택성

공통 신경 경로:
  경골신경(L4-S3) → 척수 S2-S4
  → GABAergic 억제 → 방광 부교감 차단 → OAB 개선

TIS 차별화 포인트:
  고주파 피부 통과 + 심부에서 저주파 envelope 생성
  = 표면 안전성 + 심부 효과 동시 달성
```
