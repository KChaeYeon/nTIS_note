## 1. 왜 이 기술이 필요한가?

방광 과활성(OAB)의 원인은 **발목 안쪽 경골신경(tibial nerve)** 이 과도하게 활성화되는 것입니다.

| 방법 | 침습? | 문제점 |
|------|-------|--------|
| PTNS (바늘 전극) | 최소 침습 | 병원 방문, 바늘 공포 |
| TTNS (피부 패치) | 비침습 | 심부 신경까지 안 닿음 |
| SNM (수술) | 침습 | 비용·감염 위험 |

**TIS는 이 딜레마를 "주파수 트릭"으로 해결합니다.**

## 2. 전자기장 기초

### 2.1 전기장 E (Electric Field)

**비유: 태양 주변의 중력장** — 전하가 있으면 주변 공간에 장(field)이 형성되어, 다른 전하가 직접 닿지 않아도 힘을 받습니다.

```
E = F / q        [단위: V/m]
```

- F : 전하에 작용하는 힘 [N]
- q : 전하량 [C]
- 방향: 양(+) → 음(-) 전하

**전압 φ와 전기장의 관계 — 산의 등고선 비유:**

전압은 "산의 높이", 전기장은 "경사도(기울기)"입니다.

```
E = −∇φ = −( ∂φ/∂x · x̂  +  ∂φ/∂y · ŷ  +  ∂φ/∂z · ẑ )
```

음의 부호(−)는 전기장이 전압이 **높은 곳 → 낮은 곳** 방향임을 의미합니다.

숫자 예시 — 전극 A (+1 V), 전극 B (0 V), 거리 5 mm:

```
|E| = Δφ / Δz = 1 V / 0.005 m = 200 V/m
```

---

### 2.2 맥스웰 방정식 (4개의 전자기 법칙)

```
∇·E = ρ/ε₀                         (전하 → 전기장 발산)
∇·B = 0                              (자기 홀극 없음)
∇×E = −∂B/∂t                        (변하는 B → E 생성)
∇×B = μ₀J + μ₀ε₀·(∂E/∂t)           (전류 + 변하는 E → B 생성)
```

이 4개를 모두 연립하면 전자기파(빛)가 나옵니다. 하지만 계산이 극도로 복잡합니다.

---

### 2.3 단순화 — 자기장을 무시해도 된다

TIS 자극 주파수 2 kHz에서 전자기파의 파장:

```
λ = c / f = (3×10⁸ m/s) / 2000 Hz = 150,000 m = 150 km
```

인체 크기는 약 0.3 m. 파장이 인체보다 **50만 배** 큽니다.

```
인체 크기 / λ = 0.3 m / 150,000 m ≈ 2×10⁻⁶  ≪  1
```

**비유:** 파장 100 m 파도가 10 m 수영장에 들어오면 파도의 굴곡이 안 느껴집니다. 마찬가지로 150 km 파장은 인체 내에서 파동 특성이 나타나지 않습니다.

따라서 **준정적(Electroquasistatic) 가정** 성립:

```
∇×E ≈ 0   →   E = −∇φ
```

복잡한 벡터 3개(Ex, Ey, Ez) 대신 **스칼라 1개(φ)만** 구하면 됩니다.

## 3. 몸 속 전류와 Laplace 방정식

### 3.1 조직별 전도도 σ (전류 통과 능력)

**비유: 도로의 차선 수** — σ가 클수록 전류가 잘 흐릅니다.

| 조직 | σ (S/m) @ 2 kHz | 비유 |
|------|----------------|------|
| 근육 | 0.355 | 8차선 고속도로 |
| 경골신경 | 0.265 | 4차선 도로 |
| 피부 | 0.17 | 2차선 도로 |
| 지방 | 0.057 | 비포장 산길 |
| 뼈 | 0.0035 | 막힌 길 |

뼈는 근육보다 전류를 **100배** 덜 통과시킵니다.

---

### 3.2 Laplace 방정식 유도

**전하 보존 법칙:** 어떤 점에서도 들어온 전류 = 나간 전류

```
∇·J = 0
```

미시적 옴의 법칙 (J = σE = −σ∇φ) 을 대입:

```
∇·(σ ∇φ) = 0          ← Laplace 방정식
```

- ∇φ : 전압의 경사도 (어느 방향으로 전류가 흐르려 하는가)
- σ × ∇φ : 실제 전류 밀도 (파이프 굵기 × 압력 차)
- ∇·[...] = 0 : 모든 점에서 들어온 전류 = 나간 전류

**경계 조건:**

| 경계 | 조건 |
|------|------|
| 능동 전극 | φ = 1 V (Dirichlet) |
| 기준 전극 | φ = 0 V (Dirichlet) |
| 피부 바깥 | J·n̂ = 0 (Neumann, 전류 밖으로 안 나감) |

---

### 3.3 고주파에서 유전율(ε)도 고려

2 kHz에서는 세포막 커패시터를 통한 변위 전류도 무시할 수 없습니다:

```
J = (σ + jωε)·E

→ 일반화 Laplace:  ∇·[(σ + jωε)∇φ] = 0
```

단, 2 kHz에서 전도 전류 대 변위 전류 비율:

```
σ / (ω·ε) = 0.170 / 0.00011 ≈ 1,545
```

변위 전류가 전도 전류의 약 1/1,545 수준 → 많은 경우 ∇·(σ∇φ) = 0 으로 단순화해서 사용합니다.

---

### 3.4 FEM (유한요소법) — 컴퓨터로 풀기

복잡한 인체 형상에 Laplace 방정식을 손으로 풀 수 없으므로 FEM을 사용합니다.

```
실제 발목 단면          FEM 메시 분해
    ┌──╮                ┌─┬─┬─┬─┐
    │  │      →         ├─┼─┼─┼─┤
    │  ╰──┐             ├─┼─┼─┼─┤
    └─────┘             └─┴─┴─┴─┘
  복잡한 형태         수만 개의 작은 삼각형
```

각 삼각형 내에서 φ를 단순 다항식으로 근사 → 모든 삼각형의 연립방정식 동시 풀기 → 전압 분포 φ(r) 완성

전기장: E(r) = −∇φ(r)

전류 정규화 (FEM은 1 V 기준 → 실제 1 mA 기준으로 변환):

```
E_norm = E(1V) / I_computed  ×  1 mA
```

**선형 중첩으로 계산량 폭발 방지:**

24개 전극 중 기준 전극(Ref) 하나를 고정하고 24회만 FEM 실행 후:

```
E(A→B) = E(A−Ref) − E(B−Ref)
```

이유: FEM 방정식 [K]{V} = {f}가 선형 시스템이므로, 같은 기준점을 공유한 두 해를 빼면 기준점 기여가 정확히 소거됩니다.

→ 24회 FEM으로 C(24,4) = 10,626 가지 4전극 조합 전부 계산 가능

## 4. TIS 원리 — 맥놀이(Beat) 현상

### 4.1 소리로 먼저 이해하기

피아노 건반 두 개를 동시에 칩니다:
- 건반 A: 440 Hz (라 음)
- 건반 B: 442 Hz (조금 다른 라 음)

"웅~웅~" 하는 2 Hz 떨림(맥놀이)이 들립니다.

```
f_beat = |f₂ − f₁| = |442 − 440| = 2 Hz
```

TIS는 동일한 원리를 전기장에 적용합니다.

---

### 4.2 수식 유도 — 1D, 같은 진폭 (E₁ = E₂ = E₀)

```
E(t) = E₀·cos(ω₁t) + E₀·cos(ω₂t)
```

삼각함수 적화 공식 (cos A + cos B = 2·cos((A−B)/2)·cos((A+B)/2)) 적용:

```
          ┌──────────────────────┐   ┌────────────────────┐
E(t) = 2E₀│cos(π·Δf·t)          │ × │cos(2π·f̄·t)         │
          └──────────────────────┘   └────────────────────┘
           포락선 (저주파, 실제 자극)   캐리어 (고주파, 통과용)

  Δf = f₂ − f₁       (예: 2020 − 2000 = 20 Hz)
  f̄  = (f₁ + f₂)/2   (예: (2000+2020)/2 = 2010 Hz)
```

| 항 | 주파수 | 뉴런 반응 |
|----|--------|----------|
| 캐리어 cos(2π·f̄·t) | f̄ ≈ 2010 Hz | 반응 없음 |
| 포락선 2E₀·cos(π·Δf·t) | Δf = 20 Hz | **반응함** |

**왜 뉴런은 캐리어(2 kHz)를 무시하는가?**

신경막의 RC 시상수:

```
τ = RC ≈ 1~10 ms

kHz 신호 주기:  T = 1/2000 Hz = 0.5 ms  ≪  τ  → 반응 없음
Beat 주기:      T = 1/20 Hz  = 50 ms   ≫  τ  → 발화!
```

**비유:** 배를 흔들려면 파도 주기가 배의 자연 진동 주기와 비슷해야 합니다. 너무 빠른 잔물결(kHz)은 배를 흔들지 못하고, 천천히 오는 너울(수십 Hz)이 배를 흔듭니다.

---

### 4.3 일반적 경우 (E₁ ≠ E₂)

진폭이 다를 때 포락선이 0까지 내려가지 않습니다:

```
E_max = |E₁ + E₂|
E_min = | |E₁| − |E₂| |

E_TI = E_max − E_min
     = 2 · min(|E₁|, |E₂|)
     = | |E₁+E₂| − |E₁−E₂| |
```

숫자 예시 (E₁ = 1.5 V/m, E₂ = 0.5 V/m):

```
E_TI = 2 × min(1.5, 0.5) = 2 × 0.5 = 1.0 V/m

포락선:  최대 = 1.5 + 0.5 = 2.0 V/m
         최소 = 1.5 − 0.5 = 1.0 V/m
         진동 폭 = 1.0 V/m  ← 이것이 자극 세기
```

**교훈:** 자극 세기는 **더 약한 채널**이 결정합니다. 두 채널을 신경 위치에서 같은 세기로 최적화해야 합니다.

**비유: 줄다리기**

```
10명 vs 10명: 줄이 양쪽으로 최대한 당겨짐 → 최대 진폭
10명 vs  5명: 더 강한 팀 쪽으로 치우침 → 5명 힘만큼만 진동
```

---

### 4.4 심부 선택적 자극 원리

**비유: 두 손전등 교차**

```
손전등 A ─────────────────►
                   ✕  ← 교차점만 훨씬 밝음
손전등 B ─────────────────►
```

발목에서 위치별 E_TI 크기:

```
전극 A 근처:      E₁ ≫ E₂  →  E_TI = 2·E₂  (작음)
★ 경골신경 위치:  E₁ ≈ E₂  →  E_TI = 2·E₁  (최대!)
전극 B 근처:      E₂ ≫ E₁  →  E_TI = 2·E₁  (작음)
```

## 5. 3차원 전기장 — 벡터와 타원 궤적

### 5.1 실제 조직의 3D 전기장

공간의 각 점에서 전기장은 방향이 있는 3차원 벡터입니다:

```
E⃗(t) = E⃗₁·cos(ω₁t) + E⃗₂·cos(ω₂t)
```

Phasor 표현 (복소수):

```
E⃗₁ = Re( Ẽ₁ · e^(jω₁t) )
E⃗₂ = Re( Ẽ₂ · e^(jω₂t) )
```

시간에 따라 벡터 끝이 그리는 궤적: **타원**

```
     E_y
      ↑
      │    ╭────────╮
      │   ╱  타원   ╲
      │  │     ●     │  ← E⃗(t)의 끝이 타원을 따라 이동
      │   ╲         ╱
      │    ╰────────╯
      └──────────────→ E_x

  장반경 a = 가장 강한 자극 방향
  단반경 b = 수직 방향
  b = 0이면 선형 (한 방향만 자극)
  a = b이면 원형 (모든 방향 균등 자극)
```

타원의 두 축 크기는 행렬 [Ẽ₁, Ẽ₂] 의 **특이값(singular value)**으로 결정됩니다.

---

### 5.2 2-phase vs 3-phase의 타원 비교

```
2-phase TIS:              3-phase TIS:
─────────                 ─────────
E_y ↑                    E_y ↑
    │ ╱ (선형)                │   ╭──╮
    │╱                        │  ╱    ╲
    ●─────→ E_x               │ │  ●  │
                               │  ╲    ╱
선형에 가까움                  │   ╰──╯
(한 방향 집중)                  └────────→ E_x
                           원형에 가까움
                           (3D 전방향 균등)
```

## 6. 2-phase TIS

### 전극 구성 및 조향

```
  (+f₁) ────────────────── (−f₁)   ← 전극쌍 1
  (+f₂) ────────────────── (−f₂)   ← 전극쌍 2
```

조향 원리 — 두 쌍의 진폭 비율 조절:

```
I₁ = I₀ + ΔI     I₂ = I₀ − ΔI

  ΔI = 0  → 표적이 중앙
  ΔI > 0  → 표적이 전극쌍 1 쪽으로 이동
  ΔI < 0  → 표적이 전극쌍 2 쪽으로 이동
```

**한계:** 타원이 2D 평면 내에만 존재 → 깊이(상하) 방향 조향 불가

---

## 7. 3-phase TIS

### 7.1 120° 위상 분리

```
Iₐ = I · cos(ωt + 0°)
I_b = I · cos(ωt + 120°)
I_c = I · cos(ωt + 240°)
```

**세 전류의 합이 항상 0:**

```
Iₐ + I_b + I_c = I · [cos(0°) + cos(120°) + cos(240°)]
               = I · [1 + (−1/2) + (−1/2)]
               = 0
```

→ Kirchhoff 전류 법칙 자동 만족 (GND 전극 불필요!)

이것은 가정용 3상 교류(220 V / 380 V)와 동일한 원리입니다.

---

### 7.2 Phasor 표현

```
  허수축 (Im)
     ↑   ● I_b (120°)
     │  ╱
     │ ╱  120°
     │╱  ╱120°
   ──●──────────→ 실수축 (Re)
   Iₐ│╲ 120°
     │ ╲
     │  ╲● I_c (240°)

  세 페이저가 120°씩 간격 → 3D 공간에서 타원 궤적
```

---

### 7.3 2-phase vs 3-phase 비교

| 항목 | 2-phase TIS | 3-phase TIS |
|------|-------------|-------------|
| 전극 수 | 4개 (2쌍) | 3개 |
| 자극 평면 | 2D | 3D |
| 조향 방향 | 좌/우만 | 상/하/좌/우/앞/뒤 전방향 |
| AM_max 계산 | 상대적 단순 | SVD 필요 |
| 전류 합 | 별도 GND 필요 | 자동으로 0 |

## 8. AM_max — 최대 자극 방향 계산

### 8.1 문제 정의

어떤 방향 단위벡터 n̂으로 전기장을 투영할 때 자극이 최대가 되는가?

```
AM_max = max  AM(n̂)
         ‖n̂‖=1
```

### 8.2 행렬 구성

COMSOL에서 export한 복소 phasor Ẽ₁, Ẽ₂ ∈ ℂ³ 를 실수 행렬로 쌓습니다:

```
M = [ Re(Ẽ₁),  Im(Ẽ₁),  Re(Ẽ₂),  Im(Ẽ₂) ]   ∈ ℝ³ˣ⁴
```

### 8.3 SVD 분해

```
M = U · Σ · Vᵀ

U ∈ ℝ³ˣ³   : 자극 방향 행렬 (직교)
Σ = diag(σ₁, σ₂, σ₃)  : 특이값  (σ₁ ≥ σ₂ ≥ σ₃ ≥ 0)
```

**결과:**

```
AM_max = 2 · σ₁           (최대 특이값 × 2)
n̂*     = u₁               (U의 첫 번째 열 = 가장 강한 자극 방향)
```

인수 2의 이유: 타원 장반경 = σ₁, 피크-투-피크 진폭 = 2·σ₁

---

### 8.4 Lagrange 승수법 — 왜 SVD가 답인가?

*(송솔웅이 팀 내부에서 유도한 핵심 결과)*

**문제:**

```
maximize:   E⃗(t) · n̂
subject to: ‖n̂‖² = 1
```

**Lagrangian 구성:**

```
ℒ(n̂, λ) = E⃗(t)·n̂  −  λ·(‖n̂‖² − 1)
```

정류 조건 ∇_n̂ ℒ = 0 을 전개하면:

```
MᵀM · n̂ = λ · n̂         ← MᵀM의 고유값 문제
```

SVD에서 MᵀM = V·Σ²·Vᵀ 이므로, 고유값 = σᵢ².
최대 고유값 = σ₁²  →  최대 투영 = σ₁  →  **AM_max = 2·σ₁  ■**

## 9. COMSOL 시뮬레이션 파이프라인

팀(송솔웅 + 김대용)이 실제 사용하는 전체 흐름:

### Step 1 — 조직 모델 구축

```
MRI/CT 영상
    ↓ 세그멘테이션 (피부/지방/근육/신경/뼈)
    ↓ STL/mesh 생성
    ↓ COMSOL import → 3D 해부학 모델
```

### Step 2 — 물질 속성 설정

각 조직에 전도도 σ 할당:

| 조직 | σ (S/m) |
|------|---------|
| 피부 | 0.17 |
| 지방 | 0.057 |
| 근육 | 0.355 |
| 경골신경 | 0.265 |
| 뼈 | 0.0035 |

### Step 3 — 주파수 도메인 해석 (2회)

지배 방정식:

```
∇·(σ∇V) = 0     (quasi-static 근사)
```

```
f₁으로 1회 실행  →  export: Ẽ₁(x, y, z)   [복소수 phasor]
f₂로 1회 실행   →  export: Ẽ₂(x, y, z)   [복소수 phasor]
```

### Step 4 — MATLAB에서 AM_max 계산

```matlab
for each point r:
    M = [real(E_f1), imag(E_f1), ...
         real(E_f2), imag(E_f2)];   % 3×4 행렬
    [U, S, V] = svd(M);
    AM_max(r) = 2 * S(1,1);          % 최대 특이값 × 2
    n_opt(r)  = U(:,1);              % 최적 자극 방향
end
```

### Step 5 — COMSOL 재import 및 시각화

```
AM_max(r) 를 공간 보간 함수로 load
    → 조직 모델 위에 색상맵 오버레이
    → 등값면(Isosurface), 단면도(Slice), 3D 산점도 생성
```

## 10. 집중도 지표 — Peak Ratio (PR)

**비유: 다트 점수판** — 화살이 정중앙(경골신경)에 몰릴수록 높은 점수

```
PR = E_TI(경골신경 영역 평균) / E_TI(전체 영역 평균)
```

**Kim 2023 (인체 FEM) 3단계 최적화 결과:**

```
단계 0 (최악 배치)         PR = 0.039
      ↓ 전극 위치 최적화
단계 1                     PR = 3.396   (+87배!)
      ↓ 전극 연결 방향 교차
단계 2                     PR = 3.788   (+12%)
      ↓ I₁:I₂ 비율 조정
단계 3 (최적)              PR = 4.206   (+11%)

교훈: 전극 위치가 결과의 90%를 결정한다!
```

## 12. 전체 수식 모음

| 번호 | 수식 | 의미 |
|------|------|------|
| 1 | E = −∇φ | Electroquasistatic 전기장 |
| 2 | λ = c/f = 150,000 m (@ 2 kHz) | EM 파장 |
| 3 | ∇·(σ∇φ) = 0 | Laplace 방정식 |
| 4 | J = (σ + jωε)·E | 전류 밀도 |
| 5 | E(t) = 2E₀·cos(π·Δf·t)·cos(2π·f̄·t) | TIS 신호 분해 |
| 6 | E_TI = 2·min(\|E₁\|, \|E₂\|) | TI 포락선 세기 |
| 7 | Iₐ + I_b + I_c = 0 | 3-phase 전류 균형 |
| 8 | M = U·Σ·Vᵀ,  AM_max = 2·σ₁ | SVD 기반 AM_max |
| 9 | PR = E_target / E_total | 집중도 지표 |
| 10 | f(z) = (1/rᵢ)·∂²Vₑ/∂z² | 신경 활성화 함수 |
