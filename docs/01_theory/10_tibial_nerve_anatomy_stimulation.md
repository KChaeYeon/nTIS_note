# STEP 2: 경골신경 해부학 & 자극 기전

## 학습 목표

경골신경의 해부학적 위치·내부 구조 → 전기자극이 신경을 활성화하는 원리 → 척수에서 방광 억제로 연결되는 경로까지 이해.

---

## 1. 경골신경 위치 & 주행 경로

```
         무릎 뒤쪽 (오금)
              │
         경골신경 시작
              │
        종아리 안쪽으로 내려옴
              │
         발목 안쪽 복숭아뼈
         (내과, Medial Malleolus)
              │
    ┌─────────┴─────────┐
    │                   │
 발바닥 신경          발뒤꿈치 신경
(족저신경)           (종골신경)
```

- **PTNS 자침 위치:** 발목 안쪽 복숭아뼈에서 손가락 3마디 위 (~5 cm)
- 좌골신경(Sciatic nerve)에서 갈라진 가지

---

## 2. 척수 뿌리 레벨

```
척수
 ├─ L4 ──┐
 ├─ L5   ├──► 경골신경 (Tibial nerve)
 ├─ S1   │    → 좌골신경(Sciatic)에서 분지
 ├─ S2   │
 └─ S3 ──┘
```

**핵심:** S2, S3 뿌리 포함 → 방광 조절 부교감신경 출발지 S2-S4와 동일 레벨 → 경골신경 자극이 방광에 영향을 주는 신경해부학적 근거.

---

## 3. 신경 내부 구조

```
         경골신경 단면
    ┌─────────────────────┐
    │  Epineurium         │ ← 신경 전체를 감싸는 외피
    │  ┌───┐  ┌───┐       │
    │  │ F │  │ F │  ...  │ ← Fascicle (신경다발)
    │  └───┘  └───┘       │
    │   각 다발 안에:       │
    │   ● ● ● ● ●         │ ← 개별 신경섬유 (Axon)
    └─────────────────────┘
```

| 구조 | 역할 |
|------|------|
| Epineurium | 신경 전체 보호 외피 |
| Perineurium | 각 다발 감싸는 막 |
| Fascicle | 기능별로 묶인 신경섬유 다발 |
| Axon | 실제 전기 신호 전달하는 선 |
| Myelin sheath | 축삭 감싸는 절연체 → 신호 빠르게 전달 |

---

## 4. 신경섬유 종류 — 자극 반응 순서

```
Aα ──── 굵고 빠름 (70-120 m/s) ── 근육 운동, 고유감각
Aβ ──── 중간     (30-70 m/s)  ── 촉각, 압각
Aδ ──── 얇음     (5-30 m/s)   ── 가벼운 통증, 온도
C  ──── 매우 얇음 (0.5-2 m/s)  ── 깊은 통증, 자율신경 (무수초)
```

- **굵을수록:** 자극 역치 낮음 = 적은 전류로 자극 가능
- **PTNS:** 주로 Aβ, Aδ 섬유 활성화

---

## 5. 전기자극 → 탈분극 → 활동전위 발생 기전

### 5-1. 안정막전위 (평소 상태)

```
세포 바깥:  Na+ 많음  (+)
            ━━━━━━━━━━━━  ← 세포막
세포 안쪽:  K+, 단백질 많음  (-)

전위차: 안쪽 -70 mV
```

### 5-2. 전기자극 → Na+ 채널 강제 개방

```
전극 (음극) 근처 전류 흐름
        ↓
세포막 바깥 전위 낮아짐 → 안팎 전위차 감소
        ↓
-70 mV → -55 mV (역치 도달)
        ↓
Na+ 채널 강제 개방
        ↓
Na+ 가 바깥에서 안으로 유입
        ↓
안쪽이 (+)로 역전 = 탈분극
```

> 외부 전기 자극이 본래 수용체 신호 역할을 대신해 Na+ 채널을 강제 개방하는 것.

### 5-3. 활동전위 전파 (도미노 원리)

```
자극 지점  위치 1    위치 2    위치 3
           [+++]  →  [---]  →  [---]   처음

           [재분극] → [+++]  →  [---]   이동

           [끝남]  → [재분극] → [+++]   계속 전파
```

탈분극된 부위(+)가 옆 부위(-)를 끌어당겨 인접 Na+ 채널 개방 → 연쇄 전파.

### 5-4. Myelin과 도약 전도 (Saltatory Conduction)

```
Myelin 없는 신경 (C섬유):
●─────────────────────→  느리게 모든 구간 전파

Myelin 있는 신경 (A섬유):
●══════╗  ╔══════╗
       ↓  ↓      ↓
      [+]        [+]    ← Ranvier node에서만 탈분극, 점프!
```

- Myelin = 절연체 → Ranvier node 틈새에서만 탈분극 발생
- 신호가 점프하며 이동 → 훨씬 빠른 전도 속도

## 6. 전기자극 핵심 파라미터

```
┌─────────────────────────────────────┐
│  펄스 하나의 모양:                   │
│                                     │
│  ┌──┐         ┌──┐                  │
│  │  │         │  │                  │
│──┘  └─────────┘  └──────            │
│  ←→  ←───────→                      │
│  PW      IPI                        │
└─────────────────────────────────────┘
PW  = Pulse Width (펄스 폭, μs)
IPI = Inter-Pulse Interval
주파수 = 1초에 몇 번 (Hz)
진폭   = 자극 강도 (mA, V)
```

| 파라미터 | PTNS 표준값 | 의미 |
|----------|------------|------|
| 주파수 | 20 Hz | 1초에 20번 자극 |
| 펄스폭 | 200 μs | 각 펄스 지속 시간 |
| 진폭 | 0.5~9 mA | 발가락 움직임 느낄 정도 |
| 시간 | 30분/회, 12주 | 표준 치료 기간 |

## 7. 척수에서 방광 억제로 연결되는 경로

```
경골신경 Aδ, Aβ 섬유 활성화
        ↓ 구심성 신호
척수 후각 S2-S4
        │
        ├─► GABAergic interneuron 활성화
        │   (GABA = 억제성 신경전달물질)
        │         ↓
        │   방광 부교감 신호 차단
        │   (배뇨근 수축 명령 억제)
        │
        └─► 상행 경로 → PAG → PMC 게이트 강화
                        (배뇨 억제 보강)
        ↓
방광 과활성 감소 → OAB 증상 개선
```

**근거:** Tai et al. — 쥐 S2 레벨에서 GABAergic interneuron이 경골신경 자극에 의한 방광 억제를 매개함을 확인.

## 8. 치료 효과의 전체 인과 사슬

```
[전기 자극 인가]
      ↓
경골신경 Na+ 채널 강제 개방
      ↓
탈분극 → 활동전위 발생
      ↓
도약 전도로 척수 방향 이동
      ↓
S2-S4 후각 도달
      ↓
GABAergic interneuron 활성화
      ↓
방광 부교감 흥분 신호 차단
      ↓
방광 과수축 억제
      ↓
OAB 증상 감소 (빈뇨·절박뇨 개선)
```

### 장기 효과 — 신경 가소성 (Neuroplasticity)

```
단기 효과 (자극 중):
  Na+ 채널 → 활동전위 → GABA 억제 → 방광 수축 즉시 감소

장기 효과 (12주 치료 후 수개월 지속):
  반복 자극 → 억제 회로 자체 강화 (신경 가소성)
  → 뇌-척수-방광 조절 회로 재편성
```

## 9. PTNS vs TTNS vs TIS 비교

```
PTNS (Percutaneous):
  발목 바늘 → 신경 직접 자극
  [전극]─────[바늘]───► 경골신경

TTNS (Transcutaneous):
  피부 표면 전극 → 피부·지방 통과 → 신경 도달
  [전극] ~피부~ ~지방~ ►경골신경

TIS (우리 연구):
  전극 4개 → 고주파 2쌍 간섭 → 경골신경 위치에서만 저주파 envelope
  [A1]  [A2]
    ↘  ↙
     ✕  ← 경골신경 위치에서 집속
    ↗  ↖
  [B1]  [B2]
```

| | PTNS | TTNS | TIS |
|-|------|------|-----|
| 침습도 | 침습 (바늘) | 비침습 | 비침습 |
| 자극 깊이 | 깊음 (신경 직접) | 얕음 | 깊음 (간섭 집속) |
| 피부 불편감 | 바늘 통증 | 따끔거림 | 적음 (고주파) |
| 자가 사용 | 불가 | 가능 | 가능 |

## 개념 요약

```
경골신경 (L4-S3)
    │
    ├─ 위치: 발목 안쪽 내과 5 cm 위
    ├─ 내부: Epineurium → Fascicle → Axon → Myelin
    ├─ 섬유: Aα > Aβ > Aδ > C (굵을수록 자극 쉬움)
    │
    ├─ 자극 기전:
    │   전류 → Na+ 채널 개방 → 탈분극 → 도약 전도 → 척수 전달
    │
    ├─ 방광 억제 경로:
    │   Aδ/Aβ 활성화 → 척수 S2-S4 → GABA interneuron
    │   → 방광 부교감 차단 → 과활성 감소
    │
    └─ 자극 방식 진화:
        PTNS (바늘) → TTNS (표면 전극) → TIS (간섭 집속, 비침습+깊이)
```
