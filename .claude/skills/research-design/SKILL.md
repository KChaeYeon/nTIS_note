# Research Design — nTIS 연구설계 스킬

## FINES 평가 실행 절차

### Step 1: 다관점 가설 수집 (GPT + Gemini 의무)

연구 후보 각각에 대해 CCG 실행:

```
oh-my-claudecode:ccg 프롬프트:
"다음 nTIS 연구 후보를 평가해주세요: [주제]
각각 답해주세요:
1. 이 연구의 가장 강력한 과학적 근거는?
2. 예상되는 가장 큰 방법론적 약점은?
3. 유사 분야(non-invasive brain stimulation, VNS 등)에서 참고할 방법론은?
4. 1.5년 내 완료 가능성 평가 (낙관/중립/비관)"
```

세 AI의 의견을 통합하여 FINES 채점에 반영.

### Step 2: FINES 채점

각 기준 1–5점, 반드시 근거 문장 포함:

**F: Feasibility (가중치 0.30)**
세부 항목:
- 보유 장비로 실험 가능한가? (TIS ✅, ECG/HRV ✅, Rat ✅, EEG ❌)
- 1.5년 내(2027-12) 논문 제출 가능한가?
- 피험자/동물 모집 현실적인가?
- 팀 기술 스택이 맞는가?

Kill-switch: 위 4개 중 하나라도 1점이면 F=1 → 즉시 탈락

**I: Impact (가중치 0.25)**
- IF ≥ 4 저널 투고 가능성
- 임상 번역 가능성 (환자에게 언제 도달 가능한가)
- 갭 크기 (🔴 > 🟡 > 🟢)

**N: Novelty (가중치 0.20)**
- 지난 12개월 PubMed + bioRxiv에서 동일 갭 논문 없음
- 선점 우위 확보 여부
- 재현 연구가 아닌 새로운 기여

**E: Expertise fit (가중치 0.15)**
- 연구자 현재 스킬: EEG 신호처리, MRI 분석, ML/DL, ECG/HRV, 생체신호처리
- 팀 스킬: TIS 자극기 운용, Rat 실험, FEM 모델링(팀에 있음)
- 6개월 내 습득 가능한 스킬은 +0.5점 허용

**S: Scalability (가중치 0.10)**
- 후속 논문 2편 이상 연결 가능한가?
- 그랜트(NRF, IITP 등) 연결 가능한가?
- 박사 학위 논문 챕터로 성장 가능한가?

**가중 합산:** F×0.30 + I×0.25 + N×0.20 + E×0.15 + S×0.10

## TIS 실험 프로토콜 템플릿

### 자극 파라미터 설계 기준

```
캐리어 주파수 (f1, f2):
- 현재 가장 많이 사용: 1 kHz, 2 kHz, 9 kHz
- 안전 기준: f ≥ 1 kHz (신경 직접 흥분 역치 이하)
- 참고: 뇌전증 연구(2025 Brain Stim) → 9 kHz > 1 kHz 시사 (미확정)

차이 주파수 (Δf):
- 목표 신경 리듬 = Δf 설정 원칙 (미검증 가설)
- 수면 방추파: Δf = 12–14 Hz
- 서파 (SO): Δf = 0.5–1 Hz
- 알파 리듬: Δf = 8–12 Hz
- 미주신경 HRV: Δf = 10–20 Hz (추정)

전극 배치:
- n-phase TIS: n쌍 전극 → 초점성·방향성 제어
- FEM 시뮬레이션으로 envelope 위치 사전 확인 필수
```

### FEM 모델링 체크리스트

```
소프트웨어: SimNIBS 4.x / ROAST / 자체 COMSOL
헤드 모델: MNI 표준 또는 개인화 T1-MRI 기반
도전율값: 뇌(0.275), 두개골(0.01), 두피(0.465) S/m
검증 기준:
  - envelope 피크가 목표 구조물 내에 위치
  - 피질 표면 최대 E-field ≤ 1 V/m (안전)
  - 초점성(focality): 반최대폭(FWHM) ≤ 2 cm
```

## 연구 질문 정제 (PICO/PECOS)

```
P (Population): 누구를 대상으로? (Rat Sprague-Dawley, 건강 성인, ...)
I (Intervention): 어떤 자극? (nTIS: f1=Xkz, f2=Ykz, Δf=ZHz, X분, n회)
C (Comparator): 비교군? (샴 자극 / taVNS / 기존 자극 방법)
O (Outcome): 무엇을 측정? (HRV RMSSD, EEG 방추파 밀도, 행동 점수)
S (Setting): 어디서? (단일 기관, 단일 세션 or 다중 세션)
```

## 통계 검정력 계산 기준

```
목표 검정력: 1-β = 0.80 (최소), 0.90 (권장)
유의수준: α = 0.05 (양측)
효과크기: 선행연구 기반 (없으면 Cohen's d = 0.5 중간 효과 가정)
→ G*Power 또는 pwr 패키지로 최소 표본 수 계산 후 프로토콜에 명시
```
