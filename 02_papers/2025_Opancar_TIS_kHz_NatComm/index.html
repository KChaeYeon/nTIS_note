    Opančar et al. 2025 — TIS and kHz Share the Same Biophysical Mechanism

    Citation: Opančar A, Bhaskaran A, Bhadra N, Kilgore KL, Grill WM, Budde RB. The same biophysical mechanism is involved in both temporal interference and direct kHz stimulation of peripheral nerves. Nature Communications 2025;16:9006. DOI: 10.1038/s41467-025-64059-w

    핵심 요약

    TIS와 kHz 직접 자극은 동일한 생물리학적 메커니즘을 공유한다. Locusta migratoria(메뚜기) 및 인간 피험자 실험에서 변조(TIS)·비변조(kHz) 파형이 동일한 강도-주파수 관계를 보임 → TIS의 포락선 선택성 주장에 근본 의문 제기.

    연구 설계

    항목 세부 사항
    모델 1 Locusta migratoria 말초신경 (0.5–100 kHz)
    모델 2 인간 감각·운동 경로 (0.5–12.5 kHz)
    비교 TIS(변조) vs kHz 단독(비변조)
    측정 강도-주파수 관계(strength-frequency curve), 발화 역치

    주요 결과

    조건 결과
    0.5–12.5 kHz 범위 (인간) TIS ≈ kHz 단독: 역치 중첩
    0.5–100 kHz (메뚜기) 동일한 강도-주파수 커브
    진폭 변조(AM) 의존성 최소 → 포락선 특이효과 없음
    캐리어 주파수 의존성 kHz 캐리어 자체가 발화 역치 결정

    핵심 결론

    "TIS는 포락선(Δf)이 아닌 kHz 정류(rectification)를 통해 작용한다."

    • suprathreshold TIS → kHz 정류가 실질적 자극 메커니즘
    • Δf 선택성(주파수 특이 자극)은 subthreshold 조건에서만 가능할 수 있음

    Budde 2023과의 관계

    Budde 2023의 "peak-to-peak 진폭 결정" 주장을 인간·곤충 이중 모델로 확장 검증. 메커니즘으로 kHz 정류 가설을 제안하여 구체화.

    nTIS 연구와의 연관성

    • 경골신경 TIS 설계 시 캐리어 주파수 선택이 Δf 못지않게 중요함을 시사
    • 인간 대상 in vivo 강도-주파수 데이터 → 경골신경 자극 파라미터 참고 가능
    • 논쟁: 선택성(selectivity) 확보를 위한 subthreshold TIS 전략 필요성 대두
