    경골신경 해부학 & 자극 기전

    학습 목표

    경골신경의 해부학적 위치·내부 구조 → 전기자극이 신경을 활성화하는 원리 → 척수에서 방광 억제로 연결되는 경로까지 이해.

    1. 경골신경 위치 & 주행 경로

             무릎 뒤쪽 (오금)
              │
         경골신경 시작
              │
        종아리 안쪽으로 내려옴
              │
         발목 안쪽 복숭아뼈
         (내과, Medial Malleolus)
              │
    ┌─────────┴─────────┐
    │                   │
 발바닥 신경          발뒤꿈치 신경
(족저신경)           (종골신경)
    • PTNS 자침 위치: 발목 안쪽 복숭아뼈에서 손가락 3마디 위 (~5 cm)
    • 좌골신경(Sciatic nerve)에서 갈라진 가지

    2. 척수 뿌리 레벨

    척수
 ├─ L4 ──┐
 ├─ L5   ├──► 경골신경 (Tibial nerve)
 ├─ S1   │    → 좌골신경(Sciatic)에서 분지
 ├─ S2   │
 └─ S3 ──┘

    핵심: S2, S3 뿌리 포함 → 방광 조절 부교감신경 출발지 S2-S4와 동일 레벨 → 경골신경 자극이 방광에 영향을 주는 신경해부학적 근거.

    3. 신경 내부 구조

             경골신경 단면
    ┌─────────────────────┐
    │  Epineurium         │ ← 신경 전체를 감싸는 외피
    │  ┌───┐  ┌───┐       │
    │  │ F │  │ F │  ...  │ ← Fascicle (신경다발)
    │  └───┘  └───┘       │
    │   각 다발 안에:       │
    │   ● ● ● ● ●         │ ← 개별 신경섬유 (Axon)
    └─────────────────────┘
    구조 역할
    Epineurium 신경 전체 보호 외피
    Perineurium 각 다발 감싸는 막
    Fascicle 기능별로 묶인 신경섬유 다발
    Axon 실제 전기 신호 전달하는 선
    Myelin sheath 축삭 감싸는 절연체 → 신호 빠르게 전달

    4. 신경섬유 종류 — 자극 반응 순서

    Aα ──── 굵고 빠름 (70-120 m/s) ── 근육 운동, 고유감각
Aβ ──── 중간     (30-70 m/s)  ── 촉각, 압각
Aδ ──── 얇음     (5-30 m/s)   ── 가벼운 통증, 온도
C  ──── 매우 얇음 (0.5-2 m/s)  ── 깊은 통증, 자율신경 (무수초)
    • 굵을수록: 자극 역치 낮음 = 적은 전류로 자극 가능
    • PTNS: 주로 Aβ, Aδ 섬유 활성화

    5. 전기자극 → 탈분극 → 활동전위 발생 기전

    5-1. 안정막전위 (평소 상태)

    세포 바깥:  Na+ 많음
    +   +   +   +   +   +   +
    ━━━━━━━━━━━━━━━━━━━━━━━━━  ← 세포막
    -   -   -   -   -   -   -
세포 안쪽:  K+, 단백질 많음

→ 전위차: 안쪽이 -70 mV (더 음전하)
→ Na+ 채널: 닫혀 있음

    5-2. 전기자극 → Na+ 채널 강제 개방

    전극 (음극, cathode) 근처 전류 흐름
        ↓
세포막 바깥쪽 전위 낮아짐
        ↓
안팎 전위차 감소: -70 mV → -55 mV (역치)
        ↓
Na+ 채널 강제 개방!

     ← Na+ 유입 ←
    +   +  [채널]  +   +
    ━━━━━━━━↓━━━━━━━━━━━  ← 세포막
    -   -  Na+→   -   -
               ↓
        안쪽이 (+)로 역전 = 탈분극

    외부 전기 자극이 본래 수용체 신호 역할을 대신해 Na+ 채널을 강제 개방하는 것.

    5-3. 활동전위 전파 (도미노 원리)

    자극 지점  위치 1    위치 2    위치 3
           [+++]  →  [---]  →  [---]   처음

           [재분극] → [+++]  →  [---]   이동

           [끝남]  → [재분극] → [+++]   계속 전파

    탈분극된 부위(+)가 옆 부위(-)를 끌어당겨 인접 Na+ 채널 개방 → 연쇄 전파.

    5-4. Myelin과 도약 전도 (Saltatory Conduction)

    Myelin 없는 신경 (C섬유):
●─────────────────────→  느리게 모든 구간 전파

Myelin 있는 신경 (A섬유):
●══════╗  ╔══════╗
       ↓  ↓      ↓
      [+]        [+]    ← Ranvier node에서만 탈분극, 점프!
    • Myelin = 절연체 → Ranvier node 틈새에서만 탈분극 발생
    • 신호가 점프하며 이동 → 훨씬 빠른 전도 속도

    6. 전기자극 핵심 파라미터

    ┌─────────────────────────────────────┐
│  펄스 하나의 모양:                   │
│                                     │
│  ┌──┐         ┌──┐                  │
│  │  │         │  │                  │
│──┘  └─────────┘  └──────            │
│  ←→  ←───────→                      │
│  PW      IPI                        │
└─────────────────────────────────────┘
PW  = Pulse Width (펄스 폭, μs)
IPI = Inter-Pulse Interval
주파수 = 1초에 몇 번 (Hz)
진폭   = 자극 강도 (mA, V)
    파라미터 PTNS 표준값 의미
    주파수 20 Hz 1초에 20번 자극
    펄스폭 200 μs 각 펄스 지속 시간
    진폭 0.5~9 mA 발가락 움직임 느낄 정도
    시간 30분/회, 12주 표준 치료 기간

    7. 척수에서 방광 억제로 연결되는 경로

    경골신경 Aδ, Aβ 섬유 활성화
        ↓ 구심성 신호
척수 후각 S2-S4
        │
        ├─► GABAergic interneuron 활성화
        │   (GABA = 억제성 신경전달물질)
        │         ↓
        │   방광 부교감 신호 차단
        │   (배뇨근 수축 명령 억제)
        │
        └─► 상행 경로 → PAG → PMC 게이트 강화
                        (배뇨 억제 보강)
        ↓
방광 과활성 감소 → OAB 증상 개선

    근거: Tai et al. — 쥐 S2 레벨에서 GABAergic interneuron이 경골신경 자극에 의한 방광 억제를 매개함을 확인.

    8. 치료 효과의 전체 인과 사슬

    [전기 자극 인가]
      ↓
경골신경 Na+ 채널 강제 개방
      ↓
탈분극 → 활동전위 발생
      ↓
도약 전도로 척수 방향 이동
      ↓
S2-S4 후각 도달
      ↓
GABAergic interneuron 활성화
      ↓
방광 부교감 흥분 신호 차단
      ↓
방광 과수축 억제
      ↓
OAB 증상 감소 (빈뇨·절박뇨 개선)

    장기 효과 — 신경 가소성 (Neuroplasticity)

    단기 효과 (자극 중):
  Na+ 채널 → 활동전위 → GABA 억제 → 방광 수축 즉시 감소

장기 효과 (12주 치료 후 수개월 지속):
  반복 자극 → 억제 회로 자체 강화 (신경 가소성)
  → 뇌-척수-방광 조절 회로 재편성

    9. PTNS vs TTNS vs TIS 비교

    PTNS (Percutaneous):
  발목 바늘 → 신경 직접 자극
  [전극]─────[바늘]───► 경골신경

TTNS (Transcutaneous):
  피부 표면 전극 → 피부·지방 통과 → 신경 도달
  [전극] ~피부~ ~지방~ ►경골신경

TIS (우리 연구):
  전극 4개 → 고주파 2쌍 간섭 → 경골신경 위치에서만 저주파 envelope
  [A1]  [A2]
    ↘  ↙
     ✕  ← 경골신경 위치에서 집속
    ↗  ↖
  [B1]  [B2]
    PTNS TTNS TIS
    침습도 침습 (바늘) 비침습 비침습
    자극 깊이 깊음 (신경 직접) 얕음 깊음 (간섭 집속)
    피부 불편감 바늘 통증 따끔거림 적음 (고주파)
    자가 사용 불가 가능 가능

    10. 방광 억제 3-Level 기전 프레임워크

    경골신경 전기자극의 효과는 단일 경로가 아닌 세 가지 층위에서 동시에 작동한다.
    (Al-Danakh et al. 2022 + Fowler & Griffiths 2010 + Blok et al. 2022 통합)

    Level 1. 척수 — Gate Control + Enkephalin 억제

    1965년 Melzack & Wall의 관문 이론을 방광에 적용한 모델:

    비유: 좁은 골목(척수 후각)에
  큰 차(Aβ/Aδ 자극 신호)가 먼저 진입하면
  작은 차(C 섬유 절박 신호)는 통과 못 함

    전기자극 → Aβ/Aδ 섬유 활성화
      ↓
척수 후각 억제성 인터뉴런 활성화
  ├─ GABAergic interneuron (GABA 억제)
  └─ Enkephalinergic interneuron (오피오이드성 억제)
      ↓
방광 C 섬유 구심성 신호 차단
      ↓
배뇨근 수축 명령 억제

    Enkephalin 근거: Finazzi-Agrò et al. — Naloxone(오피오이드 길항제) 투여 시
    PTNS 유발 방광 억제 효과가 부분적으로 소실 → 오피오이드성(enkephalinergic) 억제 회로가 관여함을 확인.

    Level 2. 뇌간 — PMC/PAG 조절

    Level 1 억제 → 방광 구심성 신호 감소
      ↓
PAG(수도관주위회색질) 입력 감소
      ↓
PAG → PMC(교뇌 배뇨중추) 흥분 신호 감소
      ↓
PMC "배뇨 버튼" 민감도 하강
      ↓
배뇨 역치 상승 (방광이 더 많이 차야 요의 발생)

    SNM은 이 경로를 통해 PMC 억제를 직접 강화 — 피질·피질하 영역을 조절해 방광 억제 회복.

    Level 3. 대뇌피질 — 장기 신경 재구성 (fMRI 근거)

    fMRI 증거 (Blok et al., Frontiers in Physiology, 2022):

    조건 좌측 DLPFC 활성화
    정상인 (방광 충만 시) 정상 활성 → 하향 억제 유지
    OAB 환자 (방광 충만 시) 감소 → 억제 실패 → 절박뇨
    SNM/PTNS 2–4주 후 정상화 → 하향 억제 회복 
    [반복 전기자극 12주]
      ↓
억제 회로 자체 강화 (신경 가소성)
      ↓
전두엽 DLPFC 하향 억제 경로 재건
      ↓
PTNS 치료 종료 후에도 수개월 효과 지속되는 이유

    3-Level 요약

    Level 1 (척수) = 즉각 효과 / Level 2 (뇌간) = 단기 조절 / Level 3 (피질) = 장기 재구성. 임상에서 12주 치료가 필요한 이유가 Level 3 변화에 있음.

    11. 신경섬유 논쟁 — Aδ vs C-fiber, 어느 것이 주역인가?

    TNS 효과를 내는 신경섬유가 어느 것인지 현재도 논쟁 중이다.

    가설 A: Aδ 섬유가 핵심
  근거: 동물 실험에서 Aδ 선택 차단 시 방광 억제 소실
  지지: 10–20 Hz 저주파 → Aδ 우선 활성화

가설 B: C 섬유가 핵심
  근거: 2022 연구 (연속 충만 쥐 모델, PMC10365128)
        – C 섬유 선택 광유전학(optogenetics) 자극 시 방광 억제 재현
        – TNS 중 C 섬유가 실제로 활성화됨을 확인

→ 두 가설 모두 부분적 증거 보유, 단일 정답 없음

    2025 돌파구 — ECAP 실시간 피드백 제어:

    Precise control of tibial nerve stimulation via ECAP feedback, Nature Communications, 2025

    ECAP(유발복합활동전위) 실시간 모니터링
→ 자극 중 어느 섬유(Aβ/Aδ/C)가 활성화되는지 실시간 확인
→ 원하는 섬유만 선택적으로 자극 제어 가능
→ 이 논쟁에 실험적으로 종지부를 찍을 기술적 기반 제공

    TIS 연구 함의:
    TIS의 고주파 반송파(carrier) 파라미터 설계 시 어느 섬유 섬유를 타겟으로 할 것인지가
    envelope 주파수 선택만큼 중요한 설계 변수임.

    개념 요약

    경골신경 (L4-S3)
    │
    ├─ 위치: 발목 안쪽 내과 5 cm 위
    ├─ 내부: Epineurium → Fascicle → Axon → Myelin
    ├─ 섬유: Aα > Aβ > Aδ > C (굵을수록 자극 쉬움)
    │
    ├─ 자극 기전:
    │   전류 → Na+ 채널 개방 → 탈분극 → 도약 전도 → 척수 전달
    │
    ├─ 방광 억제 경로 (3-Level):
    │   Level 1 (척수): Aβ/Aδ → Gate Control + Enkephalin → C 섬유 신호 차단
    │   Level 2 (뇌간): PAG → PMC 민감도 하강 → 배뇨 역치 상승
    │   Level 3 (피질): DLPFC 하향 억제 회복 → 장기 재구성 (fMRI 확인)
    │
    ├─ 신경섬유 논쟁: Aδ vs C-fiber (2025 ECAP 기술로 해결 가능)
    │
    └─ 자극 방식 진화:
        PTNS (바늘) → TTNS (표면 전극) → TIS (간섭 집속, 비침습+깊이)
