    표기법 고정 (세션 전체 통일)

    전극쌍 1: I1 = A·sin(2πf1·t),  전극쌍 2: I2 = A·sin(2πf2·t)   (f1 > f2, Δf = f1 - f2)
중첩 전기장: E(t) = E1·sin(2πf1·t + φ1) + E2·sin(2πf2·t + φ2)
Envelope 진폭: |E_AM(t)|
n-phase 일반화: E(t) = Σ_{k=1}^{n} E_k·sin(2πf_k·t + φ_k)

    (ti-efield-theory 스킬 "핵심 수식 골격"과 동일 — 여기서 재확인만 함)
