    OAB 병태생리학 (Overactive Bladder Pathophysiology)

    학습 목표

    경골신경 TIS 자극이 OAB를 조절하는 원리를 이해하기 위한 기초.
    방광 해부학 → 정상 배뇨 기전 → 신경 경로 → OAB 발생 기전 → 치료 옵션 순으로 정리.

    1. OAB 정의 & 역학

    ICS 정의 (2002):

    "Urgency, with or without urge incontinence, usually with frequency and nocturia" — 요로감염 등 명백한 병인 없이

    증상 정의 비고
    Urgency (절박뇨) 갑작스럽고 참기 어려운 요의 OAB 필수 증상
    Frequency (빈뇨) ≥8회/day
    Nocturia (야간뇨) ≥1회/night
    Urge incontinence (절박성 요실금) urgency와 함께 오는 불수의적 요누출 OAB-wet
    • 전세계 유병률: ~11% (Milsom et al. 2001, BJU Int)
    • 한국 유병률: ~12.2% (40세 이상, Lee et al. 2011)
    • 나이와 상관: 40대 ~8% → 75세+ ~31%
    • QoL 저하: 당뇨·우울증과 동등한 수준

    핵심 구분

    OAB는 증상 기반 진단. Urodynamic 검사에서 Detrusor Overactivity (DO) 확인은 약 50~60%에 불과.
    OAB ≠ DO — 연구에서 반드시 구분해야 함.

    2. 방광 해부학

    풍선 비유 — 쉬운 이해

      [풍선 = 방광]

  ┌──────────────────────┐
  │  안쪽 벽 (Urothelium) │ ← 소변을 감지하는 능동 센서
  │                      │   (방광 팽창 → ATP 분비)
  │  근육층 (Detrusor)    │ ← 조이고 풀리는 근육
  │                      │   (부교감: 수축 / 교감: 이완)
  └──────────┬───────────┘
             │
          수도꼭지 1 (IUS) ← 자동, 평활근, 교감신경 지배
             │
          수도꼭지 2 (EUS) ← 내가 직접 조이는 것, 횡문근, 음부신경
    • 소변 저장: 방광 근육 이완 + 수도꼭지 둘 다 잠금
    • 소변 배뇨: 방광 근육 수축 + 수도꼭지 둘 다 열림

    해부학적 단면도

                     ┌──────────────────────┐
  Ureter ───────►│      방광 (Bladder)   │◄─── Ureter
                 │                      │
                 │  ① Urothelium        │  ← 능동 sensor (ATP, ACh 분비)
                 │  ② Lamina Propria    │  ← Interstitial cells 위치
                 │  ③ Detrusor Muscle   │  ← 평활근, 배뇨/저장 핵심
                 └──────────┬───────────┘
                            │
                     Bladder Neck
                            │
                 ┌──────────▼───────────┐
                 │  Internal Urethral   │  ← 평활근, 교감신경(불수의)
                 │  Sphincter (IUS)     │
                 └──────────┬───────────┘
                            │  Urethra
                 ┌──────────▼───────────┐
                 │  External Urethral   │  ← 횡문근, 음부신경(수의)
                 │  Sphincter (EUS)     │
                 └──────────────────────┘

    핵심 구조별 역할

    구조 종류 역할
    Urothelium 상피 방광 팽창 감지 → ATP·ACh 분비 (능동 센서)
    Lamina propria 결합조직 Interstitial cells 위치, 신호 전달
    Detrusor muscle 평활근 부교감 → 수축(배뇨), 교감 → 이완(저장)
    IUS 평활근 교감신경 지배, 불수의적
    EUS 횡문근 음부신경 지배, 수의적 조절

    Urothelium의 역할

    단순 방어막이 아니라 방광 팽창을 감지해 ATP·ACh를 분비하는 능동 신호 기관.
    OAB 발생 기전 중 요로상피성 가설의 핵심.

    3. 정상 배뇨 기전 (Micturition Reflex)

    저장기 (Storage Phase)

    방광 팽창 → A-δ fiber 구심성 신호
    ↓
척수 → PAG (수도관주위회색질) → PMC 억제 유지
    ↓
교감신경 활성화 (T10-L2, Hypogastric nerve)
  β3-adrenoceptor → Detrusor 이완 (저장)
  α1-adrenoceptor → IUS 수축 (잠금)
체성신경 활성화 (S2-S4, Pudendal nerve)
  EUS 수축 (Guarding reflex)

결과: 낮은 방광 내압 + 요도 폐쇄 = 소변 저장

    배뇨기 (Voiding Phase)

    방광 팽창 임계치 도달 (~300-400 mL)
    ↓
강한 A-δ + C fiber 신호
    ↓
PAG → PMC (Pontine Micturition Center) 활성화
    ↓
부교감신경 활성화 (S2-S4, Pelvic nerve)
  M3 muscarinic receptor → Detrusor 수축
교감·체성신경 동시 억제
  IUS, EUS 이완
    ↓
배뇨 완료

    핵심 개념

    배뇨의 ON/OFF 스위치는 척수가 아닌 뇌간 PMC (Pontine Micturition Center) 에 있다.
    척수 손상 환자가 배뇨를 조절하지 못하는 이유. 경골신경 자극이 S2-S4 수준에서 작용하는 근거.

    4. 방광 조절 신경 3가지 경로

    리모컨 비유

    리모컨 1: 교감신경   → "저장 모드" 담당
리모컨 2: 부교감신경 → "배뇨 모드" 담당
리모컨 3: 체성신경   → "내가 직접 조이는" 담당

    교감신경 (Sympathetic) — 저장 담당 (리모컨 1)

    허리 척수 (T10-L2)
        │
        └─► 하복신경 (Hypogastric nerve)
                 │
        ┌────────┴────────┐
        ▼                 ▼
  방광 근육 (β3-AR)    요도 입구 (α1-AR)
        │                 │
   "이완해!" →        "잠가!" →
   방광이 늘어남        소변 못 나감
   (소변 저장)         (소변 저장)
    \[\text{T10-L2} \xrightarrow{\text{Hypogastric n.}} \begin{cases} \beta_3\text{-AR} \to \text{Detrusor 이완 (저장)} \\ \alpha_1\text{-AR} \to \text{IUS 수축 (잠금)} \end{cases}\]

    비유: 교감신경 = 풍선 불 때 풍선 입구 막아주는 손

    부교감신경 (Parasympathetic) — 배뇨 담당 (리모컨 2)

    엉치 척수 (S2-S4)
        │
        └─► 골반신경 (Pelvic nerve)
                 │
                 ▼
           방광 근육 M3 수용체 (무스카린 수용체)
                 │
            "수축해!" →
            방광이 조여짐 → 소변 배출
    \[\text{S2-S4} \xrightarrow{\text{Pelvic n.}} M_3\text{-receptor} \to \text{Detrusor 수축 (배뇨)}\]

    항무스카린제 (Oxybutynin 등)가 이 M3 수용체를 차단 → 방광 수축 억제.
    M3 수용체가 침샘에도 분포 → 구강건조 부작용 원인.

    비유: 부교감신경 = 풍선 손 놓으면서 바람 나오게 하는 것

    체성신경 (Somatic) — 수의 조절 (리모컨 3)

    엉치 척수 (S2-S4)
        │
        └─► 음부신경 (Pudendal nerve)
                 │
                 ▼
           외요도괄약근 (EUS) [횡문근]
                 │
         "꽉 잡아!" or "놔줘!"
         (화장실 참기 or 배뇨)

    비유: 체성신경 = 내가 직접 쥐고 있는 밸브

    3가지 경로 협력 구조

    상태 교감 부교감 체성 결과
    저장 ON (β3 이완, α1 잠금) OFF ON (EUS 잠금) 소변 저장
    배뇨 OFF ON (M3 수축) OFF (EUS 열림) 배뇨

    5. 뇌-방광 신경 조절 경로 전체 지도

    ┌─────────────────────────────────────────┐
│               상위 뇌 중추               │
│  PFC (전전두피질)  ← 수의적 억제         │
│  ACC (전대상피질)  ← 긴박감 인지         │
│  PAG              ← 배뇨 gate 역할      │
│  PMC (Barrington) ← 배뇨 ON/OFF switch  │
└─────────────────┬───────────────────────┘
                  │ 하행 명령
┌─────────────────▼───────────────────────┐
│                척수                      │
│  T10-L2 → Hypogastric nerve (교감)      │
│    β3: Detrusor 이완 / α1: IUS 수축     │
│  S2-S4 → Pelvic nerve (부교감)          │
│    M3: Detrusor 수축                    │
│  S2-S4 → Pudendal nerve (체성)          │
│    EUS 수의 조절                         │
└─────────────────┬───────────────────────┘
                  │ 구심성 ↑ / 원심성 ↓
┌─────────────────▼───────────────────────┐
│         방광 / 요도 / 경골신경           │
│  A-δ fiber: 압력 감지 (저역치)           │
│  C fiber:   화학적·고강도 (고역치)        │
│  Tibial nerve (L4-S3) ← 우리 표적!      │
└─────────────────────────────────────────┘

    6. 경골신경과 방광의 연결 (연구 핵심 근거)

    경골신경 (Tibial nerve): 발목 안쪽을 주행하는 말초신경, 뿌리: L4-S3

    발목 안쪽
    │
경골신경 (Tibial nerve, L4-S3)
    │
    └──► 척수 진입 위치: S2-S4
                │
                ▼
    부교감신경(배뇨 명령) 출발지와 동일 레벨
                │
    [경골신경 체성 구심성 신호 → 방광 부교감 반사 억제]
                │
                ▼
         방광 과활성 감소

    핵심 원리: 경골신경(발목)에서 척수 S2-S4로 들어온 체성 구심성 신호가, 같은 레벨의 배뇨 부교감 반사를 억제하는 신경 회로 활용.

    PTNS vs TIS 비교

    PTNS (기존) TIS (우리 연구)
    자극 방법 발목에 바늘 삽입 피부 전극 4개
    침습도 침습 (바늘) 비침습
    빈도 주 1회 병원 방문 자가 적용 가능
    신경 경로 경골신경 → S2-S4 경골신경 → S2-S4 (동일)

    7. OAB 발생 기전 3가지 가설

    가설 1: 근원성 (Myogenic)

    노화·허혈 → Gap junction (Connexin 43) 증가
    ↓
국소 자발 수축 → 전체 Detrusor 전파
    ↓
Detrusor Overactivity (DO)

    비유: 도미노처럼 혼자 무너짐.

    가설 2: 신경성 (Neurogenic)

    중추 억제 손상 (뇌졸중·PD·척수 손상)
또는 C-fiber 민감화 (역치 하강)
    ↓
PMC 억제 gate 실패 → 조기 배뇨반사

    비유: 브레이크가 고장난 차.

    가설 3: 요로상피성 (Urothelial) ← 최근 주목

    방광 팽창 → Urothelium → ATP 과다 분비
    ↓
P2X3 purinergic receptor 활성화
    ↓
구심성 신경 과자극 → Urgency

    비유: 화재경보기 민감도가 너무 높아서 밥 냄새에도 울림.

    항무스카린 약물 무반응 환자 (~30%)에서 요로상피 기전이 우세할 가능성 높음.

    8. 현재 치료 옵션 & 한계

    치료 단계 치료 기전 한계
    1차 생활습관 교정 수분·카페인·알코올 조절, 방광 훈련, 골반저 운동 효과 제한적
    2차 항무스카린제 (Oxybutynin, Solifenacin 등) M3 차단 → Detrusor 이완 구강건조 30–40%·변비 5–10%·인지장애(노인 위험)·1년 내 46% 중단
    2차 β3 작용제 (Mirabegron) β3-AR → Detrusor 이완 혈압 상승(조절 안 된 고혈압 금기)·빈맥
    3차 PTNS 경골신경 경피 자극 (바늘) 침습·주 1회 병원 방문·반응률 ~54%
    3차 TTNS 경골신경 표면 전극 자극 자극 깊이 제한·선택성 낮음
    3차 Botox 방광 주사 신경근 접합부 차단 반복 시술(6–9개월)·요저류 위험 20–30%
    4차 SNM 천수 이식형 자극기 (S3 직접) 수술·비용·배터리 교체·MRI 제한

    약물 중단율 46%의 의미

    항무스카린제 복용 환자의 46%가 1년 내 복용 중단 (주원인: 구강건조, 변비, 인지기능 저하). 노인에서 특히 위험 — 장기 항무스카린제 복용과 치매 위험 연관성 보고. Mirabegron은 부작용 프로파일이 낫지만 혈압 상승 우려. → 전신 부작용 없는 신경조절 3차 치료의 임상적 필요성을 뒷받침.

    신경조절 치료 등장 배경:

    1. 약물 부작용 & 낮은 장기 순응도 (1년 내 46% 중단)
    2. 방광 국소 치료보다 신경 회로 수준 조절이 더 근본적
    3. PTNS·SNM 효과 입증 → 비침습화 필요 → TIS 접근 동기
