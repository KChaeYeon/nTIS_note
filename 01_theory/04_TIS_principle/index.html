    Temporal Interference Stimulation (TIS) 원리

    학습 목표

    TIS의 물리적 원리를 파동 간섭 → 전기장 수식 → COMSOL 시뮬레이션 워크플로우까지 연속된 흐름으로 이해하고,
    2-phase TIS와 3-phase TIS의 차이를 AM_max 계산 관점에서 설명할 수 있다.

    전체 그림

    [문제: 깊은 신경을 비침습적으로 자극할 수 없다]
         ↓
  두 개의 고주파 전류(f1, f2)를 피부 전극으로 인가
         ↓
  조직 내부에서 두 전기장이 중첩(superposition)
         ↓
  공간적으로 겹치는 영역에서만 "맥놀이(beat)" 발생
         ↓
  저주파 envelope(f_beat = |f1 - f2|)이 신경 발화 유도
         ↓
[피부·표면 조직: kHz 고주파 → 자극 없음]
[내부 표적 지점: 저주파 envelope → 신경 발화]

핵심 통찰: "자극은 envelope이 한다. kHz 자체는 통과할 뿐."

    1. TIS 탄생 배경

    왜 기존 방법은 실패했는가?

    기존 경두개 전기자극(tDCS/tACS)의 딜레마:

  피부 전극 ──────────────────────── 두피/두개골
       │  전류가 표면을 따라 흐름     │
       │  내부는 낮은 전류밀도       │
       └─────────────── 표적(내부) ──┘
                           ↑
               여기까지 충분한 세기로 도달하려면
               표면에서 너무 강해져 통증·화상 유발

물리적 이유:
  - 두개골 conductivity ≈ 0.01 S/m (근육의 1/40)
  - 대부분의 전류가 두개골을 우회하여 표면 경로 이동
  - 내부 전류밀도 ∝ 1/r² (거리 제곱에 반비례 감소)

    Grossman et al. 2017 (Cell) — TIS 제안

    • 논문: Noninvasive Deep Brain Stimulation via Temporally Interfering Electric Fields
    • 핵심 아이디어: 주파수가 조금 다른 두 kHz 전류를 따로 인가 → 표면에서는 고주파만 존재 (신경 자극 없음) → 내부 표적에서 두 전기장이 중첩 → 저주파 envelope 생성 → 신경 발화
    • 동물 실험 결과: 마취된 마우스에서 운동피질·해마 선택적 자극 확인
    Grossman 2017 실험 구성:

  ┌─────────────────────────────────────┐
  │           마우스 두뇌               │
  │                                     │
  │   전극쌍 1 ──f1=2000Hz──┐          │
  │                          ├→ 표적 ←─┤
  │   전극쌍 2 ──f2=2001Hz──┘          │
  │                                     │
  │   결과: 표적에서 1Hz 자극 envelope  │
  │         발에서 움직임 관찰          │
  └─────────────────────────────────────┘

참조: Grossman N et al., Cell 2017; 169(6):1029–1041

    2. 물리적 직관: 맥놀이(Beat) 현상

    소리로 먼저 이해하기

    피아노에서 라(A) 음을 440Hz와 442Hz로 동시에 치면: → "웅~웅~" 하는 2Hz의 떨림(맥놀이) 들림 → 두 음파의 주파수 차이 = 맥놀이 주파수

    TIS는 이와 동일한 원리를 전기장에 적용한 것.

    맥놀이 시각화:

  파동 1 (f1=2000Hz):   ∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿
  파동 2 (f2=2002Hz):   ∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿

  중첩 결과:
                         ╭──╮        ╭──╮
  ∿∿∿∿╭∿∿∿∿∿∿∿╮∿∿∿∿╭∿∿∿∿∿∿∿╮∿∿∿∿

  맥놀이 주파수 = |2002 - 2000| = 2Hz
  고주파 carrier = (2002 + 2000)/2 = 2001Hz

    3. 수학적 표현

    3.1 1D 스칼라 경우 (기초)

    두 전기장 성분이 같은 방향으로 중첩될 때:

    \[E(t) = E_1 \cos(\omega_1 t) + E_2 \cos(\omega_2 t)\]

    여기서 \(\omega_1 = 2\pi f_1\), \(\omega_2 = 2\pi f_2\)

    \(E_1 = E_2 = E_0\)인 특수한 경우 (두 쌍의 진폭이 같을 때):

    적화공식(sum-to-product) 적용:

    \[E(t) = 2E_0 \cos\!\left(\frac{\omega_1 - \omega_2}{2}t\right)\cos\!\left(\frac{\omega_1 + \omega_2}{2}t\right)\]
    \[= \underbrace{2E_0 \cos(\pi\Delta f \cdot t)}_{\text{느린 envelope}}\underbrace{\cos(2\pi \bar{f} \cdot t)}_{\text{빠른 carrier}}\]
    항 의미 주파수
    \(\cos(2\pi\bar{f}t)\) 고주파 carrier \(\bar{f} = (f_1+f_2)/2\) ≈ 수kHz
    \(2E_0\cos(\pi\Delta f \cdot t)\) 저주파 envelope $\Delta f =

    Envelope 진폭 = \(|2E_0 \cos(\pi\Delta f \cdot t)|\)
    → 주기: \(1/\Delta f\) 초 (맥놀이 1회 = 신경 1회 자극)

    팀 실험 파라미터 (하이퍼틱스, 2026):
    f1 = 5000Hz, f2 = 5002Hz → \(\Delta f\) = 2Hz beat, carrier ≈ 5001Hz
    진폭: 1.6mA/채널 → 쥐 운동피질에서 spike firing rate 증가 확인

    3.2 \(E_1 \neq E_2\)인 일반적 경우

    진폭이 다를 때 envelope은 완전히 0까지 내려가지 않음:

    \[E_{env}(t) = \sqrt{(E_1 + E_2)^2\cos^2(\pi\Delta f \cdot t) + (E_1 - E_2)^2\sin^2(\pi\Delta f \cdot t)}\]
    • 최대 envelope 진폭 = \(E_1 + E_2\) (보강 간섭 시)
    • 최소 envelope 진폭 = \(|E_1 - E_2|\) (소상 간섭 시)
    • AM 깊이 = \(2\min(E_1, E_2)\) (진폭 변조의 크기)

    3.3 3D 벡터 전기장 (실제 조직에서)

    실제 조직에서 전기장은 3차원 벡터:

    \[\vec{E}(t) = \vec{E}_1\cos(\omega_1 t + \phi_1) + \vec{E}_2\cos(\omega_2 t + \phi_2)\]

    여기서 \(\vec{E}_1\), \(\vec{E}_2\)는 각각 전극 쌍 1, 2가 만드는 공간 전기장 벡터.

    핵심: 공간의 각 점에서 \(\vec{E}(t)\)는 시간에 따라 타원 궤적을 그린다.

    4. 전기장 벡터의 타원 궤적 (Elliptical Trajectory)

    공간의 한 점에서 전기장 벡터 끝이 그리는 궤적:

     E_y
      ↑
      │    ╭────────╮
      │   ╱  타원   ╲
      │  │     ●     │  ← 전기장 벡터 E(t)가
      │  │           │     회전하며 타원을 그림
      │   ╲         ╱
      │    ╰────────╯
      └──────────────→ E_x

  - 장반경(semi-major axis) a: 가장 강한 자극 방향
  - 단반경(semi-minor axis) b: 수직 방향 자극
  - b = 0이면 선형(linear) → 특정 방향만 자극
  - a = b이면 원형(circular)

2-phase TIS: 타원이 한 평면 내에 제한됨
3-phase TIS: 타원이 3D 공간에서 임의 방향 가능

    Phasor 표현으로 타원 기술:

    COMSOL에서 export하는 복소수 phasor \(\tilde{E}\)를 이용:

    \[\vec{E}_1 = \text{Re}[\tilde{\vec{E}}_1 e^{j\omega_1 t}], \quad \vec{E}_2 = \text{Re}[\tilde{\vec{E}}_2 e^{j\omega_2 t}]\]

    타원의 두 축은 행렬 \([\tilde{\vec{E}}_1, \tilde{\vec{E}}_2]\)의 특이값(singular value)으로 결정됨.

    5. AM_max 계산 — 어느 방향으로 가장 강하게 자극되는가?

    5.1 단일 방향 AM (Single-directed AM)

    특정 방향 단위벡터 \(\hat{n}\)으로의 전기장 성분:

    \[E_{\hat{n}}(t) = \vec{E}(t) \cdot \hat{n} = E_{1n}\cos(\omega_1 t) + E_{2n}\cos(\omega_2 t)\]

    이 방향으로의 AM 진폭 = 위 1D 공식 적용.

    COMSOL에서 2-phase 단일 방향 시뮬레이션 (김대용 팀, 2025):

  x방향 AM:  E_x(t) = Ex1·cos(ω1t) + Ex2·cos(ω2t) → AM_x
  y방향 AM:  E_y(t) = Ey1·cos(ω1t) + Ey2·cos(ω2t) → AM_y

  → 그림: 타원 투영으로 보이는 AM 분포 맵

    5.2 AM_max — 최대 자극 방향 탐색

    문제: 어느 방향 \(\hat{n}^*\)이 AM을 최대화하는가?

    \[AM_{max} = \max_{\|\hat{n}\|=1} AM(\hat{n})\]

    이것이 팀 연구의 핵심 계산 문제.

    해석적 풀이 (송솔웅, 2026 — Lagrange 승수법):

    타원 궤적의 각 점 \(P(t)\)를 단위 구(unit sphere)에 투영:

    제약 조건: 1. \(\|\hat{n}\| = 1\) (단위벡터) 2. \(\hat{n}\)이 타원 궤적의 특정 점과 접촉 (최대 투영 조건)

    Lagrangian 구성:

    \[\mathcal{L}(\hat{n}, \lambda_1, \lambda_2) = (\vec{E}(t) \cdot \hat{n}) - \lambda_1(\|\hat{n}\|^2 - 1) - \lambda_2 \cdot g(\hat{n})\]

    Stationarity 조건 (\(\nabla_{\hat{n}}\mathcal{L} = 0\))에서:

    \[AM_{max} = \sigma_1\]

    여기서 \(\sigma_1\)은 타원 field tensor의 최대 특이값(largest singular value).

    AM_max 계산 요약:

  입력: 각 점에서 E_phasor_f1 = (Ex1, Ey1, Ez1) (complex)
                  E_phasor_f2 = (Ex2, Ey2, Ez2) (complex)

  행렬 구성: M = [Re(Ẽ₁), Im(Ẽ₁), Re(Ẽ₂), Im(Ẽ₂)] ∈ ℝ³ˣ⁴

  SVD 분해: M = UΣVᵀ

  AM_max = 2 × σ₁  (최대 특이값 × 2, 피크-투-피크)

  최적 방향: û* = u₁ (U의 첫 번째 열)

    왜 인수 2인가?
    타원의 최대 투영 = 장반경 \(a\), 피크-투-피크 진폭 = \(2a = 2\sigma_1\)

    6. 안티페이즈(Anti-phase) 전극 구성 원리

    기본 개념: 왜 안티페이즈가 필요한가?

    ❌ 안티페이즈 없는 경우 (단순 2전극):

  (+)전극 ──────────────── (-)GND
       │  전류가 표면 경로  │
       │  (전극 바로 밑)   │
       │  집중됨           │
       └───────────────────┘

  문제: 전극 근처 과도한 전류 밀도 → 통증, 화상

✅ 안티페이즈 적용 (2전극 쌍):

  전극쌍 1:  (+f1)──────────(- f1)  ← f1 고주파, 안티페이즈
  전극쌍 2:  (+f2)──────────(- f2)  ← f2 고주파, 안티페이즈

  각 쌍의 전류 합 = 0 → 외부 유출 없음
  내부 교차 영역에서만 envelope 생성

    수학적 설명:

    표면 근처 임의 점 P에서:

  E₁ = +A·cos(ω₁t)   (전극1에서 오는 성분)
  E₁' = -A·cos(ω₁t)  (안티페이즈 전극에서 오는 성분, 반대 위상)

  합 = A·cos(ω₁t) + (-A·cos(ω₁t)) = 0  ← 표면 전류 소거

  단, 교차 영역 내부에서는:
  E₁(at target) ≠ -E₁'(at target)  ← 경로 차이로 소거 불완전
  → 유효 자극 생성

    팀 COMSOL 관찰 (김대용, 2025.10):

    GND 설정에 따른 차이:

  Cross-talk 유발 구성:
  ┌─────────────────────┐
  │  (+f1)   (GND)      │  ← GND 공유 시
  │          ↑           │    다른 주파수 전류가 GND 통해
  │  (+f2) ──┘          │    혼선(cross-talk) 발생
  └─────────────────────┘

  Anti-phase 구성 (권장):
  ┌─────────────────────┐
  │  (+f1)   (-f1)      │  ← 각 쌍 독립 회로
  │                     │    GND 없이 차동 구동
  │  (+f2)   (-f2)      │    → cross-talk 방지
  └─────────────────────┘

    실험 파라미터 (Kwak et al., Brain Stimulation, 2023): - f1 = 2000Hz (pair 1: 0.95mA), f2 = 2002Hz (pair 2: 1.05mA) - Beat frequency = 2Hz, 진폭 비대칭 → steering 효과 - 타겟: 마우스 선조체(striatum)

    8. 2-phase TIS vs 3-phase TIS 비교

    8.1 2-phase TIS (기본형)

    전극 구성:

  (+f1) ──────── (-f1)    ← 전극쌍 1 (x방향)

  (+f2) ──────── (-f2)    ← 전극쌍 2 (y방향)

Phasor 관점:

  허수축
    ↑
    │    ● E₂
    │   /
    │  /
    │ /    ← 타원이 x-y 평면 내에만 있음
    │/
    ●────────→ 실수축
   E₁

자극 형태:
  - 타원이 2D 평면(자극 평면) 내에 제한
  - Steering: 두 쌍의 진폭 비율 조절 → 좌/우 이동
  - 제한: 상/하(깊이 방향) 조절 자유도 없음

    COMSOL 결과 (2-phase, 팀 시뮬레이션): - 중심 영역: 타원 궤적이 선형에 가까움 (steering 가능) - 전극 근처: 타원이 크게 찌그러짐

    8.2 3-phase TIS (팀 핵심 연구)

    전극 구성 (120° 위상 분리):

  전극 a:  I·cos(ωt + 0°)
  전극 b:  I·cos(ωt + 120°)
  전극 c:  I·cos(ωt + 240°)

  합: Ia + Ib + Ic = I[cos(0) + cos(120°) + cos(240°)] = 0
                     → Kirchhoff 전류법칙 자동 만족

3-phase의 Phasor 표현:

  허수축
    ↑   ● b (120°)
    │  ╱
    │ ╱
    │╱
  ──●──────────→ 실수축
  a │╲
    │ ╲
    │  ╲● c (240°)

대칭적 3-상 → 전기장 벡터가 3D 공간에서 타원 그림

    3-phase vs 2-phase 비교:

    항목 2-phase TIS 3-phase TIS
    전극 수 4개 (2쌍) 3개
    자극 평면 2D (평면적) 3D (입체적)
    Steering 방향 좌/우만 상/하/좌/우/앞/뒤
    AM_max 계산 상대적 단순 SVD 필요
    타원 방향 전극 배치 고정 전류 비율로 조절 
    김대용 팀 COMSOL 결과 비교 (251014):

  중심 영역 전기장 벡터 궤적:

  2-phase:              3-phase:
  ─────────             ─────────
  E_y ↑                E_y ↑
      │  ─              │   ╭──╮
      │ ╱               │  ╱    ╲
      │╱                │ │  ●  │
      ●─────→ E_x       │  ╲    ╱
                         │   ╰──╯
                         └────────→ E_x
  선형에 가까움           원형에 가까움
  (한 방향 집중)          (모든 방향 균등)

    9. 자극 조향 (Steering)

    진폭 비율 조절로 표적 이동

    2-phase TIS 조향 원리:

  I₁ = I₀ + ΔI (전극쌍 1의 진폭)
  I₂ = I₀ - ΔI (전극쌍 2의 진폭)

  ΔI = 0:        표적이 중앙
  ΔI > 0:        표적이 전극쌍 1 쪽으로 이동
  ΔI < 0:        표적이 전극쌍 2 쪽으로 이동

시각화:

  ΔI < 0    ΔI = 0    ΔI > 0
  ┌──────┐  ┌──────┐  ┌──────┐
  │  ●   │  │  ●   │  │  ●   │  ← 전극쌍 2
  │ ●    │  │  ●   │  │   ●  │  ← 자극 표적
  │●     │  │  ●   │  │    ● │  ← 이동!
  │  ●   │  │  ●   │  │  ●   │  ← 전극쌍 1
  └──────┘  └──────┘  └──────┘

    3-phase TIS 조향 (팀 연구, 하이퍼틱스 2026):

    3개 전류의 크기/위상 조합으로:

  상하 조향:   ↑↓
  좌우 조향:   ←→
  사선 조향:   ↗↙

  ← 전극 이동 없이 소프트웨어로 조향!

응용 (경골신경 OAB 치료):
  - 경골신경 단면 내에서 표적 섬유(방광지배) 선택적 활성화 가능
  - 발목 전극 고정 → 전류 파라미터만 변경으로 최적 자극 탐색

    11. 신경이 왜 kHz는 무시하고 envelope에만 반응하는가?

    신경 발화 필터링:

  고주파(kHz) → 신경막 무반응:

    신경막의 RC 시상수: τ = RC ≈ 1–10ms
    kHz 신호 주기: T = 1/2000Hz = 0.5ms << τ
    → 막전위가 충분히 변하기 전에 역전 → 순 효과 없음

  저주파(beat) → 신경막 반응:

    Beat 주파수: 1–200Hz (신경 발화 범위)
    Beat 주기: T = 1/2Hz = 500ms >> τ
    → 막전위가 서서히 변화 → 역치 초과 → 발화

물리적 비유:
  배를 흔들려면 파도 주기가 배의 자연주기와 맞아야 함.
  너무 빠른 잔물결(kHz)은 배(신경)를 흔들지 못함.
  천천히 오는 너울(Hz)이 배를 흔든다.

참조: Hutcheon B & Yarom Y, Trends Neurosciences, 2000
      (뉴런의 공명 주파수 및 내재 주파수 선호도)

    12. 개념 요약

    TIS 핵심 개념 트리:

TIS
├── 원리
│   ├── 두 kHz 전류 → beat frequency → 신경 자극
│   ├── 고주파: 신경막 통과, 자극 없음
│   └── 저주파 envelope: 막전위 변화 → 발화
│
├── 수학
│   ├── 1D: E(t) = E₁cos(ω₁t) + E₂cos(ω₂t)
│   ├── Envelope = |2E₀cos(π·Δf·t)| (E₁=E₂일 때)
│   └── 3D: 전기장 벡터 → 타원 궤적 → SVD로 AM_max
│
├── 전극 구성
│   ├── 안티페이즈: 표면 전류 소거, 내부 간섭
│   ├── 2-phase: 4전극, 2D 조향
│   └── 3-phase: 3전극, 3D 조향, 자유도 향상
│
└── COMSOL 파이프라인
    ├── 조직 모델 → 물질 속성 → frequency domain 해석
    ├── phasor export (f1, f2 각각)
    ├── MATLAB SVD → AM_max
    └── COMSOL 재import → 시각화

    14. 연구자 노트

    확립된 지식 (논문에서 검증됨): - TIS로 마우스 운동피질·해마 선택적 자극 가능 (Grossman 2017, Cell) - kHz 고주파는 신경 자극 없이 조직 통과 (membrane time constant 원리) - Anti-phase 전극 구성이 표면 자극 최소화에 효과적 (하이퍼틱스 팀 확인) - 3-phase TIS의 3D 조향 수학적 원리 증명 (팀 내부 발표, 2025–2026)

    아직 불확실한 것 (연구 갭): - 경골신경에서 TIS envelope이 실제 신경 발화를 유도하는 최소 역치 (in vivo 미확인) - 3-phase TIS의 조향 정밀도가 경골신경 단면(직경 ~3–5mm)에서 충분한지 - 피하 지방층 두께 개인차가 AM_max 분포에 미치는 영향 - 만성 OAB 환자에서의 반복 자극 효과 및 안전성

    팀 특이 사항: - 송솔웅의 Lagrange multiplier 기반 해석적 AM_max 풀이는 시뮬레이션 속도를 크게 향상 - 김대용의 boundary mask 솔루션은 COMSOL 결과 신뢰도 향상에 핵심 기여 - 전극 파라미터 (직경 1mm, 간격 2cm)는 추정값 — 경골신경 특화 최적화 필요
