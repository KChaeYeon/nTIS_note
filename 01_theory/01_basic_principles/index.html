    TIS 기초 원리 — 완전 가이드

    신경공학·전기공학 배경 없이도 TIS 원리, 수식, 응용을 이해할 수 있는 가이드.

    1. TIS가 해결하는 문제: 비침습적 심부 자극의 딜레마

    기존 방법들의 한계

    뇌나 신경을 전기로 자극하는 방법은 크게 두 방향이 있다.

    방법 침습 여부 장점 한계
    심부뇌자극(DBS) 침습 (수술 필요) 심부까지 정확히 자극 뇌 개두술, 감염 위험, 비용
    경두개 직류자극(tDCS) 비침습 안전, 간편 표면에 머물고 심부 도달 불가
    경두개 자기자극(TMS) 비침습 비교적 강한 자극 초점 정밀도 낮음, 심부 자극 제한
    경피적 경골신경 자극(PTNS) 최소 침습 (바늘) 심부 신경 도달 바늘 삽입, 병원 방문 필요

    핵심 딜레마: 비침습적일수록 심부 도달이 어렵고, 심부에 도달하려면 침습이 필요하다.

    TIS가 제안한 해결책

    Grossman et al. (2017, Cell)이 제안한 아이디어:

    1. 두 쌍의 전극을 피부 표면에 붙인다.
    2. 각 전극 쌍에 아주 조금 다른 주파수의 고주파 전류를 흘린다 (예: 2000 Hz와 2020 Hz).
    3. 두 전류가 조직 속 특정 지점에서 만나면 맥놀이(beat) 현상이 발생한다.
    4. 맥놀이 주파수 = \(|f_1 - f_2|\) = \(\Delta f\) = 20 Hz → 이것이 실제 뉴런을 자극하는 신호.
    5. 피부 표면에서는 2000 Hz의 고주파만 흐르므로 뉴런이 반응하지 못한다.
    6. 두 전류가 교차하는 심부 지점에서만 20 Hz 맥놀이가 형성 → 선택적 심부 자극.
    [피부 표면]
  전극 A (+)  ───────────────────────  전극 A (-)
              \                       /
               \   2000 Hz 전류     /
                \                 /
                 \     ★ 교차점    /   ← 20 Hz 맥놀이 발생
                /   2020 Hz 전류  \
               /                   \
  전극 B (+)  ───────────────────────  전극 B (-)
[심부]

    비유: AM 라디오

    AM 라디오 방송 원리와 거의 동일하다.

    AM 라디오 TIS
    캐리어 주파수 (수백 kHz) \(f_1\), \(f_2\) (고주파)
    음성 신호 (수백 Hz) \(\Delta f\) (실제 자극 주파수)
    캐리어에 음성을 실어 전달 고주파에 저주파 자극을 실어 심부 전달

    결론: 고주파를 이용해 피부와 얕은 조직을 "통과"하고, 심부에서만 실제 자극 주파수(\(\Delta f\))를 만들어낸다.

    2. 수식으로 이해하기 (직관 중심)

    두 전류의 합산

    전극 쌍 1에서 흐르는 전류:

    \[I_1 = A \cos(2\pi f_1 t)\]

    전극 쌍 2에서 흐르는 전류:

    \[I_2 = A \cos(2\pi f_2 t)\]

    두 전류가 조직 내 한 지점에서 만나면 (삼각함수 적화 공식):

    \[I_\text{total} = I_1 + I_2 = 2A \cdot \underbrace{\cos(\pi \Delta f \cdot t)}_{\text{포락선 (느린 자극 신호)}} \cdot \underbrace{\cos(2\pi f_\text{avg} \cdot t)}_{\text{캐리어 (빠른 고주파)}}\]

    여기서 \(f_\text{avg} = (f_1 + f_2)/2\), \(\Delta f = f_2 - f_1\).

    직관적 해석

    \[\boxed{I_\text{total} = \underbrace{2A \cos(\pi \Delta f \cdot t)}_{\text{포락선 (envelope)}} \times \underbrace{\cos(2\pi f_\text{avg} \cdot t)}_{\text{캐리어}}}\]
    항 이름 주파수 예시 뉴런 반응
    \(\cos(\pi \Delta f \cdot t)\) 포락선 (envelope) 20 Hz 반응함
    \(\cos(2\pi f_\text{avg} \cdot t)\) 캐리어 2010 Hz 반응 못함

    핵심 원리: 뉴런은 포락선(\(\Delta f\))에만 반응하고, 빠른 캐리어 주파수에는 반응하지 못한다. 뉴런의 이온 채널이 수천 Hz의 전류를 따라가지 못하기 때문이다 — 선풍기 날개가 너무 빠르게 돌면 바람만 느껴지는 것처럼.

    숫자 예시

    • \(f_1 = 2000\) Hz, \(f_2 = 2020\) Hz
    • \(\Delta f = 20\) Hz → 뉴런 자극 주파수
    • \(f_\text{avg} = 2010\) Hz → 피부 통과용 캐리어
    • 결과: 피부 표면에서는 2010 Hz 고주파만 느껴지고, 심부 교차점에서는 20 Hz 자극이 만들어진다.

    3. 경골신경 TIS와 방광 치료 (개요)

    경골신경(tibial nerve)은 발목 내측을 통과하며 방광 조절 반사에 관여한다. TIS를 발목에 적용하면 바늘 없이 피부 표면 전극만으로 경골신경에 도달할 수 있다.

    • PTNS (FDA 승인): 바늘 전극 침습 → 경골신경 직접 자극 → OAB 억제
    • TIS 가설: 비침습 전극 패치 4개로 동등한 효과 달성 → 재가 치료 가능

    4. 전기장 모델과 FEM 개요

    전기장 계산 핵심 수식

    생물학적 조직 내 준정적(quasi-static) 조건에서:

    \[\nabla \cdot (\sigma \nabla V) = 0\]

    여기서 \(\sigma\)는 조직 전도도(S/m), \(V\)는 전위(V). 전기장: \(\mathbf{E} = -\nabla V\)

    TI 전기장 (Grossman 2017)

    두 채널 전기장의 시간 간섭으로 만들어지는 포락선 진폭:

    \[E_{\text{TI}}(\mathbf{r}) = \Big|\,|\mathbf{E}_1(\mathbf{r}) + \mathbf{E}_2(\mathbf{r})| - |\mathbf{E}_1(\mathbf{r}) - \mathbf{E}_2(\mathbf{r})|\,\Big|\]
    • \(\mathbf{E}_1 = \mathbf{E}_2\) → \(E_\text{TI} = 2E_1\) (최대)
    • \(\mathbf{E}_1 \perp \mathbf{E}_2\) → \(E_\text{TI} = 0\) (자극 없음)

    5. 말초신경 TIS vs. 뇌 TIS: 메커니즘의 차이

    뇌 TIS 작동 방식 (Grossman 2017)

    뇌의 뉴런은 비선형(nonlinear) 특성을 갖는다. - 이온 채널이 전류 크기에 따라 비선형적으로 반응 - 빠른 캐리어(2000 Hz)는 이온 채널이 따라가지 못해 "평균화"됨 - 느린 포락선(\(\Delta f\), 20 Hz)의 진폭 변화는 감지하고 반응

    비유: 음악의 리듬(저주파)에는 박자를 맞추지만, 고음역 배음(고주파)에는 따로 반응하지 않는 것과 유사.

    말초신경 TIS 작동 방식 (Budde 2023 발견)

    Wang & Budde (2023)는 말초신경이 뇌와 다른 방식으로 반응한다는 것을 발견했다.

    • 말초신경의 유수신경섬유(myelinated fiber)는 훨씬 빠른 시간 분해능을 가짐
    • 고주파 캐리어(2000 Hz) 자체도 말초신경에 영향을 줄 수 있음
    • 실제 활성화는 포락선 탐지가 아니라, 합산 파형의 순간 전기장 크기에 의해 결정됨
    구분 뇌 TIS 말초신경 TIS
    반응 방식 포락선(\(\Delta f\)) 탐지 순간 전기장 크기에 직접 반응
    핵심 메커니즘 비선형 이온 채널 평균화 고주파 캐리어도 신경 활성화 가능
    설계 원칙 \(\Delta f\) 주파수와 위치 제어 합산 파형의 피크 전기장 제어

    비유로 이해하기: - 뇌 TIS = FM 라디오 수신기: 캐리어에 실린 주파수 정보(Δf)를 추출해서 반응 - 말초신경 TIS = 스피커 콘(cone): 전기 신호의 순간 크기에 직접 진동으로 반응

    임상적 의미: - 뇌 TIS 원리를 그대로 말초신경 TIS에 적용하면 예상과 다른 결과가 나올 수 있다. - 캐리어 자체도 주변 신경을 활성화할 수 있음을 고려해야 한다.

    6. 핵심 요약

    항목 한 줄 설명
    TIS 원리 두 고주파 전류가 심부에서 만나 맥놀이 생성 → 선택적 심부 자극
    \(\Delta f\) 두 주파수 차이 = 실제 뉴런이 반응하는 자극 주파수 (예: 20 Hz)
    캐리어 주파수 피부·조직 통과용 빠른 전류 (예: 2000 Hz) — 뉴런은 무시함
    포락선(envelope) 고주파 캐리어의 진폭이 느리게 변하는 부분 = 실제 자극 신호
    경골신경 TIS 발목 내측 전극으로 방광 조절 신경을 비침습적으로 자극
    FEM 컴퓨터 시뮬레이션으로 전기장 분포를 미리 예측하는 방법
    Budde 2023 말초신경은 뇌와 다른 메커니즘 → 별도 설계 원칙 필요
