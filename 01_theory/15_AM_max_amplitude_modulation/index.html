    1. Quasi-electrostatic FEM 기초

    1.1 왜 준정적 근사인가 — 정량적 확인

    TIS에서 사용하는 주파수는 kHz 대역이다. 이 주파수에서 생체조직 내부의 전자기파 파장을 계산하면:

    λ = v / f

v ≈ c/√ε_r ≈ 1×10⁸ m/s (생체조직 내)
f = 2,000 Hz

λ = 1×10⁸ / 2000 = 50,000 m = 50 km !!!

사람 몸통 크기: ~0.5 m
λ / 몸 = 50,000 / 0.5 = 100,000배

    파장이 몸보다 10만 배 크다. 따라서 파동이 공간을 퍼져나가는 효과(propagation)는 전혀 없고, 전기장이 즉각적으로 준정적으로 분포한다.

    Displacement current vs Ohmic current 비율:

    Maxwell 방정식에서 총 전류 = Ohmic + Displacement
  |Displacement| / |Ohmic| = ωε / σ

f = 2000 Hz,  ε ≈ 80ε₀ ≈ 7×10⁻¹⁰ F/m,  σ ≈ 0.2 S/m (뇌)

비율 = (2π × 2000 × 7×10⁻¹⁰) / 0.2 ≈ 4.4×10⁻⁵

→ Displacement current = Ohmic current의 0.004%  ← 완전 무시 가능

    따라서 변위전류를 버리고, 단순히 "조직에서 전류 흐름" 문제로 취급해도 된다.

    1.2 Grossman 2017 모델 구성

    4층 동심구(concentric sphere) 모델:

  ┌─────────────────────────────────────────┐
  │  두피 (scalp)    r=8.5 mm  σ=0.465 S/m │
  │  ┌───────────────────────────────────┐  │
  │  │  두개골 (skull) r=8.0 mm σ=0.010  │  │
  │  │  ┌─────────────────────────────┐  │  │
  │  │  │  CSF          r=7.5 mm σ=1.65│  │  │
  │  │  │  ┌───────────────────────┐  │  │  │
  │  │  │  │  뇌 (brain) r=7.0 mm  │  │  │  │
  │  │  │  │  σ=0.276 S/m          │  │  │  │
  │  │  │  └───────────────────────┘  │  │  │
  │  │  └─────────────────────────────┘  │  │

전극쌍 1 (f₁): 두피 위 2개 전극  →  E⃗₁(r⃗)
전극쌍 2 (f₂): 두피 위 2개 전극  →  E⃗₂(r⃗)

    조직별 전도도 차이가 핵심: - 두개골(σ=0.01)이 낮아 전류 흐름을 제한한다 - CSF(σ=1.65)가 높아 전류가 표면으로 퍼지는 경로를 만든다 - 이 때문에 단순 단일채널은 뇌 심부를 자극하기 어렵다

    1.3 FEM 지배 방정식 — 기초 물리법칙 3개로 유도

    법칙 1 — 전하 보존 (Current Continuity):

    어떤 작은 부피 dV에서 들어오는 전류 = 나가는 전류 (정상상태)

미분 형태:  ∇·J⃗ = 0

의미: 전하가 쌓이는 곳이 없다 → 한쪽에서 들어오면 반드시 나간다

    법칙 2 — Ohm's Law:

    J⃗ = σ E⃗

σ = electrical conductivity [S/m]
  높을수록 같은 전기장에서 전류가 더 잘 흐름

예:  CSF    σ=1.65  ← 전류 잘 흐름
     두개골 σ=0.01  ← 전류 잘 안 흐름 (100배 차이)

    법칙 3 — 전위와 전기장의 관계:

    E⃗ = -∇φ

의미: 전기장은 전위의 공간 기울기
     전위가 급격히 낮아지는 방향 = 전기장 방향

    세 법칙을 합치면:

    (3)을 (2)에:  J⃗ = σ(-∇φ) = -σ∇φ
(1)에 대입:   ∇·(-σ∇φ) = 0

∴  ∇·(σ∇φ) = 0  ← FEM solver가 푸는 지배 방정식

σ 균일하면:  ∇²φ = 0  (Laplace 방정식)

    FEM은 이 방정식을 수천~수만 개의 요소(element)로 나눈 mesh 위에서 연립 방정식으로 푼다.

    결과 취득:

    ∇·(σ∇φ) = 0  →  φ(r⃗) 획득
E⃗ = -∇φ       →  E⃗(r⃗) 획득
J⃗ = σE⃗        →  J⃗(r⃗) 획득

전극쌍 1 (f₁) 독립 실행: E⃗₁(r⃗) 저장
전극쌍 2 (f₂) 독립 실행: E⃗₂(r⃗) 저장

    2. 두 고주파 전기장의 시간 표현

    2.1 시간 도메인

    FEM은 정적 공간 분포를 계산한다. 실제 시간 신호는 다음과 같다:

    전극쌍 1 (f₁ = 2000 Hz):
  E⃗₁(r⃗, t) = E⃗₁(r⃗) · cos(2πf₁t)

전극쌍 2 (f₂ = 2001 Hz):
  E⃗₂(r⃗, t) = E⃗₂(r⃗) · cos(2πf₂t)

합성 전기장:
  E⃗_total(r⃗, t) = E⃗₁(r⃗)cos(2πf₁t) + E⃗₂(r⃗)cos(2πf₂t)

중요: E⃗₁(r⃗)은 FEM이 계산하는 공간 분포 (정적 벡터장)
      cos(2πf₁t)는 시간 변동 부분
      → 공간과 시간이 분리됨

    2.2 Δf = 1 Hz의 역할

    f₁ = 2000 Hz, f₂ = 2001 Hz → 각각 뉴런이 직접 따라가기 불가
                               (뉴런 AP 범위: 수 Hz ~ 수백 Hz)

Δf = f₂ - f₁ = 1 Hz → 이 주파수로 진폭이 변조(AM)됨
                    → 뉴런이 1 Hz 리듬으로 반응 가능!

┌─────────────────────────────────────────────────────┐
│  TIS의 핵심 트릭:                                    │
│  깊은 조직에서 두 고주파 전기장이 만나면              │
│  그 차이 주파수(Δf)로 진동하는 envelope가 생긴다.    │
│  겉에서는 kHz 자극, 안에서는 Δf Hz로 뉴런이 반응.   │
└─────────────────────────────────────────────────────┘

  두개골 ▓▓▓▓▓▓ 중         두개골 ░░░░░░░░ 약
  뇌표면 ▓▓▓▓▓▓ 중         뇌표면 ░░░░░░░░ 약
  뇌심부 ░░░░░░ 약         뇌심부 ████████ 강  ← 표적!

    3. 뉴런 방향 n̂으로의 투영

    3.1 왜 투영이 필요한가

    전기장 E⃗는 3D 벡터다.

뉴런은 특정 방향으로 누워 있다:
  → axon 주행 방향
  ↓ cortical column 방향
  ↗ 사선 방향 등

뉴런의 막전위 변화는 전기장 방향이 뉴런과 일치할 때 최대.
→ 실제 자극 강도 = E⃗ · n̂ (투영, dot product)
   n̂ = 뉴런 방향 단위벡터

    3.2 투영 후 1D 신호

    임의의 n̂ 방향에서 뉴런이 느끼는 신호:

E_n(t) = E⃗_total(r⃗,t) · n̂
        = [E⃗₁(r⃗)cos(ω₁t) + E⃗₂(r⃗)cos(ω₂t)] · n̂
        = (E⃗₁·n̂)cos(ω₁t) + (E⃗₂·n̂)cos(ω₂t)

약칭:
  a = E⃗₁(r⃗)·n̂  (실수, 음수 가능)
  b = E⃗₂(r⃗)·n̂  (실수, 음수 가능)

E_n(t) = a·cos(ω₁t) + b·cos(ω₂t)

← 3D 벡터 문제가 두 코사인의 합 문제로 단순화됨!

    4. 1D AM 포락선 — AM = 2·min(|a|,|b|) 완전 유도

    4.1 포락선 진폭 계산

    E_n(t) = a·cos(ω₁t) + b·cos(ω₂t)

ω̄ = (ω₁+ω₂)/2 (평균 주파수),  Δω = ω₂-ω₁ (작은 값)

합차 공식 적용 후 정리하면 포락선 진폭 A(t)는:

A(t)² = (a+b)²cos²(Δω·t/2) + (a-b)²sin²(Δω·t/2)

τ = Δω·t/2 (느리게 변하는 위상)라 하면:

τ = 0   → A_max = |a+b|
τ = π/2 → A_min = |a-b|

(a > b > 0인 경우):
  A_max = a+b
  A_min = a-b

    4.2 AM 정의와 계산

    AM = A_max - A_min = (a+b) - (a-b) = 2b = 2·min(a,b)

경우별:
  a > b > 0:  AM = 2b
  a = b > 0:  AM = 2a = 2b  (완전 변조! A_min = 0)
  a > 0, b < 0:  |b|로 환원 가능 (n̂ → -n̂ 뒤집으면)

∴  AM(n̂) = 2·min(|E⃗₁·n̂|, |E⃗₂·n̂|)

    4.3 직관: 왜 "약한 쪽이 결정하는가"

    비유: 두 소방관이 줄을 당겼다 놓았다

소방관 A (a = 8 V/m): 강하게 당김
소방관 B (b = 3 V/m): 약하게 당김

최강 순간 (동위상): A+B = 11 V/m
최약 순간 (역위상): A-B = 5 V/m  ← 5 V/m가 남는다!

진동 폭 = 11 - 5 = 6 = 2×3 = 2×b

B(약한 쪽)가 최솟값을 5로 "올려놓아서"
진동 폭이 2b로 제한됨.
→ A를 아무리 키워도 AM = 2b로 고정!

완전 변조 (a = b):
  최강 = 2a,  최약 = 0
  AM = 2a = 2b  → 완전 ON/OFF 스위칭!

    5. E_AM 정의 — 한 방향에서의 envelope 진폭

    특정 방향 n̂에서:

  E_AM(n̂, r⃗) = 2·min(|E⃗₁(r⃗)·n̂|, |E⃗₂(r⃗)·n̂|)

이것은 n̂을 어떻게 잡느냐에 따라 달라진다.

n̂ = E⃗₁ 방향 → E⃗₁·n̂ 최대, E⃗₂·n̂ 매우 작음 → AM ≈ 0
n̂ = E⃗₂ 방향 → E⃗₂·n̂ 최대, E⃗₁·n̂ 매우 작음 → AM ≈ 0
n̂ = 중간 방향 → 둘 다 적당히 큼 → AM 최대 가능

Grossman 논문 표현:
  E_AM(n̂, r⃗) ∝ |(E⃗₁+E⃗₂)·n̂| · |(E⃗₁-E⃗₂)·n̂| 의 기하평균
  (두 투영의 조화평균 형태)

    6. E_AM_max — 최적화 문제 정의

    6.1 정의

    E_AM_max(r⃗) = max over all unit n̂ of: 2·min(|E⃗₁·n̂|, |E⃗₂·n̂|)

물음:
  위치 r⃗에서 어떤 방향의 뉴런이 있다고 가정할 때
  받을 수 있는 최대 envelope modulation amplitude는?

이것은 min을 최대화하는 max-min 최적화 문제다.

    6.2 가정 (일반성 손실 없음)

    가정 1: |E⃗₁| ≥ |E⃗₂|  (채널 번호를 바꿔 항상 만족)
가정 2: α = ∠(E⃗₁, E⃗₂) ∈ [0°, 90°)  (E⃗₂ 부호를 뒤집어 만족)

    6.3 두 조건으로 나뉘는 이유

    핵심 질문: n̂을 E⃗₂ 방향으로 잡는 것이 항상 최선인가?

n̂ = E⃗₂/|E⃗₂| (E⃗₂ 방향)으로 고정하면:

  |E⃗₂·n̂| = |E⃗₂|              ← E⃗₂의 그림자 = 최대!
  |E⃗₁·n̂| = |E⃗₁|cosα          ← E⃗₁의 그림자

두 그림자 비교:
  |E⃗₁|cosα  vs  |E⃗₂|

  Case 1: |E⃗₁|cosα > |E⃗₂|
    → E⃗₂ 방향에서도 E⃗₁ 그림자 > E⃗₂ 그림자
    → min = |E⃗₂|  (E⃗₂가 항상 병목)
    → n̂ = E⃗₂ 방향이 최선!

  Case 2: |E⃗₁|cosα ≤ |E⃗₂|
    → E⃗₂ 방향에서 오히려 E⃗₁ 그림자 ≤ E⃗₂ 그림자
    → min = |E⃗₁|cosα  (E⃗₁이 병목!)
    → 더 좋은 방향이 반드시 존재한다

경계: |E⃗₂| = |E⃗₁|cosα

    7. Case 1: |E⃗₂| < |E⃗₁|cosα

    7.1 조건의 기하학적 의미

    |E⃗₁|cosα = E⃗₁을 E⃗₂ 방향으로 투영한 길이

조건: |E⃗₂| < |E⃗₁|cosα

시각화:
  E⃗₂ 방향 →
  ─────────────────────────────────
  ←─── E⃗₂ ───→
  ←──────── |E⃗₁|cosα ──────────→
            (E⃗₁의 그림자)

E⃗₂가 E⃗₁ 그림자 안에 완전히 들어온다!
어느 방향에서 보아도 E⃗₂가 항상 더 작다.

    7.2 왜 n̂* = E⃗₂ 방향인가 — 증명

    n̂ = E⃗₂ 방향에서:
  |E⃗₂·n̂| = |E⃗₂|           (E⃗₂는 최대)
  |E⃗₁·n̂| = |E⃗₁|cosα       (항상 > |E⃗₂|)
  min = |E⃗₂|
  AM = 2|E⃗₂|

n̂을 E⃗₂ 방향에서 벗어나면:
  |E⃗₂·n̂| = |E⃗₂|cos(δ) < |E⃗₂|  (δ = 벗어난 각도)
  min ≤ |E⃗₂|cos(δ) < |E⃗₂|
  AM < 2|E⃗₂|

∴ n̂* = E⃗₂ 방향,  E_AM_max = 2|E⃗₂|

    7.3 물리적 직관

    두 채널이 심하게 불균형:
  강한 채널(E⃗₁)을 아무리 크게 해도
  약한 채널(E⃗₂)이 병목이 되어 AM은 2|E⃗₂|로 고정.

최선 전략:
  E⃗₂ 전체를 살리는 방향(E⃗₂ 방향)으로 뉴런을 맞춘다.

실험 설계 교훈:
  경골신경 위치에서 두 채널이 심하게 불균형하면
  약한 채널의 전류를 키우는 것이 핵심.
  강한 채널을 키우는 것은 AM 증가에 기여하지 못한다.

    8. Case 2: |E⃗₂| ≥ |E⃗₁|cosα — 삼각형 높이

    8.1 조건의 의미

    |E⃗₂| ≥ |E⃗₁|cosα

→ E⃗₂ 방향에서는 E⃗₁이 병목 (|E⃗₁|cosα ≤ |E⃗₂|)
→ E⃗₁ 방향 근처에서는 E⃗₂가 병목 가능

따라서:
  E⃗₁ 방향 근처 n̂ → a 크고 b 작음 → E⃗₂ 병목
  E⃗₂ 방향 근처 n̂ → b 크고 a 작음 → E⃗₁ 병목

중간에 a = b가 되는 방향이 반드시 존재 → 그곳이 최적!

    8.2 등사영 조건 — 왜 a = b에서 최대인가

    min(a, b) 최대화 문제:

  n̂을 움직일 때 a와 b 변화:

  a > b인 구간: AM = 2b
    → n̂을 조금 움직여 b를 키우면 AM 증가
    → a > b를 유지하는 한 계속 개선 가능

  a < b인 구간: AM = 2a
    → n̂을 조금 움직여 a를 키우면 AM 증가

  ∴ 개선 불가한 점: a = b  (등사영 조건)

최적 조건: |E⃗₁·n̂*| = |E⃗₂·n̂*|

    8.3 최적 방향 n̂* 결정

    등사영 조건:
  E⃗₁·n̂* = E⃗₂·n̂*  (같은 부호로 정렬 후)

변환:
  (E⃗₁ - E⃗₂)·n̂* = 0

∴ n̂* ⊥ (E⃗₁ - E⃗₂)

★ 핵심: 최적 방향 n̂*는 두 전기장의 차이 벡터에 수직!

    8.4 기하학적 해석 — 삼각형

    세 점:
  O = 원점 (0,0)
  A = E⃗₁의 끝점
  B = E⃗₂의 끝점

  OA⃗ = E⃗₁,  OB⃗ = E⃗₂,  BA⃗ = E⃗₁ - E⃗₂

       A (E⃗₁ 끝점)
      /│
     / │
    /  │ h  ← 높이 (O에서 BA에 내린 수선의 발)
   /   │
  O────B (E⃗₂ 끝점)

n̂* = BA에 수직인 방향 (= OA와 OB에 동일하게 h로 투영되는 방향)

등사영 확인:
  E⃗₁·n̂* = h  (O에서 A까지의 n̂* 방향 성분)
  E⃗₂·n̂* = h  (O에서 B까지의 n̂* 방향 성분)

∴  E_AM_max = 2h

    8.5 삼각형 높이 h 계산

    삼각형 OAB의 넓이를 두 방법으로 계산한다:

    방법 1 — 외적 (두 변 OA, OB가 만드는 넓이):
  넓이 = (1/2)|E⃗₁ × E⃗₂|
  (|E⃗₁×E⃗₂| = |E⃗₁||E⃗₂|sinα = 평행사변형 넓이)

방법 2 — 밑변 × 높이 (밑변 = BA):
  넓이 = (1/2) × |E⃗₁ - E⃗₂| × h

같다고 놓으면:
  (1/2)|E⃗₁ × E⃗₂| = (1/2)|E⃗₁ - E⃗₂| × h

  h = |E⃗₁ × E⃗₂| / |E⃗₁ - E⃗₂|

∴  E_AM_max = 2h = 2|E⃗₁ × E⃗₂| / |E⃗₁ - E⃗₂|

    8.6 물리적 직관

    Case 2에서 두 채널이 "균형을 맞출 수 있는" 상황이다.

최적 방향에서:
  두 채널이 같은 크기(h)로 기여한다
  → 완전 변조에 가까운 상태
  → AM 최대

  "두 소방관이 정확히 같은 힘으로 당겼다 놓는다"
  → 가장 효과적인 신경 자극 패턴

전극 설계 목표:
  표적 신경 위치에서 Case 2 조건 달성
  + n̂*이 신경 주행 방향과 일치하도록 전극 배치

    9. 두 조건 비교 및 경계 연속성

    9.1 비교표

    ┌──────────────┬──────────────────────────┬────────────────────────────────────┐
│  항목         │  Case 1                  │  Case 2                            │
├──────────────┼──────────────────────────┼────────────────────────────────────┤
│  조건         │ |E⃗₂| < |E⃗₁|cosα        │ |E⃗₂| ≥ |E⃗₁|cosα                  │
│  물리         │ E⃗₂가 항상 병목           │ 균형 방향이 존재                    │
│  최적 n̂*     │ E⃗₂ 방향                  │ E⃗₁-E⃗₂에 수직                      │
│  E_AM_max    │ 2|E⃗₂|                    │ 2|E⃗₁×E⃗₂|/|E⃗₁-E⃗₂|               │
│  기하 의미    │ E⃗₂ 크기의 두 배          │ 삼각형 OAB 높이의 두 배             │
│  설계 교훈    │ 약한 채널이 결정함        │ 균형 방향 ↔ 신경 방향 정렬 필요    │
└──────────────┴──────────────────────────┴────────────────────────────────────┘

    9.2 경계에서 두 수식이 연속임을 확인

    경계: |E⃗₂| = |E⃗₁|cosα

Case 1: E_AM_max = 2|E⃗₂|

Case 2 수식에 경계 조건 대입:
  |E⃗₁ × E⃗₂| = |E⃗₁||E⃗₂|sinα

  |E⃗₁ - E⃗₂|²  = |E⃗₁|² + |E⃗₂|² - 2|E⃗₁||E⃗₂|cosα
  경계에서 |E⃗₂| = |E⃗₁|cosα이므로:
                 = |E⃗₁|² + |E⃗₁|²cos²α - 2|E⃗₁|²cos²α
                 = |E⃗₁|²sin²α
  → |E⃗₁ - E⃗₂| = |E⃗₁|sinα

  Case 2 수식:
    2 × |E⃗₁||E⃗₂|sinα / |E⃗₁|sinα = 2|E⃗₂|  ✓

두 수식은 경계에서 연속적으로 이어진다.

    10. 수치 예시

    예시 A — Case 1

    설정: |E⃗₁| = 10 V/m,  |E⃗₂| = 3 V/m,  α = 30°

조건 확인:
  |E⃗₁|cosα = 10 × 0.866 = 8.66 V/m
  |E⃗₂| = 3 < 8.66  → Case 1 ✓

결과:
  E_AM_max = 2 × 3 = 6 V/m
  n̂* = E⃗₂ 방향

해석:
  강한 채널(E⃗₁)이 아무리 커도 AM = 6 V/m로 고정.
  약한 채널(E⃗₂)을 키워야 한다.

    예시 B — Case 2

    설정: |E⃗₁| = 10 V/m,  |E⃗₂| = 8 V/m,  α = 60°

조건 확인:
  |E⃗₁|cosα = 10 × 0.5 = 5 V/m
  |E⃗₂| = 8 ≥ 5  → Case 2 ✓

계산:
  |E⃗₁ × E⃗₂| = 10 × 8 × sin60° = 80 × 0.866 = 69.28 V²/m²

  |E⃗₁ - E⃗₂|² = 100 + 64 - 2×10×8×0.5 = 164 - 80 = 84
  |E⃗₁ - E⃗₂| = √84 = 9.17 V/m

  h = 69.28 / 9.17 = 7.56 V/m

  E_AM_max = 2 × 7.56 = 15.12 V/m

비교: E⃗₂ 방향만 봤을 때:
  |E⃗₁|cosα = 5 V/m  (이게 병목)
  AM = 2 × 5 = 10 V/m  ← 최적이 아님!

최적 n̂* 사용: AM = 15.12 V/m  (51% 개선!)

    예시 C — 직교 케이스 손 계산

    E⃗₁ = (3, 0, 0) V/m,  E⃗₂ = (0, 4, 0) V/m  (직교, α = 90°)

조건 확인:
  |E⃗₁|cosα = 3 × cos90° = 0
  |E⃗₂| = 4 ≥ 0  → Case 2 ✓

등사영 조건: 3cosφ = 4sinφ
  tanφ* = 3/4 → φ* = arctan(3/4) = 36.87°

n̂* = (cos36.87°, sin36.87°, 0) = (0.800, 0.600, 0)

검증:
  E⃗₁·n̂* = 3 × 0.800 = 2.400
  E⃗₂·n̂* = 4 × 0.600 = 2.400  ← 정확히 같다! ✓

E_AM_max = 2 × 2.400 = 4.800 V/m

공식으로 확인:
  |E⃗₁ × E⃗₂| = 3 × 4 × sin90° = 12
  |E⃗₁ - E⃗₂| = √(9+16) = 5
  E_AM_max = 2 × 12/5 = 4.800 V/m  ✓

주목: n̂* = (0.8, 0.6) → E⃗₁(=3)을 향해 기울어짐
      더 약한 쪽(E⃗₁)을 살리기 위해 강한 쪽(E⃗₂)에서 멀어짐

    11. 타원 궤적 — 핵심 물리적 직관

    공간 점 P에서 E⃗(t) 벡터 끝점의 궤적:

         E_y
      ↑
      │   b (단반경)
      │ ╭────────────╮
      │╱      a       ╲
      │   ←──────────→ │   a = 장반경 (semi-major axis)
      │╲               ╱   b = 단반경 (semi-minor axis)
      │ ╰────────────╯
      └──────────────────→ E_x

E⃗(t)가 carrier 주기(≈1/f̄)로 이 타원 위를 돈다
envelope이 beat 주기(=1/Δf)로 타원 크기를 변조

    타원과 AM_max의 관계

    타원을 단위벡터 n̂ 방향으로 투영:

n̂ = 장반경 방향 → 투영 최대 = a
n̂ = 단반경 방향 → 투영 최소 = b

⭐ AM_max = 타원 장반경 a
   최적 방향 n̂* = 장반경 방향

    왜 타원이 생기나

    고정 slow-time τ = Δω·t/2에서:

  E⃗(t|τ) = A⃗(τ)·cos(ω̄t) + B⃗(τ)·sin(ω̄t)

  A⃗(τ) = (E⃗₁+E⃗₂)cos(τ)  ← 타원 한 축
  B⃗(τ) = (E⃗₂-E⃗₁)sin(τ)  ← 타원 다른 축

→ A⃗cos + B⃗sin 형태 = 타원 ✓

    12. SVD — 타원 장반경 계산

    행렬 구성

    COMSOL에서 복소 phasor export 후:

    Ẽ₁ = Re(Ẽ₁) + j·Im(Ẽ₁)  (f₁ 주파수 phasor)
Ẽ₂ = Re(Ẽ₂) + j·Im(Ẽ₂)  (f₂ 주파수 phasor)

M = [ Re(Ẽ₁x)  Im(Ẽ₁x)  Re(Ẽ₂x)  Im(Ẽ₂x) ]  ← x성분
    [ Re(Ẽ₁y)  Im(Ẽ₁y)  Re(Ẽ₂y)  Im(Ẽ₂y) ]  ← y성분
    [ Re(Ẽ₁z)  Im(Ẽ₁z)  Re(Ẽ₂z)  Im(Ẽ₂z) ]  ← z성분

M ∈ R^{3×4}

실수 phasor(위상 지연 없는 경우): Im = 0 → M = [E⃗₁, E⃗₂] ∈ R^{3×2}

    SVD 분해

    M = U Σ Vᵀ

  U ∈ R^{3×3}: 공간 방향 (열 = 직교 방향들)
  Σ: 특이값 σ₁ ≥ σ₂ ≥ σ₃ ≥ 0

특이값 의미:
  σ₁ → 타원 장반경 a  (가장 강한 방향의 크기)
  σ₂ → 타원 단반경 b

AM_max = 2 × σ₁
n̂*    = u₁  (U 행렬의 첫 번째 열)

    MATLAB 코드 (팀 실제 코드, 송솔웅 2026)

    % 각 점에서 복소 phasor (COMSOL export)
E1 = [E1x; E1y; E1z];  % complex 3x1
E2 = [E2x; E2y; E2z];  % complex 3x1

% 행렬 구성
M = [real(E1), imag(E1), real(E2), imag(E2)];  % 3x4 real

% SVD
[U, S, V] = svd(M, 'econ');

% AM_max
AM_max = 2 * S(1,1);
n_opt  = U(:,1);  % 최적 방향

    13. AM_max 정의 비교 — 두 가지 해석

    정의 A (Modulation Depth, 물리적 변조 깊이):
  AM_A(n̂) = 2 × min(|E⃗₁·n̂|, |E⃗₂·n̂|)
  특징: "약한 쪽"이 AM을 결정
  최적 조건: E₁n = E₂n (등사영)
  E_AM_max 수식 (Case 1, 2): 이 정의 기반

정의 B (Envelope Peak Amplitude, Grossman / 팀 MATLAB):
  AM_B(n̂) = sqrt((E⃗₁·n̂)² + (E⃗₂·n̂)²)
  특징: carrier 진폭의 RMS 기반
  SVD와 직접 연결: AM_B_max = 2σ₁

정의 A vs B 비교:
  E⃗₁ ∥ E⃗₂: 두 정의가 같은 n̂*를 선택
  E⃗₁ ⊥ E⃗₂ 예시 (E⃗₁=(3,0), E⃗₂=(0,4)):
    Def A: n̂* = (0.8, 0.6),  AM = 4.80
    Def B: n̂* = (0, 1, 0) = E⃗₂ 방향,  AM = 8.00  ← 다르다!

팀 코드(MATLAB): 정의 B 사용 → SVD 적용 가능

    15. Steering 원리

    전극 위치 또는 전류 비율 변경:
  → E⃗₁(r⃗), E⃗₂(r⃗) 공간 분포 변화
  → 각 위치에서 |E⃗₁|, |E⃗₂|, α 모두 변화
  → E_AM_max(r⃗) 분포 변화
  → 최대값 위치가 이동 = Steering!

Grossman 2017 발견:
  전류 비율을 바꾸면(전극 이동 없이)
  envelope peak가 두 전극쌍 사이를 이동한다.
  → 전류 비율만으로 표적 위치 조절 가능!

경골신경 TIS 적용:
  전극 배치 목표:
  1. 경골신경 위치에서 E_AM_max ≥ 자극 역치
  2. n̂*이 신경 주행 방향과 ≤ 30° 오차
  3. 주변 조직에서 E_AM_max << 역치 (선택성)

    16. E_AM_max의 한계

    E_AM_max는 "최적 방향 뉴런이 있다면"의 이론적 상한이다.

실제 자극은 항상 작거나 같다:

  실제 AM = E_AM_max × cos(θ_mismatch)
  θ_mismatch = n̂*와 실제 뉴런 방향의 각도

추가 한계:
  1. 세포 유형별 역치가 다름
     (motor axon vs sensory axon vs C-fiber)
  2. 조직 비등방성
     (DTI 기반 백질 섬유 방향성이 E⃗ 분포에 영향)
  3. 네트워크 효과
     (단일 뉴런 모델만으로 설명 불가)
  4. 환자별 해부학 차이

Grossman 논문 명시:
  "모델이 뇌 파라미터, 세포 유형, 지역별 흥분성 차이를
   완전히 반영하지 않는다"

    17. 물리적 검증 방법

    방법 1: Brute-force 수치 탐색

    import numpy as np

E1 = np.array([3., 0., 0.])
E2 = np.array([0., 4., 0.])

# SVD 방법
M = np.column_stack([E1, E2])
_, s, _ = np.linalg.svd(M, full_matrices=False)
AM_svd = 2 * s[0]

# Brute-force (1000×1000 방향 탐색)
max_am = 0
for theta in np.linspace(0, np.pi, 1000):
    for phi in np.linspace(0, 2*np.pi, 1000):
        n = np.array([np.sin(theta)*np.cos(phi),
                      np.sin(theta)*np.sin(phi),
                      np.cos(theta)])
        am = 2 * min(abs(E1@n), abs(E2@n))  # 정의 A
        if am > max_am:
            max_am = am

print(f"SVD(정의B):    {AM_svd:.4f} V/m")
print(f"Brute-force(A): {max_am:.4f} V/m")

    방법 2: COMSOL에서 시각적 검증

    검증 절차:
1. COMSOL에서 E_AM_max 맵 생성 (SVD 기반)
2. 같은 점에서 AM_x, AM_y, AM_z 각각 계산
3. 반드시: AM_max >= AM_x, AM_y, AM_z 확인
4. 신경 주행 방향(예: z축)과 AM_max 방향 각도 비교

평가 기준:
  AM_z     → 보수적 (실제 신경 주행 방향)
  AM_max   → 이론적 상한선 (어떤 신경이든 최대 받을 수 있는 값)

    18. 개념 요약

    [FEM 기초]
  ∇·(σ∇φ) = 0  ← ∇·J⃗=0 + J⃗=σE⃗ + E⃗=-∇φ 세 법칙 조합
  준정적 가정: λ=50km >> 몸 크기, 변위전류 무시
  → 두 전극쌍에 대해 각각 풀어 E⃗₁(r⃗), E⃗₂(r⃗) 획득

[AM 포락선]
  E_n(t) = (E⃗₁·n̂)cos(ω₁t) + (E⃗₂·n̂)cos(ω₂t)
  AM(n̂) = 2·min(|E⃗₁·n̂|, |E⃗₂·n̂|)  ← 약한 쪽 투영의 두 배

[E_AM_max 최적화]
  max over n̂ of AM(n̂)  ← max-min 문제

  Case 1: |E⃗₂| < |E⃗₁|cosα
    n̂* = E⃗₂ 방향,  E_AM_max = 2|E⃗₂|
    (약한 field가 명확한 병목)

  Case 2: |E⃗₂| ≥ |E⃗₁|cosα
    n̂* ⊥ (E⃗₁-E⃗₂),  E_AM_max = 2|E⃗₁×E⃗₂|/|E⃗₁-E⃗₂|
    (등사영 조건 → 삼각형 높이의 두 배)

[계산]
  타원 직관: AM_max = 장반경 a
  SVD:       M=[Re Ẽ₁, Im Ẽ₁, Re Ẽ₂, Im Ẽ₂],  AM=2σ₁,  n̂*=u₁

[핵심 직관]
  "두 채널의 투영을 같게 만드는 방향이 최적"
  "한쪽이 너무 약하면 그 방향이 최선"
  "균형이 가능하면 삼각형 높이 방향이 최선"

    19. 자기평가 문항

    Level 1

    1. 공간의 한 점에서 x방향 AM과 y방향 AM이 다른 이유는?
    2. AM(n̂) = 2min(E₁n, E₂n)에서 E₁n = 3, E₂n = 1 V/m일 때 AM은?
    3. 준정적 근사에서 변위전류를 무시할 수 있는 이유를 수치로 설명하라.

    Level 2

    1. 최적 방향 n̂*에서 반드시 E₁n = E₂n이 되어야 하는 이유를 직관적으로 설명하라.
    2. E⃗₁=(10,0), E⃗₂=(3,0), α=0°일 때 Case 1/2 중 어느 것인가? E_AM_max는?
    3. Case 2에서 왜 n̂*가 (E⃗₁-E⃗₂)에 수직인지를 등사영 조건으로 유도하라.

    Level 3

    1. E⃗₁=(10,0), E⃗₂=(0,8) V/m일 때 (α=90°): (a) Case 1 또는 Case 2 판정 (b) E_AM_max 계산 (공식과 수치 모두) (c) n̂* 방향 각도 계산

    2. 타원 궤적에서 장반경 방향으로 투영했을 때 AM이 최대인 이유를 기하학적으로 설명하라.

    Level 4

    1. Case 2의 E_AM_max = 2|E⃗₁×E⃗₂|/|E⃗₁-E⃗₂|가 삼각형 OAB의 높이의 두 배임을 넓이 공식으로 유도하라.
    2. Case 1과 Case 2의 경계(|E⃗₂|=|E⃗₁|cosα)에서 두 수식이 같은 값을 줌을 대수적으로 증명하라.

    Level 5

    1. SVD에서 σ₁이 타원 장반경과 같은 수학적 이유를 ||Mv||의 최대화 관점에서 설명하라.
    2. 정의 A와 정의 B가 서로 다른 n̂*를 주는 구체적 예시를 들고, 경골신경 TIS에서 어느 정의가 더 적절한지 논하라.
    3. 경골신경(z축 주행)에 대해 E_AM_max와 AM_z 중 어느 것이 더 적절한 자극 평가 지표인가? 임상적 의미를 논하라.
