    STEP 6: TIS Amplitude Modulation과 AM_max — 최대 진폭 축 완전 이해

    작성: 2026-06-24 커리큘럼: 6/13단계 참고: Grossman N et al., Cell 2017; 169(6):1029-1041 / 송솔웅 nTIS 전기장 계산 2026 / 김대용 COMSOL 2025-2026 수식 표기: Unicode/ASCII (LaTeX 없음, CLI 직독 가능)

    학습 목표

    TIS에서 단방향(x, y, z축) AM이 왜 불충분한지 이해하고, 최대 진폭 축(AM_max 방향 n*)을 타원 궤적 직관 + SVD 수식으로 도출하며, Grossman 2017 FEM 방법론을 구체적으로 파악할 수 있다.

    0. 전체 그림

    이미 아는 것:
  f1, f2 두 채널 → 심부에서 beat envelope → 신경 자극
  전기장 맵핑 → AM 분포 시각화

    1. Grossman 2017 FEM 시뮬레이션 파악

    모델 구성

    4층 동심구(concentric sphere) 모델:

  ┌─────────────────────────────────────────┐
  │  두피 (scalp)    r=8.5mm  σ=0.465 S/m  │
  │  ┌───────────────────────────────────┐  │
  │  │  두개골 (skull) r=8.0mm σ=0.01   │  │
  │  │  ┌─────────────────────────────┐ │  │
  │  │  │  CSF          r=7.5mm σ=1.65│ │  │
  │  │  │  ┌───────────────────────┐  │ │  │
  │  │  │  │  뇌 (brain) r=7.0mm  │  │ │  │
  │  │  │  │         σ=0.276 S/m   │  │ │  │
  │  │  │  └───────────────────────┘  │ │  │
  │  │  └─────────────────────────────┘ │  │
  │  └───────────────────────────────────┘  │
  └─────────────────────────────────────────┘

    지배 방정식

    Quasi-static 가정 (파장 150km >> 인체 0.3m):
  → 자기장 영향 무시
  → E = -∇V

전하 보존 (Laplace):
  ∇·(σ∇V) = 0

경계 조건:
  전극 A: +I 인가 (전류원)
  전극 B: -I 인가 (return)
  외부: 절연 n·J = 0

    FEM 해석 순서

    ┌─────────────────────────────────────────────────────────┐
│  STEP 1: 조직 모델 + 물질 σ 할당                         │
│  STEP 2: Frequency domain 해석 (두 번)                   │
│     1st run: f1 → E⃗₁(r) phasor 계산·저장               │
│     2nd run: f2 → E⃗₂(r) phasor 계산·저장               │
│  STEP 3: 각 공간점 r에서 AM_max 계산                     │
│  STEP 4: AM_max 분포 맵 시각화                           │
│     → 표적 영역: AM_max 높음                             │
│     → 비표적 영역: AM_max 낮음 (선택적 자극 확인)        │
└─────────────────────────────────────────────────────────┘

    핵심 결과

    전통적 단일채널:         TIS (Grossman 2017):
───────────────          ───────────────────
두피  ████████ 강        두피  ░░░░░░░░ 약
두개골 ▓▓▓▓▓▓ 중         두개골 ░░░░░░░░ 약
뇌표면 ▓▓▓▓▓▓ 중         뇌표면 ░░░░░░░░ 약
뇌심부 ░░░░░░ 약         뇌심부 ████████ 강  ← 표적!

    2. 왜 단방향 AM이 불충분한가

    3D 벡터 전기장

    실제 조직에서 각 공간점 P에 두 벡터가 존재:

    전극쌍 1 (f1): E⃗₁ = (E₁x, E₁y, E₁z)  V/m
전극쌍 2 (f2): E⃗₂ = (E₂x, E₂y, E₂z)  V/m

      E⃗₂ ↗
         ╱
        ╱  ← 두 벡터 방향이 다를 수 있다
       ╱
      ●──────────→ E⃗₁
      P

    단방향 AM의 한계

    x방향 측정: AM_x = f(E₁x, E₂x)  → y, z 정보 버림
y방향 측정: AM_y = f(E₁y, E₂y)  → x, z 정보 버림

실제 최대 AM 방향이 사선이라면:
  AM_x < AM_max   (과소 평가)
  AM_y < AM_max   (과소 평가)

→ x, y 방향 AM만 보면 실제 자극 강도를 과소평가한다

    3. 1D AM 공식 복습 (단방향)

    방향 단위벡터 n̂이 주어졌을 때:

    E_n(t) = E⃗(t)·n̂
       = (E⃗₁·n̂)·cos(ω₁t) + (E⃗₂·n̂)·cos(ω₂t)
       =  E₁n   ·cos(ω₁t) +  E₂n  ·cos(ω₂t)

    E₁n ≥ E₂n > 0 일 때 포락선:

    최대 = E₁n + E₂n ───╮
                     │  AM 깊이 = 2 × min(E₁n, E₂n) = 2E₂n
최소 = E₁n - E₂n ───╯

    n̂을 바꾸면 E₁n, E₂n이 바뀌고 AM도 달라진다.

    4. 최적화 문제 공식화

    AM_max = max_{||n̂||=1}  2 × min(E⃗₁·n̂, E⃗₂·n̂)

목표: 어느 방향 n̂*이 AM을 최대화하는가?

    직관적 분석

    n̂ = E⃗₁ 방향  → E₁n 최대, E₂n 매우 작음 → min = 작음 → AM 작음
n̂ = E⃗₂ 방향  → E₂n 최대, E₁n 매우 작음 → min = 작음 → AM 작음
n̂ = 두 벡터 "중간" → 둘 다 적당히 큼 → min = 최대 → AM 최대!

⭐ 최적 n̂*은 E₁n = E₂n이 되는 방향이다

    2D 예시 (E⃗₁ ⊥ E⃗₂)

          E⃗₂ ↑
         │      n̂* (최적, 45도)
         │     ╱
         │    ╱
         │   ╱ 45°
         │  ╱
         ● ──────────→ E⃗₁

E⃗₁ 방향: AM = 2min(E₁, 0) = 0
E⃗₂ 방향: AM = 2min(0, E₂) = 0
45° 방향: AM = 2min(E₁/√2, E₂/√2) = 최대  ← 균등 기여

    5. 타원 궤적 — 핵심 물리적 직관

    공간 점 P에서 E⃗(t) 벡터 끝점의 궤적:

         E_y
      ↑
      │   b (단반경)
      │ ╭────────────╮
      │╱      a       ╲
      │   ←──────────→ │   a = 장반경 (semi-major axis)
      │╲               ╱   b = 단반경 (semi-minor axis)
      │ ╰────────────╯
      └──────────────────→ E_x

E⃗(t)가 carrier 주기(≈1/f̄)로 이 타원 위를 돈다
envelope이 beat 주기(=1/Δf)로 타원 크기를 변조

    왜 타원이 생기나

    ω₁ = ω̄ + Δω/2,  ω₂ = ω̄ - Δω/2  (Δω << ω̄)

E⃗(t) = E⃗₁·cos(ω₁t) + E⃗₂·cos(ω₂t)
      = cos(ω̄t)·[(E⃗₁+E⃗₂)cos(Δωt/2)]   ← in-phase 성분
      + sin(ω̄t)·[(E⃗₂-E⃗₁)sin(Δωt/2)]   ← quadrature 성분

고정 slow-time τ = Δωt/2에서:
  E⃗(t|τ) = A⃗(τ)·cos(ω̄t) + B⃗(τ)·sin(ω̄t)

  A⃗(τ) = (E⃗₁+E⃗₂)cos(τ)  ← 타원 한 축 방향 × 진폭
  B⃗(τ) = (E⃗₂-E⃗₁)sin(τ)  ← 타원 다른 축 방향 × 진폭

→ A⃗cos + B⃗sin 형태 = 타원 ✓

    타원과 AM_max의 관계

    타원을 단위벡터 n̂ 방향으로 투영(projection):

         n̂
          ↑
    ╭─────┼────╮
   ╱      │     ╲
  │   타원 │      │
   ╲      │     ╱
    ╰─────┼────╯
          ↓ 투영(그림자)

          ████████

n̂ = 장반경 방향 → 투영 최대 = a
n̂ = 단반경 방향 → 투영 최소 = b

⭐ AM_max = 타원 장반경 a
   최적 방향 n̂* = 장반경 방향

    6. SVD — 타원 장반경을 계산하는 방법

    행렬 구성

    COMSOL에서 복소 phasor export 후:

    Ẽ₁ = Re(Ẽ₁) + j·Im(Ẽ₁)  (f1 주파수 phasor)
Ẽ₂ = Re(Ẽ₂) + j·Im(Ẽ₂)  (f2 주파수 phasor)

M = [ Re(Ẽ₁x)  Im(Ẽ₁x)  Re(Ẽ₂x)  Im(Ẽ₂x) ]  ← 행: x성분
    [ Re(Ẽ₁y)  Im(Ẽ₁y)  Re(Ẽ₂y)  Im(Ẽ₂y) ]  ← 행: y성분
    [ Re(Ẽ₁z)  Im(Ẽ₁z)  Re(Ẽ₂z)  Im(Ẽ₂z) ]  ← 행: z성분

M ∈ R^{3×4}

    실수 phasor(위상 지연 없는 경우): Im = 0이므로 M → [E⃗₁, E⃗₂] ∈ R^{3×2}

    SVD 분해

    M = U Σ Vᵀ

  U ∈ R^{3×3}: 공간 회전 (열 = 직교 방향들)
  Σ ∈ R^{3×4}: 대각 (σ₁ ≥ σ₂ ≥ σ₃ ≥ 0)
  V ∈ R^{4×4}: 입력 공간 회전

특이값 의미:
  σ₁ → 타원 장반경 a  (가장 강한 방향의 크기)
  σ₂ → 타원 단반경 b  (두 번째 방향)

AM_max  = 2 × σ₁       (피크-투-피크, σ₁이 0→peak이므로 ×2)
최적 방향 n̂* = u₁       (U 행렬의 첫 번째 열)

    왜 SVD가 장반경을 주는가

    M이 하는 일:
  ℝ⁴의 단위구(unit sphere) → ℝ³의 타원체(ellipsoid)로 변환
  단위 에너지 v (||v||=1) → Mv ∈ ℝ³ (타원 위의 점)

||Mv||를 최대화하는 v = V의 첫 번째 열 v₁
이때 ||Mv₁|| = σ₁ = 장반경

물리적 해석:
  E⃗₁, E⃗₂가 만드는 공간에서
  단위 에너지로 만들 수 있는 가장 강한 전기장 벡터 크기 = σ₁

    MATLAB 코드 (팀 실제 코드, 송솔웅 2026)

    % 각 점에서 복소 phasor (COMSOL export 데이터)
E1 = [E1x; E1y; E1z];  % complex, 3x1
E2 = [E2x; E2y; E2z];  % complex, 3x1

% 행렬 구성
M = [real(E1), imag(E1), real(E2), imag(E2)];  % 3x4 real

% SVD
[U, S, V] = svd(M, 'econ');

% AM_max
AM_max   = 2 * S(1,1);
n_opt    = U(:,1);      % 최적 방향

    7. 물리적 검증 방법

    방법 1: Brute-force 수치 탐색

    import numpy as np

E1 = np.array([3., 0., 0.])  # 실수 phasor 예시
E2 = np.array([0., 4., 0.])

# SVD 방법
M = np.column_stack([E1, E2])
_, s, _ = np.linalg.svd(M, full_matrices=False)
AM_svd = 2 * s[0]

# Brute-force (1000×1000 방향 탐색)
max_am = 0
for theta in np.linspace(0, np.pi, 1000):
    for phi in np.linspace(0, 2*np.pi, 1000):
        n = np.array([np.sin(theta)*np.cos(phi),
                      np.sin(theta)*np.sin(phi),
                      np.cos(theta)])
        am = np.sqrt((E1@n)**2 + (E2@n)**2)
        if am > max_am:
            max_am = am

print(f"SVD:         {AM_svd:.4f} V/m")
print(f"Brute-force: {2*max_am:.4f} V/m")
# 두 값이 같으면 SVD 공식 검증 완료

    방법 2: COMSOL에서 시각적 검증

    검증 절차:
1. COMSOL에서 AM_max 맵 생성 (SVD 기반)
2. 같은 점에서 AM_x, AM_y, AM_z 각각 계산
3. 반드시: AM_max >= AM_x, AM_y, AM_z 확인
4. 신경 주행 방향(z축)과 AM_max 방향 비교
   → 만약 AM_max 방향 ≠ z축: 신경 방향이 최적 아님을 의미

평가 기준:
  AM_z       → 보수적 (실제 신경 주행 방향)
  AM_max     → 공정한 상한선 (어떤 신경 방향이든 최대 받을 수 있는 값)

    8. AM_max 정의 비교 — 두 가지 해석

    정의 A (Modulation Depth, 물리적 변조 깊이):
  AM_A(n̂) = 2 × min(|E⃗₁·n̂|, |E⃗₂·n̂|)
  특징: 두 소스의 "약한 쪽"이 결정
  최적 조건: E₁n = E₂n (균등 기여)

정의 B (Envelope Peak Amplitude, Grossman / 팀 MATLAB):
  AM_B(n̂) = sqrt((E⃗₁·n̂)² + (E⃗₂·n̂)²)
  특징: carrier 진폭의 RMS 기반
  SVD와 직접 연결: AM_B_max = 2σ₁

정의 A vs B:
  E⃗₁ ∥ E⃗₂인 경우: 두 정의가 같은 방향을 최적으로 선택
  E⃗₁ ⊥ E⃗₂인 경우: 최적 n̂*가 서로 달라짐

팀 코드(MATLAB): 정의 B 사용 → SVD 적용 가능

    9. Grossman FEM의 전체 AM 계산 파이프라인

    10. 개념 요약

    AM_max 핵심 흐름:

1D 단방향 AM
  E_n(t) = E₁n·cos(ω₁t) + E₂n·cos(ω₂t)
  AM = 2×min(E₁n, E₂n)
         ↓ n̂ 방향에 따라 값이 달라진다

3D 최적화 문제
  max_{||n̂||=1} AM(n̂) = AM_max
  최적 조건: E⃗₁·n̂ = E⃗₂·n̂ (균등 기여)
         ↓
타원 궤적 직관
  E⃗(t)가 carrier 주기로 타원을 그린다
  AM_max = 장반경 a
  최적 방향 = 장반경 방향
         ↓
SVD로 계산
  M = [Re(Ẽ₁), Im(Ẽ₁), Re(Ẽ₂), Im(Ẽ₂)]
  SVD: σ₁ = 장반경
  AM_max = 2σ₁,  n̂* = u₁
