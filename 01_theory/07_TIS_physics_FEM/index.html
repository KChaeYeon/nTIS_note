    관련 파일: TIS 원리 기초 | 경골신경 TIS

    Part 1. 전류가 몸 속에서 어떻게 흐르는가

    🔑 핵심 비유: 물이 파이프를 흐르는 것과 같다

    전기가 몸 속을 흐르는 것은, 물이 여러 굵기의 파이프를 흐르는 것과 똑같습니다.

        좁은 파이프          굵은 파이프
  (전류가 힘들게 흐름)   (전류가 쉽게 흐름)
      ━━━━━━━━━━             ══════════
    물이 조금만 흐름         물이 많이 흐름

몸 속 조직에서:
  뼈 ─────── 가장 좁은 파이프 (전류가 거의 못 통과)
  지방 ────── 좁은 파이프
  신경 ────── 중간 파이프
  근육 ────── 굵은 파이프   (전류가 잘 통과)

    이 "파이프 굵기"를 전도도(σ, sigma)라고 부릅니다.

    발목 횡단면 — 조직별 전도도

                        발목 횡단면 (Kim 2023 기준)

     ╔════════════════════════════════════════╗
     ║    피부 σ=0.17 S/m                    ║
     ║  ┌────────────────────────────────┐   ║
     ║  │   지방 σ=0.057 S/m            │   ║
     ║  │  ┌──────────────────────────┐ │   ║
     ║  │  │    근육 σ=0.355 S/m      │ │   ║
     ║  │  │   ┌──────┐  ┌──────────┐ │ │   ║
     ║  │  │   │  뼈  │  │  혈관   │ │ │   ║
     ║  │  │   │0.0035│  │ σ=0.232 │ │ │   ║
     ║  │  │   └──────┘  └──────────┘ │ │   ║
     ║  │  │                          │ │   ║
     ║  │  │   ★ 경골신경             │ │   ║
     ║  │  │      σ=0.265 S/m        │ │   ║
     ║  │  └──────────────────────────┘ │   ║
     ║  └────────────────────────────────┘   ║
     ╚════════════════════════════════════════╝

  σ 값이 클수록 전류가 잘 흐름
  σ 값이 작을수록 전류가 잘 못 흐름 (뼈 = 거의 절연체)

    S/m이란?

    전도도의 단위. 클수록 전기가 잘 통하는 물질. 금속(구리): ~60,000,000 S/m → 뼈: 0.0035 S/m → 약 170억 배 차이!

    1-2. FEM이 하는 일 — "전기장 지도" 그리기

    전극에서 전류를 흘리면, 몸 속 어디에 전기장이 얼마나 강한지를 컴퓨터로 계산합니다.

         [+] 전극                         [-] 전극
       │                                 │
       │  ╔═══════════════════════════╗  │
       └─►║                          ║◄─┘
          ║    전류가 이렇게 퍼짐:    ║
          ║                          ║
          ║  ──→──→──→──→──→──→──→  ║  ← 직선으로 흐르는 전류
          ║  ─→   →    →   →    →─  ║
          ║  →     ╲  ★  ╱     ←   ║  ← 경골신경 주변
          ║  ─→     ╲   ╱      →─  ║
          ║  ──→──→──→──→──→──→──→  ║
          ╚═══════════════════════════╝

FEM 계산 결과:
  각 위치(r)에서 전기장 벡터 E(r) = -▽V(r) 획득
  (▽V = 전위의 공간적 기울기)

    핵심 수식:

    \[\nabla \cdot (\sigma \nabla V) = 0\]

    이 수식이 말하는 것

    "전류는 어디서 갑자기 생기거나 사라지지 않는다." → 들어간 전류 = 나온 전류 (어느 점에서나 성립) → 이 조건을 모든 점에서 만족하는 전위 분포 V(r)를 컴퓨터가 찾아냄

    1-3. 왜 복잡한 Maxwell 방정식을 안 쓰는가?

    전자기학 완전 방정식은 매우 복잡합니다. 하지만 kHz 전류 + 생체조직이라는 조건에서는 두 가지가 극히 작아서 무시할 수 있습니다.

                    kHz 전류 + 생체조직에서:

  완전 방정식                 단순화 (Quasi-Static)
  ─────────────────────       ─────────────────────
  rot E = -dB/dt     ──→     rot E ≈ 0
                              ↓ E = -grad(V)

  rot H = J + dD/dt  ──→     rot H = J
                              ↓ J = σE

  결과: ∇·(σ∇V) = 0  ✅ 이것만 풀면 됨!

왜 무시가 되는가?
  피부 기준, 2 kHz에서:
  ┌─────────────────────────────────────────┐
  │ 전도 전류 σ    = 0.170 (단위: S/m)     │
  │ 변위 전류 ωε   = 0.00011              │
  │ 비율            = 1,545 배             │
  │                                         │
  │ 전도 전류 >> 변위 전류 → 변위 전류 무시 │
  └─────────────────────────────────────────┘

    Part 2. TI 신호란 무엇인가 — 맥놀이 현상

    🔑 핵심 비유: 두 소리굽쇠의 "맥놀이"

    소리로 이해하는 TI

    피아노 건반 두 개를 동시에 칩니다.

    • 건반 A: 440 Hz (라 음)
    • 건반 B: 441 Hz (살짝 다른 라 음)

    그러면 1 Hz로 소리가 왔다갔다 커졌다 작아졌다 합니다. 이것이 "맥놀이(beat)"입니다.

    TIS도 똑같습니다: - 채널 1: 2,000 Hz - 채널 2: 2,020 Hz - → 몸 속에서 20 Hz 맥놀이 발생!

    두 채널 신호의 시간 파형

    채널 1 (f₁ = 2,000 Hz):
  │▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼│
  시간→

채널 2 (f₂ = 2,020 Hz):
  │▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲▼▲│
  시간→  (살짝 더 빠름)

합쳐진 신호 (몸 속에서):
  │                                        │
  │  ╭────╮                  ╭────╮        │
  │ ╱      ╲                ╱      ╲       │
  │╱ 고주파  ╲              ╱  고주파 ╲     │
  │ 진동 많음 ╲            ╱           ╲   │
  │            ╲          ╱             ╲  │
  │             ╲        ╱               ╲ │
  │              ╲──────╯                 ╲│
  │                                        │
  │←─── 1/Δf = 50 ms (20 Hz) ────────────→│
  │   ↑                   ↑               │
  │ 보강(최대)          보강(최대)          │
  │          ↑                            │
  │        상쇄(최소)                      │

  이 포락선(envelope)이 Δf = 20 Hz로 진동!

    TI 전기장 수식

    \[\boxed{E_{\text{TI}}(\mathbf{r}) = \Big|\,|\mathbf{E}_1(\mathbf{r}) + \mathbf{E}_2(\mathbf{r})| - |\mathbf{E}_1(\mathbf{r}) - \mathbf{E}_2(\mathbf{r})|\,\Big|}\]

    수식 읽는 법 (3단계)

    ① \(\mathbf{E}_1(\mathbf{r})\), \(\mathbf{E}_2(\mathbf{r})\) : FEM으로 계산한 채널1, 채널2의 전기장. 위치(r)마다 다름.

    ② \(|\mathbf{E}_1 + \mathbf{E}_2|\) : 두 전기장이 완전히 보강될 때 크기 → Envelope 최대값

    ③ \(|\mathbf{E}_1 - \mathbf{E}_2|\) : 두 전기장이 완전히 상쇄될 때 크기 → Envelope 최소값

    → 차이 = 포락선이 진동하는 폭 = TI 효과의 크기

    세 가지 경우로 완전히 이해하기

    ┌─────────────────────────────────────────────────────────────┐
│ 경우 1: E₁ = E₂ (두 채널 크기 같음) → 최대 TI 효과        │
│                                                             │
│  포락선:                                                    │
│  2E ─ ─╮          ╭─────╮          ╭──                    │
│         │╲        ╱       ╲        ╱                       │
│         │  ╲    ╱           ╲    ╱                        │
│         │    ╲╱               ╲╱                          │
│  0  ────╯                                                  │
│                                                             │
│  포락선이 0에서 2E까지 완전히 진동 → E_TI = 2E (최대!)    │
├─────────────────────────────────────────────────────────────┤
│ 경우 2: E₁ >> E₂ (한쪽이 훨씬 강함) → TI 효과 약해짐     │
│                                                             │
│  포락선:                                                    │
│  E₁+E₂ ─╮          ╭────╮          ╭─                    │
│           │╲        ╱      ╲        ╱                      │
│           │  ╲    ╱          ╲    ╱                       │
│  E₁-E₂ ──╯    ╲╱              ╲╱                         │
│                                                             │
│  진동 폭 = 2E₂ 만큼만 → E_TI = 2E₂ (작음!)              │
│  ★ 전류 비율 I₁:I₂ 최적화가 필요한 이유!                   │
├─────────────────────────────────────────────────────────────┤
│ 경우 3: E₁ ⊥ E₂ (두 방향이 직각) → TI 효과 없음          │
│                                                             │
│  |E₁+E₂| = √(E₁²+E₂²)                                    │
│  |E₁-E₂| = √(E₁²+E₂²)  ← 크기가 같음!                   │
│                                                             │
│  E_TI = 0 → 자극 없음!                                    │
│  ★ 전기장 방향이 표적에서 평행해야 하는 이유!               │
└─────────────────────────────────────────────────────────────┘

    Part 3. 왜 심부(표적)에서만 자극되는가

    🔑 핵심 비유: 두 손전등 교차

    손전등으로 이해하는 심부 선택적 자극

    어두운 방에서 두 손전등을 비춥니다.

    손전등 A ────────────────────►
                    ✕  ← 교차점만 밝음!
손전등 B ────────────────────►
    • 손전등 A가 닿는 곳: 밝음
    • 손전등 B가 닿는 곳: 밝음
    • 둘이 교차하는 한 점: 훨씬 더 밝음

    TIS도 똑같습니다. 두 채널의 전기장이 교차하는 표적 위치에서만 강한 자극!

    발목에서 TI 전기장 분포

             전극A(+)                     전극A(-)
            │                            │
    ────────●────────────────────────────●────────  피부
            │                            │
            │  E₁ 강함, E₂ 약함          │
            │  → E_TI = 2×E₂ (작음)     │
            │                            │
    ────────────────────────────────────────────
            │                            │
            │         ★ 경골신경         │
            │      E₁ ≈ E₂ 교차점!      │
            │   → E_TI = 2E (최대!)     │
            │                            │
    ────────────────────────────────────────────
            │                            │
            │  E₁ 약함, E₂ 강함          │
            │  → E_TI = 2×E₁ (작음)     │
            │                            │
    ────────●────────────────────────────●────────  피부
         전극B(+)                     전극B(-)

    Part 4. Peak Ratio (PR) — "얼마나 정확하게 맞혔나?" 점수

    🔑 핵심 비유: 다트 점수판

    다트로 이해하는 PR

    다트 게임에서: - 화살이 정중앙(경골신경)에 몰려 있으면 → 높은 점수 (PR 높음) - 화살이 판 전체에 퍼져 있으면 → 낮은 점수 (PR 낮음)

    PR = (경골신경 위치 전기장) ÷ (발목 전체 평균 전기장)

    PR 수식

    \[\text{PR} = \frac{\text{경골신경 단면 평균 } E_{\text{TI}}}{\text{발목 전체 단면 평균 } E_{\text{TI}}}\]

    발목 단면에서 E_TI 분포와 PR 비교

                        발목 단면 E_TI 분포도

    Part 5. PR 지표의 한계 — 완벽하지 않은 이유

    한계 1: 평균값이라 분포를 모른다

    두 경우 모두 PR = 4 (평균 동일):

경우 A (뾰족하게 집중):
E_TI │         │
     │      ╭─╯╰─╮
     │     ╱       ╲
     │────╱─────★───╲────
     └──────────────────── 위치

E_TI │
     │   ╭─────────────╮
     │  ╱               ╲

→ PR만 보면 구별 불가! 하지만 신경 자극 효과는 다를 수 있음

    한계 2: 실제로 자극이 될 만큼 강한지 모른다

    신경 발화 역치선 ─ ─ ─ ─ ─ ─ ─ ─ (예: 10 V/m)

상황 A: PR=4, 역치 넘음 → 신경 발화 ✅
         E_TI   ╭──╮
                │  │  ← 역치보다 높음
 역치 ─ ─ ─ ─ ─│─ │─ ─ ─ ─
                ╰──╯

상황 B: PR=4이지만 전체 절대값이 낮음 → 신경 발화 ❌
 역치 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
         E_TI  ╭──╮
               │  │  ← 역치보다 낮음
               ╰──╯

→ PR이 같아도 전류가 너무 약하면 아무 효과 없음!
→ In vivo 실험이 반드시 필요한 이유

    한계 3: Budde 2023의 반론

    메커니즘 논쟁

    Grossman 2017 (TIS 제안) 주장:

    "뉴런이 포락선(envelope)을 추출해서 Δf 주파수로 발화한다" → E_TI 수식의 근거

    Budde 2023 (Johns Hopkins) 반론:

    "뉴런은 포락선을 추출하지 않는다. 뉴런막은 RC 회로처럼 작동하며, 발화 결정 요인은 peak-to-peak 진폭이다"

    Grossman 관점:        Budde 관점:
포락선 추출           단순 진폭 역치
     ↓                     ↓
Δf 주파수로 발화      총 신호 크기가 역치 넘으면 발화

    → 두 관점 모두 전극 배치 전략은 비슷하지만 (표적에서 두 채널 중첩 극대화) → 우리 in vivo 실험이 어느 쪽이 맞는지 부분적으로 검증 가능!

    총 전류 I₁+I₂ 고정, Δf만 변경:

  그룹 A: Δf = 20 Hz  ──→  ICI 변화 있으면?
  그룹 B: Δf = 100 Hz ──→  ICI 변화 있으면?

  결과 해석:
  Δf에 따라 ICI가 다르다 → Grossman 관점 지지 (포락선 추출)
